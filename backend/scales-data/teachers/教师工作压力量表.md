# 中小学教师工作压力量表

> **来源**：徐长江等编制，发表于《教育研究》1999年。以下完整53题来自公开学术平台（问卷星 wjx.cn/m/36396574）及公开文献，与发表版本高度一致。

## 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 徐长江等 |
| 适用对象 | 中小学教师 |
| 题目数量 | 53题（压力源36题 + 压力反应17题） |
| 答题形式 | 五级评分 |
| 测试时间 | 约15-20分钟 |

---

### 压力源评分标准

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 没有压力 | 该事项对我无压力 | 0分 |
| 压力较轻 | 有一点压力 | 1分 |
| 压力中等 | 有一定压力 | 2分 |
| 压力较大 | 压力比较大 | 3分 |
| 压力很大 | 压力非常大 | 4分 |

## 第二部分：工作压力反应（17题）

> 请评估您**近一个月**内出现以下状况的频率。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 感到疲劳 |
| 2 | 有头疼、气闷、耳鸣等身体不适的情形 |
| 3 | 感冒发烧 |
| 4 | 食欲下降 |
| 5 | 睡眠状况不佳 |
| 6 | 吸烟次数增多 |
| 7 | 在工作中错误率增高 |
| 8 | 对业余爱好的兴趣下降 |
| 9 | 与人交往兴趣下降 |
| 10 | 焦虑、紧张 |
| 11 | 爱发脾气，烦躁 |
| 12 | 情绪低落 |
| 13 | 注意力不如以前集中 |
| 14 | 记忆力下降 |
| 15 | 思维缓慢、混乱、反应迟钝 |
| 16 | 对自己目前的工作状况不满意，想换新的工作 |
| 17 | 对子女教育不如以前关心 |

### 压力反应评分标准

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 没有 | 完全没有出现 | 0分 |
| 很少 | 偶尔出现 | 1分 |
| 有时 | 有时出现 | 2分 |
| 经常 | 经常出现 | 3分 |
| 总是 | 几乎总是如此 | 4分 |

---

## 结果解释

- **压力源总分** = 36题得分之和（0-144分）
- **压力反应总分** = 17题得分之和（0-68分）

| 压力源总分 | 压力水平 |
|---------|---------|
| 0-36分 | 低压力 |
| 37-72分 | 中等压力 |
| 73-108分 | 较高压力 |
| 109-144分 | 极高压力，需要关注 |
