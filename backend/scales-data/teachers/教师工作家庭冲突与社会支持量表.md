# 教师工作-家庭冲突与社会支持量表系列

---

## 一、教师工作-家庭冲突问卷

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 吴明霞等（西南大学，2009年） |
| 适用对象 | 中小学教师 |
| 题目数量 | 22题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估教师在工作与家庭角色之间的冲突程度 |

### 量表结构（2个分问卷）

| 分问卷 | 题目数 | 主要内容 |
|-------|-------|---------|
| 工作干扰家庭（WIF） | 11题 | 工作压力、时间、精力影响家庭生活 |
| 家庭干扰工作（FIW） | 11题 | 家庭责任、压力影响工作投入 |

### 完整题目（22题）

> 请根据**最近三个月**内的实际情况选择最符合的选项。

**工作干扰家庭（WIF，11题）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我的工作让我没有时间陪伴家人 |
| 2 | 下班后我仍需要处理工作上的事情，影响了家庭生活 |
| 3 | 工作结束后，我太疲惫了，无法参与家庭活动 |
| 4 | 工作中的压力让我回家后仍然心情不好 |
| 5 | 由于工作繁忙，我常常忽视家庭成员的需求 |
| 6 | 工作上的问题使我难以集中精力处理家务 |
| 7 | 我常常因为工作而无法参加家庭的重要活动 |
| 8 | 工作要求导致我需要改变家庭计划 |
| 9 | 工作的时间安排影响了我照顾家人的时间 |
| 10 | 工作上的压力使我对家人的态度不好 |
| 11 | 工作占据了我本该用于家庭的精力 |

**家庭干扰工作（FIW，11题）**

| 题号 | 题目内容 |
|-----|---------|
| 12 | 家庭事务让我难以按时完成工作任务 |
| 13 | 家庭中的责任使我无法全身心投入工作 |
| 14 | 家庭问题影响了我上课的专注程度 |
| 15 | 我需要请假处理家庭事务，影响了工作 |
| 16 | 家里的压力让我在工作中无法发挥正常水平 |
| 17 | 照顾家人的需求使我难以参加学校的培训活动 |
| 18 | 家庭中的紧急事件常常打乱我的工作计划 |
| 19 | 家庭事务占用了我本应用于备课的时间 |
| 20 | 我常常把家庭中的担忧带到工作中 |
| 21 | 家庭中的情感压力影响了我的工作效率 |
| 22 | 家庭责任使我难以在工作中进取和发展 |

### 评分标准

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 1 | 极少发生（几乎从不） | 1分 |
| 2 | 偶尔发生 | 2分 |
| 3 | 有时发生 | 3分 |
| 4 | 经常发生 | 4分 |
| 5 | 总是这样 | 5分 |

- **WIF得分** = 第1-11题之和（11-55分）
- **FIW得分** = 第12-22题之和（11-55分）
- 总分越高，工作-家庭冲突越严重

## 二、教师社会支持量表（SSRS）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 社会支持评定量表（Social Support Rating Scale） |
| 编制者 | 肖水源（1986年，中国本土） |
| 适用对象 | 中小学教师（及成人群体） |
| 题目数量 | 10题 |
| 答题形式 | 混合评分 |
| 测试时间 | 约5-8分钟 |
| 用途 | 评估教师感知到的社会支持水平 |

### 量表结构（3个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 客观支持 | 3题 | 实际获得的物质和行动支持 |
| 主观支持 | 4题 | 感受到的情感支持和被尊重感 |
| 对支持的利用度 | 3题 | 倾向于主动寻求支持的程度 |

### 完整题目（10题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 您有多少关系密切，可以得到支持和帮助的朋友？（数量选择） |
| 2 | 近一年来您：（多选）A与同事的关系 B与领导的关系 C与家人的关系 |
| 3 | 您与邻居的关系如何？ |
| 4 | 您与同事的关系如何？ |
| 5 | 您从家庭成员处得到的支持和照顾如何？ |
| 6 | 过去，在您遇到急难情况时，曾经得到的经济支持和解决实际问题的帮助来源有：（多选） |
| 7 | 过去，在您遇到急难情况时，曾经得到的安慰和关心来源有：（多选） |
| 8 | 您遇到烦恼时的倾诉方式：（选择一项） |
| 9 | 您遇到烦恼时的求助方式：（选择一项） |
| 10 | 对于团体（如党团组织、宗教组织、工会、学生会等）组织活动，您：（选择一项） |

### 评分标准

- 第1-4题：每题1-4级评分
- 第5-7题：多选题，每选一个来源计1分
- 第8-10题：每题1-4级评分
- **总分** = 各题得分之和（12-66分），分数越高社会支持越丰富

## 三、教师组织支持感量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 参考编制 | 基于Eisenberger等的组织支持感理论（POS）本土化 |
| 适用对象 | 中小学教师 |
| 题目数量 | 22题（3个维度） |
| 答题形式 | 五级评分 |
| 用途 | 评估教师感知到的来自学校组织的支持 |

### 量表结构（3个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 工作支持 | 8题 | 学校提供的工作资源和条件支持 |
| 价值认同 | 7题 | 学校对教师贡献的认可和重视 |
| 关心利益 | 7题 | 学校对教师个人利益和福祉的关心 |

### 代表性题目

| 维度 | 题目示例 |
|-----|---------|
| 工作支持 | 学校为我提供了完成工作所需的资源和条件 |
| 工作支持 | 当我工作遇到困难时，学校会给予帮助 |
| 价值认同 | 学校认可和重视我对教育事业的贡献 |
| 价值认同 | 学校对我的工作成就表示肯定 |
| 关心利益 | 学校关心我的身心健康 |
| 关心利益 | 学校在做决策时会考虑我的利益 |

### 评分标准

- 五级评分：1=完全不符合，5=完全符合
- 总分范围 22-110分
- Cronbach's α = 0.941
