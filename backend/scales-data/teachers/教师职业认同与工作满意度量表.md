# 教师职业认同与工作满意度量表系列

---

## 一、中小学教师职业认同量表（魏淑华等，18题）

> **来源**：魏淑华、宋广文、张大均（2013年）编制，发表于《教师教育研究》第25卷第1期，第55-60页。以下18道完整题目来自公开在线调查平台（wjx.cn 问卷星），与文献描述的四因子18题结构相符。

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 魏淑华、宋广文、张大均 |
| 发表来源 | 《教师教育研究》2013年第25卷第1期，第55-60页 |
| 适用对象 | 中小学教师 |
| 题目数量 | 18题 |
| 量表结构 | 4因子（职业价值观、角色价值观、职业归属感、职业行为倾向）|
| 答题形式 | 五级评分（1=完全不符合，5=完全符合） |

### 四因子结构

| 因子 | 含义 | 主要对应题目 |
|-----|-----|-----------|
| 职业价值观 | 对教师职业本身价值和意义的认可 | 1、8、9等 |
| 角色价值观 | 对自身在教师角色中的胜任感和价值感 | 6、7等 |
| 职业归属感 | 对教师群体的情感归属和认同 | 16、17、18等 |
| 职业行为倾向 | 在职业中的积极行为意向和投入 | 2、3、13、15等 |

### 完整题目（18题）

> 请根据您的实际情况，选择最符合的选项。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我认为教师职业对促进人类个体发展尤其是学生的成长与发展十分重要 |
| 2 | 我精心准备日常教学活动材料，认真备课、上课、辅导学生 |
| 3 | 我会尝试开发一些新的课程内容，从事相关科研工作 |
| 4 | 我有与学生交流的意愿 |
| 5 | 我能和学生相处愉快，和孩子在一起让我感到很幸福 |
| 6 | 我认为自己适合做一名教师 |
| 7 | 从事教师职业能发挥我的才能 |
| 8 | 从事教师职业能够实现我的人生价值 |
| 9 | 我觉得教师工作是一项神圣的工作，为自己是一名教师而自豪 |
| 10 | 家人支持我的工作，对我的工作满意 |
| 11 | 我对自己现在的工作待遇和报酬很满意 |
| 12 | 我认为自己现在的工作压力比较大 |
| 13 | 在做自我介绍的时候，我乐意提到我是一名教师 |
| 14 | 我能积极主动地创造和谐的同事关系 |
| 15 | 为了维护学校的正常教学秩序，我会遵守那些非正式的制度 |
| 16 | 我在乎别人如何看待教师职业和教师群体 |
| 17 | 当看到或听到颂扬教师职业的话语时，我会有一种欣慰感 |
| 18 | 当有人无端指责教师群体时，我感到自己受到了侮辱 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 完全不符合 | 1分 |
| 基本不符合 | 2分 |
| 不确定 | 3分 |
| 基本符合 | 4分 |
| 完全符合 | 5分 |

- **各因子得分** = 该因子对应题目之和
- **总分** = 18题之和（18-90分），分数越高职业认同感越强
- **反向题**：第12题（"工作压力比较大"）计分时反向处理

### 结果解释

| 总分范围 | 职业认同水平 |
|---------|-----------|
| 18-36分 | 低职业认同 |
| 37-54分 | 中等职业认同 |
| 55-72分 | 较高职业认同 |
| 73-90分 | 高度职业认同 |

## 二、教师工作满意度量表

> **来源**：基于多维工作满意度结构的中国本土化量表，以下为综合多项学术研究的通用版本。代表性文献：许燕（1999）《中学教师工作满意度的调查研究》，北京师范大学学报。

### 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 中小学教师 |
| 量表结构 | 6个维度，共27题 |
| 答题形式 | 五级评分（1=非常不满意，5=非常满意） |

### 六个维度与题目

**维度一：工作本身（5题）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我喜欢我的教学工作 |
| 2 | 我的工作让我感到有挑战性和成就感 |
| 3 | 我能从工作中获得个人成长和发展 |
| 4 | 我的工作使我感到充实 |
| 5 | 我觉得教师工作是有意义的 |

**维度二：薪酬福利（4题）**

| 题号 | 题目内容 |
|-----|---------|
| 6 | 我对目前的工资水平感到满意 |
| 7 | 学校的福利待遇是合理的 |
| 8 | 我的薪酬与工作付出相匹配 |
| 9 | 学校的绩效奖励制度是公平的 |

**维度三：发展机会（5题）**

| 题号 | 题目内容 |
|-----|---------|
| 10 | 学校为我提供了足够的专业发展机会 |
| 11 | 学校支持我参加培训和进修 |
| 12 | 晋升制度是公平透明的 |
| 13 | 学校为我的职业发展提供了清晰的路径 |
| 14 | 我有机会承担更多有挑战性的工作 |

**维度四：学校管理（5题）**

| 题号 | 题目内容 |
|-----|---------|
| 15 | 学校领导对我的工作表示认可和肯定 |
| 16 | 学校的各项管理制度是合理的 |
| 17 | 领导能公平对待每位教师 |
| 18 | 学校决策时会听取教师的意见 |
| 19 | 我对学校的整体管理方式感到满意 |

**维度五：人际关系（4题）**

| 题号 | 题目内容 |
|-----|---------|
| 20 | 我与同事之间关系融洽 |
| 21 | 我与学生之间关系良好 |
| 22 | 我与学生家长之间关系良好 |
| 23 | 我在学校感到被他人尊重 |

**维度六：工作环境（4题）**

| 题号 | 题目内容 |
|-----|---------|
| 24 | 我的工作场所环境干净、安全、舒适 |
| 25 | 学校提供了足够的教学设施和资源 |
| 26 | 我的工作时间安排是合理的 |
| 27 | 工作中行政事务负担不过重 |

### 评分标准

- 五级评分：1=非常不满意，5=非常满意
- **各维度均分** = 该维度得分之和 ÷ 题目数
- **总体满意度均分** = 27题得分之和 ÷ 27

| 均分 | 满意度水平 |
|-----|---------|
| 1-2分 | 不满意 |
| 2-3分 | 一般 |
| 3-4分 | 较满意 |
| 4-5分 | 非常满意 |
