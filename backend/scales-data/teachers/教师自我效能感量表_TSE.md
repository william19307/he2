# 教师自我效能感量表（TSE）系列

---

## 一、教师效能感量表（Tschannen-Moran & Hoy，TSES）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 教师效能感量表（Teachers' Sense of Efficacy Scale） |
| 缩写 | TSES |
| 编制者 | Tschannen-Moran M & Hoy AW（2001年） |
| 中文版 | 已完成多个中文修订版，信效度良好 |
| 适用对象 | 中小学教师 |
| 题目数量 | 24题（完整版）/ 12题（简版） |
| 答题形式 | 九级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估教师在课堂管理、教学策略和学生参与方面的效能信念 |

### 量表结构（3个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 学生参与效能感 | 8题 | 激励学生参与学习的信心 |
| 教学策略效能感 | 8题 | 运用有效教学方法的信心 |
| 课堂管理效能感 | 8题 | 维持课堂秩序和行为管理的信心 |

### 完整题目（24题）

> 在您的教学工作中，您能做到以下事情的程度如何？

**学生参与效能感**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 能有多大程度控制那些会破坏课堂的行为？ |
| 2 | 能在多大程度上激励那些学习动机低的学生？ |
| 3 | 能在多大程度上让学生相信自己在学校中能做好？ |
| 4 | 能在多大程度上让学生注意课堂规矩？ |
| 5 | 能在多大程度上激励学生重视学业？ |
| 6 | 能在多大程度上帮助学生进行批判性思维？ |
| 7 | 能在多大程度上让学生在课堂上有创造性思维？ |
| 8 | 能在多大程度上帮助学生学会解决问题？ |

**教学策略效能感**

| 题号 | 题目内容 |
|-----|---------|
| 9 | 能在多大程度上为不同学习能力的学生提供不同解释？ |
| 10 | 能在多大程度上提供好的问题让学生思考？ |
| 11 | 能在多大程度上使用多种评估方法？ |
| 12 | 能在多大程度上为困难学生提供有效的替代解释？ |
| 13 | 能在多大程度上实施多种教学策略？ |
| 14 | 能在多大程度上针对学生需求调整课程？ |
| 15 | 能在多大程度上使用各种各样的教学资源？ |
| 16 | 能在多大程度上从技术上设计出合理的学生活动？ |

**课堂管理效能感**

| 题号 | 题目内容 |
|-----|---------|
| 17 | 能在多大程度上让学生遵守课堂规则？ |
| 18 | 能在多大程度上让学生相信他们能做好学业任务？ |
| 19 | 能在多大程度上建立课堂常规？ |
| 20 | 能在多大程度上控制破坏性行为？ |
| 21 | 能在多大程度上维持课堂纪律中的几名有问题的学生？ |
| 22 | 能在多大程度上应对学生的挑战？ |
| 23 | 能在多大程度上确定学生是否理解了所教授的内容？ |
| 24 | 能在多大程度上让家长参与孩子的教育？ |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 完全没有 | 1分 |
| 很少（2） | 2分 |
| 有一些（3） | 3分 |
| 有些程度（4） | 4分 |
| 中等程度（5） | 5分 |
| 相当程度（6） | 6分 |
| 大程度（7） | 7分 |
| 非常大程度（8） | 8分 |
| 极大程度 | 9分 |

- 各维度均分 = 该维度8题得分之和 ÷ 8
- 总均分越高，教学效能感越强
- Cronbach's α = 0.888（总量表）

## 二、俞国良教师教学效能感量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 俞国良、罗晓路（1999年，中国本土） |
| 适用对象 | 中小学教师 |
| 题目数量 | 27题 |
| 答题形式 | 六级评分 |
| 维度 | 一般教学效能感（10题）+ 个人教学效能感（17题） |

### 量表结构

| 维度 | 主要内容 |
|-----|---------|
| 一般教学效能感 | 对教师行为能否影响学生的整体信念 |
| 个人教学效能感 | 对自身教学能否有效影响学生的信念 |

### 代表性题目

**一般教学效能感**：
- 只要教学方法正确，任何学生都可以学好
- 教师的教学能力能决定学生的发展
- 儿童先天的能力是不变的，教育对此无能为力（反向）

**个人教学效能感**：
- 我能有效地教导好任何学生
- 我相信自己有能力提高学生的学习成绩
- 即使努力，我也无法使某些学生进步（反向）

### 评分标准

- 六级评分：1=完全不赞成，6=完全赞成
- 反向题按6-1计分
