# 教师综合心理健康筛查量表

> 以下量表来源于通用成人版，已在中小学教师群体中广泛使用，本文提供适用于教师群体的特别指引。

---

## 一、SCL-90 适用于教师的使用指引

SCL-90（症状自评量表）在教师心理健康研究中应用最广。详细题目见 `scales/SCL90_症状自评量表.md`。

### 教师群体参考常模

研究显示，**中国中小学教师SCL-90均分**高于全国成人常模，主要体现在：
- 强迫症状因子
- 人际关系敏感因子
- 抑郁因子

| 指标 | 教师参考值 | 全国成人常模 |
|-----|---------|-----------|
| 总均分 | 1.51±0.46 | 1.44±0.43 |
| 阳性项目数 | 26.2±18.9 | 24.9±18.4 |

### 教师使用特别说明

1. **评估频率**：建议学期初和学期末各测量一次
2. **重点关注**：抑郁、焦虑、强迫3个因子
3. **阳性阈值**：任一因子均分 ≥ 2分，且持续超过2周，需进一步评估

---

## 二、SAS/SDS 适用于教师的使用指引

SAS（焦虑自评量表）和SDS（抑郁自评量表）在教师中使用时请参考以下特别说明。详细题目见 `scales/SAS_SDS_焦虑抑郁自评量表.md`。

### 教师群体中的高危信号

| 因素 | 说明 |
|-----|-----|
| 学期末考试季 | 焦虑分通常升高 |
| 工作满意度低 | 抑郁分相关 |
| 职业倦怠 | 两个量表分数均可升高 |
| 年龄35-45岁女教师 | 抑郁发生率相对较高 |

## 三、教师心理健康综合评估方案建议

### 推荐评估工具组合

**快速筛查（约20分钟）**：
1. **DASS-21**（抑郁+焦虑+压力）：5分钟
2. **MBI-ES**（职业倦怠）：10分钟
3. **简单工作满意度题目**：5分钟

**全面评估（约50分钟）**：
1. **SCL-90**（全面心理症状）：20分钟
2. **MBI-ES**（职业倦怠）：10分钟
3. **教师工作压力量表**（压力源）：15分钟
4. **职业认同量表**（工作动力）：10分钟

**专项评估**（根据问题导向）：
| 问题 | 推荐量表 |
|-----|---------|
| 职业倦怠 | MBI-ES |
| 抑郁 | PHQ-9 / SDS |
| 焦虑 | GAD-7 / SAS |
| 工作压力 | 教师工作压力量表 |
| 工作-家庭冲突 | WIF/FIW问卷 |
| 职业动力不足 | 职业认同量表 + 工作满意度量表 |

## 四、教师团体辅导心理健康评估建议

### 重点关注风险因素

| 高风险特征 | 说明 |
|---------|-----|
| 教龄6-10年 | 职业高原期，倦怠风险增加 |
| 班主任 | 额外工作负担和情绪压力 |
| 单身或已婚离异 | 社会支持相对不足 |
| 所在学校管理差 | 组织支持感低 |
| 教授非核心科目 | 职业成就感相对低 |

### 教师心理健康促进建议

1. **个人层面**：规律休息、运动、社交
2. **管理层面**：减轻行政负担、增加专业支持
3. **学校层面**：建立教师心理关怀机制
4. **政策层面**：提高教师薪酬和社会地位
