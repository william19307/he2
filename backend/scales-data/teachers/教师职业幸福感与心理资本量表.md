# 教师职业幸福感与心理资本量表系列

---

## 一、中小学教师幸福感问卷（PERMA模型，15题）

> **来源**：基于Seligman（2011年）PERMA幸福理论编制的中小学教师版本，发表于心理量表网（xinlixue.cn）。
> 总量表 Cronbach's α = 0.950；各维度 α = 0.725-0.939；重测信度 = 0.936。

### 基本信息

| 属性 | 内容 |
|------|------|
| 理论基础 | Seligman PERMA幸福模型（积极情绪、投入、关系、意义、成就） |
| 适用对象 | 中小学教师 |
| 题目数量 | 15题（5个维度，各3题） |
| 答题形式 | 六级评分（1-6分） |
| 测试时间 | 约5分钟 |

### 量表结构

| 维度 | 英文 | 对应题号 | 主要内容 |
|-----|-----|---------|---------|
| 积极情绪 | Positive Emotion | 1、2、3 | 工作中的愉悦、温暖和稳定感 |
| 意义 | Meaning | 4、5、6 | 工作的价值感和方向感 |
| 成就 | Accomplishment | 7、8、9 | 对成就和贡献的感知 |
| 投入 | Engagement | 10、11、12 | 对工作的专注和沉浸程度 |
| 关系 | Relationships | 13、14、15 | 与同事的良好人际关系 |

### 完整题目（15题）

> 请根据您对以下描述的符合程度，选择最符合的选项。

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 你在工作中，情绪总是愉快的 | 积极情绪 |
| 2 | 你不论是在工作中，还是在生活中，始终都觉得内心是温暖的 | 积极情绪 |
| 3 | 你在工作时情绪是稳定的 | 积极情绪 |
| 4 | 你认为人生只有努力奋斗才会变得更有意义 | 意义 |
| 5 | 你通常会觉得你在生活中所做的事有价值并值得去做 | 意义 |
| 6 | 你能将每天最宝贵的时间和精力用来做有意义的事 | 意义 |
| 7 | 你有优于其他人的地方 | 成就 |
| 8 | 与同龄、同性别的其他人相比，你的贡献要远远大于他人 | 成就 |
| 9 | 你经常觉得，你的存在对这个世界有着长远、巨大和正面的影响 | 成就 |
| 10 | 你的主要精力与情感都投入在了工作上 | 投入 |
| 11 | 你备课、读书，或者做其他工作时，聚精会神，经常忘记时间 | 投入 |
| 12 | 讲课的时候经常很兴奋，忘记时间 | 投入 |
| 13 | 你很在意与同事们的关系 | 关系 |
| 14 | 你在工作中，拥有良好的人际关系 | 关系 |
| 15 | 你愿意与家人或者同事谈论自己的问题 | 关系 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 非常不符合 | 1分 |
| 不符合 | 2分 |
| 比较不符合 | 3分 |
| 比较符合 | 4分 |
| 符合 | 5分 |
| 非常符合 | 6分 |

- **各维度得分** = 该维度3题得分之和（3-18分）
- **总分** = 15题得分之和（15-90分），分数越高幸福感越强

### 信度与效度

| 指标 | 数值 |
|-----|-----|
| 总量表 Cronbach's α | 0.950 |
| 各维度 α | 0.725-0.939 |
| 重测信度 | 0.936 |
| 各维度重测信度 | 0.717-0.912 |
| 结构方程模型拟合 | χ²/df=4.551, RMSEA=0.06, GFI=0.952 |

## 二、教师心理资本量表（PCQ-24）

> **来源**：Luthans等（2007年）PCQ量表；中文版已广泛发表于学术文献。

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | Luthans FB等（2007年），中文版多机构修订 |
| 适用对象 | 教师及职业人群 |
| 题目数量 | 24题（HERO模型，各6题） |
| 答题形式 | 六级评分（1=强烈不同意，6=强烈同意） |

### HERO模型（4个维度）

| 维度 | 含义 | 对应题号 |
|-----|-----|---------|
| 希望（Hope） | 实现目标的意志力和路径思维 | 1-6 |
| 效能（Efficacy） | 完成工作任务的信心 | 7-12 |
| 韧性（Resilience） | 从挫折和逆境中恢复的能力 | 13-18 |
| 乐观（Optimism） | 对工作结果的积极归因和期待 | 19-24 |

### 完整题目（24题）

**希望（第1-6题）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我对自己的职业目标充满信心，能够努力实现 |
| 2 | 如果工作中被逼到绝路，我能想出很多方法走出困境 |
| 3 | 工作中遇到困难，我能独立想出多种解决方案 |
| 4 | 此刻，我正在朝着自己的工作目标努力 |
| 5 | 在工作上，我认为自己非常成功 |
| 6 | 我能想到很多实现工作目标的方法 |

**效能感（第7-12题）**

| 题号 | 题目内容 |
|-----|---------|
| 7 | 我有信心向学生讲解课程内容 |
| 8 | 我有信心代表学校在正式会议上做报告或发言 |
| 9 | 我有信心为改进学校的教学策略做出贡献 |
| 10 | 我有信心联系与工作相关的专业领域的人士 |
| 11 | 我有信心向学生家长解释自己的教学方法 |
| 12 | 我有信心为解决工作中的复杂问题做出贡献 |

**韧性（第13-18题）**

| 题号 | 题目内容 |
|-----|---------|
| 13 | 在工作中，我能同时处理多项任务 |
| 14 | 在工作中遇到挫折时，我可以很快恢复正常 |
| 15 | 我能较好地处理工作中遇到的困难事件 |
| 16 | 我不会为工作上的困难感到长时间沮丧 |
| 17 | 以前的工作经历让我变得更加坚强 |
| 18 | 在学校里，我一般能处理任何发生的事情 |

**乐观（第19-24题）**

| 题号 | 题目内容 |
|-----|---------|
| 19 | 在工作中，很多事情都在向好的方向发展 |
| 20 | 关于工作，我总是往好的一面想 |
| 21 | 在工作中，我总是以积极的态度看待事情 |
| 22 | 我对工作上将要发生的事情感到乐观 |
| 23 | 事情不确定时，我通常期待最好的结果 |
| 24 | 我对自己的工作和生活充满期待 |

### 评分标准

- 六级评分：1=强烈不同意，6=强烈同意
- **各维度得分** = 6题之和（6-36分）
- **心理资本总分** = 4个维度之和（24-144分）

## 三、Utrecht工作投入量表（UWES）

> **来源**：Schaufeli WB等（2002年）；中文版已广泛学术发表。

### 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 教师及各职业人群 |
| 题目数量 | 17题（简版9题） |
| 答题形式 | 七级评分（0-6分） |

### 三个维度

| 维度 | 对应题号 | 内容 |
|-----|---------|-----|
| 活力（Vigor） | 1、4、8、12、15、17 | 工作时的精力和坚韧 |
| 奉献（Dedication） | 2、5、7、10、13 | 工作的热情和意义感 |
| 专注（Absorption） | 3、6、9、11、14、16 | 完全专注于工作的状态 |

### 完整题目（17题）

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 在工作中，我感到自己精力充沛 | 活力 |
| 2 | 我认为我的工作很有意义和价值 | 奉献 |
| 3 | 时间在工作中飞速流逝 | 专注 |
| 4 | 在工作中，我感到自己强健而充满活力 | 活力 |
| 5 | 我对自己所从事的工作充满热情 | 奉献 |
| 6 | 当我工作时，我会忘记周围的一切 | 专注 |
| 7 | 我的工作给我带来鼓舞和激励 | 奉献 |
| 8 | 早晨起来，我感到很想去工作 | 活力 |
| 9 | 当我工作非常投入时，我感到很愉悦 | 专注 |
| 10 | 我为从事这份工作而感到自豪 | 奉献 |
| 11 | 我完全投入到工作中 | 专注 |
| 12 | 在工作中，我能坚持，即使事情进展不顺利 | 活力 |
| 13 | 我的工作是富于挑战性的 | 奉献 |
| 14 | 在工作时，我很难使自己分心 | 专注 |
| 15 | 在工作中，我的意志很坚强 | 活力 |
| 16 | 很难从工作中分离出来 | 专注 |
| 17 | 在工作中，我总是坚持不懈，即使事情不顺利 | 活力 |

### 评分标准

| 选项 | 频率 | 得分 |
|-----|-----|-----|
| 从不 | 从来没有 | 0分 |
| 几乎没有 | 一年几次 | 1分 |
| 有时 | 一月几次 | 2分 |
| 经常 | 每月几次 | 3分 |
| 很经常 | 每周几次 | 4分 |
| 几乎总是 | 每天 | 5分 |
| 总是 | 每天数次 | 6分 |

- 各维度均分越高，工作投入度越高
- **UWES简版（9题）**：第1、2、3、4、5、6、7、8、9题
