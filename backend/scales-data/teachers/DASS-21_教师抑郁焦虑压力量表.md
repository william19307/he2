# DASS-21 抑郁-焦虑-压力量表（教师版）

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 抑郁焦虑压力量表21项（Depression Anxiety and Stress Scale-21） |
| 缩写 | DASS-21 |
| 编制者 | Lovibond SH & Lovibond PF（1995年，澳大利亚） |
| 中文版 | 多家机构完成中国教师版标准化（2023年验证） |
| 适用对象 | 中小学教师（及成人群体） |
| 题目数量 | 21题（DASS-42的简版） |
| 答题形式 | 四级评分 |
| 测试时间 | 约5-8分钟 |
| 用途 | 快速评估教师的抑郁、焦虑和压力状态 |

---

## 量表结构（3个分量表，各7题）

| 分量表 | 主要评估内容 |
|-------|------------|
| 抑郁（D） | 情绪低落、兴趣丧失、绝望、自我否定 |
| 焦虑（A） | 自主神经激活、骨骼肌效应、情境性焦虑、主观体验 |
| 压力（S） | 持续性紧张、易激惹、难以放松、低耐受性 |

---

## 完整题目（21题）

> 请根据**过去一周**内您的实际感受，选择最符合的选项。

| 题号 | 题目内容 | 分量表 |
|-----|---------|-------|

## 评分标准

### 各题评分方式（四级评分）

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 0 | 根本不符合我的情况 | 0分 |
| 1 | 有一些符合，或者偶尔有 | 1分 |
| 2 | 相当符合，或者有相当一段时间有 | 2分 |
| 3 | 非常符合，或者大多数时间都有 | 3分 |

### 计分方法

DASS-21各分量表得分 **× 2** = 换算为DASS-42等量分（便于与原版常模比较）

| 分量表 | DASS-21题目 | 原始分范围 | 换算分范围 |
|-------|-----------|---------|---------|
| 抑郁（D） | 3、5、10、13、16、17、21 | 0-21分 | 0-42分 |
| 焦虑（A） | 2、4、7、9、15、19、20 | 0-21分 | 0-42分 |
| 压力（S） | 1、6、8、11、12、14、18 | 0-21分 | 0-42分 |

### 结果解释（使用换算分，即原始分×2）

#### 抑郁分量表

| 换算分 | 严重程度 |
|-------|---------|
| 0-9 | 正常 |
| 10-13 | 轻度 |
| 14-20 | 中度 |
| 21-27 | 重度 |
| ≥ 28 | 极重 |

#### 焦虑分量表

| 换算分 | 严重程度 |
|-------|---------|
| 0-7 | 正常 |
| 8-9 | 轻度 |
| 10-14 | 中度 |
| 15-19 | 重度 |
| ≥ 20 | 极重 |

#### 压力分量表

| 换算分 | 严重程度 |
|-------|---------|
| 0-14 | 正常 |
| 15-18 | 轻度 |
| 19-25 | 中度 |
| 26-33 | 重度 |
| ≥ 34 | 极重 |

## 适用于教师群体的注意事项

1. 中国教师样本的研究显示，DASS-21 对教师抑郁的测量效果优于焦虑和压力分量表
2. 教师群体的常模参照值通常高于一般成人，解读时应考虑职业背景
3. 量表为**筛查工具**，中度及以上结果建议转介专业人员
4. 可每学期使用，追踪教师心理状态变化趋势
