## 一、儿童分离焦虑量表（SAAS-R）

> **来源**：综合国内研究编制，基于DSM-5分离焦虑障碍的核心症状维度。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 小学至初中（6-14岁）|
| 题目数量 | 12题 |
| 答题形式 | 三级评分 |

### 完整题目（12题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 当我离开家或离开父母时，我感到非常担心 |
| 2 | 我担心父母会发生意外或不测 |
| 3 | 我拒绝单独睡觉，需要家人陪伴 |
| 4 | 我害怕一个人待在家里 |
| 5 | 当我要与父母分开时，我会感到非常难过 |
| 6 | 我担心迷路或失散，与家人失去联系 |
| 7 | 上学前，我会感到非常不安或不舒服 |
| 8 | 我做过关于家人发生意外的噩梦 |
| 9 | 当要离开家时，我会出现头痛、肚子痛等不舒服 |
| 10 | 我跟随家人走，因为害怕独处 |
| 11 | 我无法独自参加朋友的活动 |
| 12 | 与家人分开时，我难以集中注意力学习 |

### 评分标准
- 三级评分：0=从不，1=有时，2=经常
- **总分** = 12题之和（0-24分）

| 总分 | 分离焦虑程度 |
|-----|-----------|
| 0-6分 | 轻微或无 |
| 7-12分 | 中度 |
| 13-24分 | 重度，建议评估 |
