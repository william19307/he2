## 三、患者健康问卷躯体症状量表（PHQ-15）

> **来源**：Kroenke K等（2002年）编制；中文版已广泛发表。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 15题 |
| 评定时限 | 过去四周 |
| 用途 | 筛查躯体化症状 |

### 完整题目（15题）

> 在过去**四周内**，您受以下问题困扰的程度？

| 题号 | 躯体症状 |
|-----|---------|
| 1 | 胃痛 |
| 2 | 背痛 |
| 3 | 上肢、腿或关节疼痛 |
| 4 | 经期痛或其他月经问题（女性）|
| 5 | 头痛 |
| 6 | 胸痛 |
| 7 | 头晕 |
| 8 | 昏厥 |
| 9 | 心跳加快或心悸 |
| 10 | 气短 |
| 11 | 性交期间疼痛或其他性功能问题 |
| 12 | 便秘、大便稀或腹泻 |
| 13 | 恶心、胀气或消化不良 |
| 14 | 精力不足或疲倦 |
| 15 | 睡眠问题 |

### 评分标准
- 三级评分：0=没有/完全没有困扰，1=有一些，2=烦扰很多
- **总分** = 15题之和（0-30分）

| 总分 | 躯体化严重程度 |
|-----|------------|
| 0-4 | 极少 |
| 5-9 | 轻度 |
| 10-14 | 中度 |
| ≥15 | 重度 |
