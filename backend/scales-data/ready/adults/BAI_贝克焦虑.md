## 一、贝克焦虑量表（BAI）

> **来源**：Beck AT等（1988年）编制；中文版已验证，α = 0.91-0.95。

### 基本信息
| 适用对象 | 成人（≥17岁）|
|---------|-----|
| 题目数量 | 21题 |
| 答题形式 | 四级评分（0-3分）|
| 评定时限 | 过去一周 |

### 完整题目（21题）

> 在过去**一周内**，包括今天，您对以下症状的困扰程度？

| 题号 | 症状描述 |
|-----|---------|
| 1 | 麻木或刺痛感 |
| 2 | 感到发热 |
| 3 | 腿部颤抖 |
| 4 | 无法放松 |
| 5 | 担心发生最坏的事情 |
| 6 | 头晕或感到轻飘飘 |
| 7 | 心跳加速或心悸 |
| 8 | 坐立不安 |
| 9 | 被吓到 |
| 10 | 神经紧张 |
| 11 | 窒息感 |
| 12 | 手发抖 |
| 13 | 颤抖 |
| 14 | 害怕失控 |
| 15 | 呼吸困难 |
| 16 | 害怕快要死去 |
| 17 | 恐慌 |
| 18 | 消化不良或腹部不适 |
| 19 | 昏厥感 |
| 20 | 脸红 |
| 21 | 出汗（不是因为暑热）|

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 无 | 0分 |
| 轻度，无多大烦扰 | 1分 |
| 中度，不舒适但尚能忍受 | 2分 |
| 重度，只能勉强忍受 | 3分 |

| 总分 | 焦虑程度 |
|-----|---------|
| **0-7分** | 无焦虑 |
| **8-15分** | 轻度焦虑 |
| **16-25分** | 中度焦虑 |
| **26-63分** | 重度焦虑 |
