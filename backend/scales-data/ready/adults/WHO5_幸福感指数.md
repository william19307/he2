## 二、WHO-5 幸福感指数

> **来源**：世界卫生组织（WHO，1998年）；中文版已广泛使用，α = 0.87-0.93。免费公开使用。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 5题 |
| 评定时限 | 过去两周 |
| 用途 | 积极心理健康快速筛查，也可用于抑郁筛查 |

### 完整题目（5题）

> 在**过去两周内**，您有以下感受的频率是？

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我感到心情愉快和高兴 |
| 2 | 我感到平静、放松 |
| 3 | 我感到积极主动、充满活力 |
| 4 | 我感到醒来精力充沛，精神焕发 |
| 5 | 我的日常生活充满了我感兴趣的事物 |

### 评分标准

| 选项 | 频率 | 得分 |
|-----|-----|-----|
| 所有时间 | 一直 | 5分 |
| 大多数时间 | >14天中的半数 | 4分 |
| 超过一半时间 | 7-14天 | 3分 |
| 不到一半时间 | <7天 | 2分 |
| 有时 | 极少 | 1分 |
| 从未 | 从未 | 0分 |

- **总分** = 5题之和（0-25分）
- **百分比分数** = 总分 × 4（0-100%），分数越高心理健康越好

| 总分 | 解释 |
|-----|-----|
| **< 13分** | 疑似抑郁，建议PHQ-9进一步评估 |
| **13-20分** | 幸福感一般，可能存在情绪问题 |
| **≥ 21分** | 幸福感较好 |
