# 认知与应对量表：认知融合、Brief COPE、Beck绝望感

---

## 一、认知融合问卷（CFQ）

> **来源**：Gillanders DT等（2014年）编制；中文版已验证，α = 0.94。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 7题 |
| 答题形式 | 七级评分 |
| 用途 | 评估对想法的过度认同和融合程度（ACT核心概念）|

### 完整题目（7题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我的想法给我的生活造成了很多困扰 |
| 2 | 我与我的想法纠缠在一起 |
| 3 | 在面对困难的想法和感受时，我很难做到我最在乎的事 |
| 4 | 我很难与令人不安的想法和感受保持距离 |
| 5 | 我与我的想法融为一体 |
| 6 | 我花了很多精力与我的想法和感受斗争 |
| 7 | 与我的想法斗争阻止了我活在当下 |

### 评分标准
- 七级评分：1=从不，7=总是
- **总分** = 7题之和（7-49分），分数越高认知融合越严重

## 二、简易应对方式问卷（Brief COPE，成人版）

> **来源**：Carver CS（1997年）编制；中文版已发表，α = 0.72-0.85（各维度）。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 28题（14种应对方式各2题）|
| 答题形式 | 四级评分 |
| 用途 | 评估应对压力的多种策略 |

### 量表结构（14种应对方式）

| 应对方式 | 题号 | 类型 |
|---------|-----|-----|
| 主动应对 | 2、7 | 问题聚焦 |
| 计划 | 14、25 | 问题聚焦 |
| 积极重新理解 | 12、17 | 情绪聚焦 |
| 接受 | 20、24 | 情绪聚焦 |
| 幽默 | 18、28 | 情绪聚焦 |
| 宗教 | 22、27 | 情绪聚焦 |
| 使用情感支持 | 5、15 | 支持寻求 |
| 使用工具性支持 | 10、23 | 支持寻求 |
| 自我分心 | 1、19 | 回避 |
| 否认 | 3、8 | 回避 |
| 发泄情绪 | 9、21 | 回避 |
| 物质使用 | 4、11 | 回避 |
| 行为脱离 | 6、16 | 回避 |
| 自责 | 13、26 | 回避 |

### 完整题目（28题）

> 请评估当您遇到压力时，您在多大程度上采用以下方式。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我把注意力集中在工作或其他活动上来分散注意力 |
| 2 | 我集中自己的努力来做一些关于这种情况的事 |
| 3 | 我告诉自己"这不是真实的" |
| 4 | 我使用酒精或其他药物让自己感觉更好 |
| 5 | 我从别人那里获得情感支持 |
| 6 | 我放弃了试图处理这种情况 |
| 7 | 我采取行动试图让情况变得更好 |
| 8 | 我拒绝相信这已经发生了 |
| 9 | 我说一些让自己的感受表达出来的话 |
| 10 | 我从别人那里获得帮助或建议 |
| 11 | 我使用酒精或其他药物帮助我渡过这个难关 |
| 12 | 我尝试从不同角度来看待这种情况 |
| 13 | 我批评自己 |
| 14 | 我尝试想出一个处理这种情况的策略 |
| 15 | 我从别人那里获得安慰和理解 |
| 16 | 我放弃了试图解决这个问题 |
| 17 | 我寻找这种情况好的方面 |
| 18 | 我拿这种情况开玩笑 |
| 19 | 我做一些其他的活动来少想一会儿这种情况 |
| 20 | 我接受了这种情况的现实 |
| 21 | 我以某种方式表达出我的负面感受 |
| 22 | 我寻求上帝或精神力量的帮助 |
| 23 | 我从别人那里获得建议或帮助 |
| 24 | 我学习与这种情况共存 |
| 25 | 我仔细思考如何处理这种情况 |
| 26 | 我自责于这件事的发生 |
| 27 | 我祈祷或冥想 |
| 28 | 我拿自己的情况开玩笑 |

### 评分标准
- 四级评分：0=根本没有，1=有一点，2=经常，3=非常多地
- 各应对方式得分 = 对应2题之和（0-6分）

## 三、Beck绝望感量表（BHS）

> **来源**：Beck AT等（1974年）编制；中文版已验证，α = 0.93。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 20题 |
| 答题形式 | 是/否 |
| 用途 | 评估对未来的消极预期（自杀风险预测因素）|

### 完整题目（20题）

> 请选择"是"或"否"，描述您目前的感受。

| 题号 | 题目内容 | 计分方向 |
|-----|---------|---------|
| 1 | 我对未来充满希望与热情 | 否=1 |
| 2 | 我不如自己努力摆脱困境，因为根本没有什么可做的 | 是=1 |
| 3 | 当事情进展不顺利时，我总是知道会好转的 | 否=1 |
| 4 | 我无法想象10年后的生活会是什么样的 | 是=1 |
| 5 | 我有足够的时间来完成我最渴望做的事情 | 否=1 |
| 6 | 在未来，我预计自己会成功地达到我最重要的目标 | 否=1 |
| 7 | 我的未来对我来说看起来很暗淡 | 是=1 |
| 8 | 我碰巧比其他人更好运，从好事中获得良好利益 | 否=1 |
| 9 | 我无法获得良好的生活 | 是=1 |
| 10 | 我过去的经历很好地为我的未来做了准备 | 否=1 |
| 11 | 我看到的未来除了不愉快之外什么都没有 | 是=1 |
| 12 | 我不指望得到我真正想要的东西 | 是=1 |
| 13 | 当我展望未来时，我期待比现在更幸福 | 否=1 |
| 14 | 事情从来不会如我所愿 | 是=1 |
| 15 | 我对未来充满信心 | 否=1 |
| 16 | 我永远得不到我想要的，所以想要任何东西都是愚蠢的 | 是=1 |
| 17 | 我几乎不可能在未来获得真正的满足 | 是=1 |
| 18 | 未来对我来说看起来很模糊和不确定 | 是=1 |
| 19 | 我可以期待比坏事更多的好事 | 否=1 |
| 20 | 为了未来的得到，努力是毫无意义的，因为我可能无论如何都无法得到我想要的 | 是=1 |

### 评分标准
- 每题答"是"或"否"，按上表计1分
- **总分** = 20题中符合计分方向的题目数（0-20分）

| 总分 | 绝望感程度 |
|-----|---------|
| 0-3分 | 无/轻微 |
| 4-8分 | 轻度 |
| 9-14分 | 中度 |
| ≥15分 | 重度（自杀风险增高）|

> ⚠️ BHS与自杀意念相关，高分（≥9分）需立即进行安全评估。
