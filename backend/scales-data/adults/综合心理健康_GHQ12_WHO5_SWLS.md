# 综合心理健康与幸福感量表：GHQ-12、WHO-5、SWLS、Flourishing Scale

---

## 一、一般健康问卷（GHQ-12）

> **来源**：Goldberg DP（1972年）编制；中文版已广泛验证，α = 0.82-0.89。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 12题 |
| 答题形式 | 四级评分（两种计分法）|
| 用途 | 非精神病性心理障碍快速筛查 |

### 完整题目（12题）

> 请与您平常的状态相比，评估最近**几周内**您的情况。

| 题号 | 题目内容 | 计分 |
|-----|---------|-----|
| 1 | 在做什么事情的时候，能集中精神吗？ | 反向 |
| 2 | 有由于过分担心而失眠的情况吗？ | 正向 |
| 3 | 觉得自己是有用的人吗？ | 反向 |
| 4 | 觉得自己有决断力吗？ | 反向 |
| 5 | 总是处于紧张状态吗？ | 正向 |
| 6 | 觉得自己不能解决问题吗？ | 正向 |
| 7 | 能享受日常活动吗？ | 反向 |
| 8 | 能够面对您所面临的问题吗？ | 反向 |
| 9 | 感到痛苦、忧虑吗？ | 正向 |
| 10 | 失去自信了吗？ | 正向 |
| 11 | 觉得自己是没有价值的人吗？ | 正向 |
| 12 | 觉得所有的事情都顺利吗？ | 反向 |

### 两种计分法

**GHQ计分法（二分法）**：
- 选"比平时更好/一样好"=0分；选"比平时差/差很多"=1分
- **总分** = 12题之和（0-12分）
- 阳性阈值：≥3分（或≥4分）提示存在心理健康问题

**李克特计分法（0-1-2-3分）**：
- 比平时更好=0，和平时一样=1，比平时差=2，差很多=3
- **总分** = 12题之和（0-36分）

## 二、WHO-5 幸福感指数

> **来源**：世界卫生组织（WHO，1998年）；中文版已广泛使用，α = 0.87-0.93。免费公开使用。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 5题 |
| 评定时限 | 过去两周 |
| 用途 | 积极心理健康快速筛查，也可用于抑郁筛查 |

### 完整题目（5题）

> 在**过去两周内**，您有以下感受的频率是？

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我感到心情愉快和高兴 |
| 2 | 我感到平静、放松 |
| 3 | 我感到积极主动、充满活力 |
| 4 | 我感到醒来精力充沛，精神焕发 |
| 5 | 我的日常生活充满了我感兴趣的事物 |

### 评分标准

| 选项 | 频率 | 得分 |
|-----|-----|-----|
| 所有时间 | 一直 | 5分 |
| 大多数时间 | >14天中的半数 | 4分 |
| 超过一半时间 | 7-14天 | 3分 |
| 不到一半时间 | <7天 | 2分 |
| 有时 | 极少 | 1分 |
| 从未 | 从未 | 0分 |

- **总分** = 5题之和（0-25分）
- **百分比分数** = 总分 × 4（0-100%），分数越高心理健康越好

| 总分 | 解释 |
|-----|-----|
| **< 13分** | 疑似抑郁，建议PHQ-9进一步评估 |
| **13-20分** | 幸福感一般，可能存在情绪问题 |
| **≥ 21分** | 幸福感较好 |

## 三、生活满意度量表（SWLS）

> **来源**：Diener E等（1985年）编制；中文版已广泛验证，公共领域可自由使用，α = 0.78-0.87。

### 基本信息
| 适用对象 | 成人（也适用于中学生以上）|
|---------|-----|
| 题目数量 | 5题 |
| 答题形式 | 七级评分 |

### 完整题目（5题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我生活中的大多数方面接近我的理想 |
| 2 | 我的生活条件很好 |
| 3 | 我对自己的生活感到满意 |
| 4 | 迄今为止我在生活中得到了想得到的重要东西 |
| 5 | 如果我能回头重走人生之路，我几乎不想改变任何东西 |

### 评分标准

七级评分：1=完全不同意，7=完全同意

**总分** = 5题之和（5-35分）

| 总分 | 生活满意度 |
|-----|---------|
| 31-35分 | 非常满意 |
| 26-30分 | 满意 |
| 21-25分 | 较为满意 |
| 20分 | 中性（既不满意也不不满意）|
| 15-19分 | 有些不满意 |
| 10-14分 | 不满意 |
| 5-9分 | 非常不满意 |

## 四、繁荣量表（Flourishing Scale）

> **来源**：Diener E等（2010年）编制；中文版已发表，α = 0.87。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 8题 |
| 答题形式 | 七级评分 |
| 用途 | 评估心理幸福感的多个核心维度 |

### 完整题目（8题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我过着有目标和意义的生活 |
| 2 | 我的社会关系是有支持性的和有意义的 |
| 3 | 我参与并对社区的幸福感感兴趣 |
| 4 | 我是有能力和高效率的，完成对我很重要的任务 |
| 5 | 我是一个好人，过着好的生活 |
| 6 | 我对自己的未来充满乐观 |
| 7 | 人们尊重我 |
| 8 | 我一直在参与并对社区感兴趣 |

### 评分标准

七级评分：1=强烈不同意，7=强烈同意

**总分** = 8题之和（8-56分），分数越高，整体幸福感越强
