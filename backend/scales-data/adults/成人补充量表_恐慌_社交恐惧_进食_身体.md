# 成人补充量表：恐慌、社交恐惧、进食态度、身体意象

---

## 一、LSAS 社交恐惧量表（Liebowitz Social Anxiety Scale）

> **来源**：Liebowitz MR（1987年）编制；中文版已验证，α = 0.97。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 24情境（恐惧程度+回避程度各24题，共48评分）|
| 用途 | 评估社交焦虑的全面程度 |

### 24个社交情境

**表现性情境（13项）**

| 题号 | 情境 |
|-----|-----|
| 1 | 在有权威的人面前讲话 |
| 2 | 在人群中参与小组活动 |
| 3 | 在公众场所进餐 |
| 4 | 在公众场所喝饮料 |
| 5 | 与陌生人交谈 |
| 6 | 成为被观察的一方 |
| 7 | 演出、表演或发言 |
| 8 | 接受电话指导 |
| 9 | 参加聚会 |
| 10 | 公开写字 |
| 11 | 在亲密关系中 |
| 12 | 就面试 |
| 13 | 小组发言 |

**社交性情境（11项）**

| 题号 | 情境 |
|-----|-----|
| 14 | 拒绝别人的要求 |
| 15 | 与不很熟识的人谈话 |
| 16 | 在电话里与不熟识的人交谈 |
| 17 | 与陌生人约会 |
| 18 | 去表达不同意见或不满 |
| 19 | 与权威人士面谈 |
| 20 | 参加社交聚会 |
| 21 | 注视他人表演 |
| 22 | 回到商店退货 |
| 23 | 出席热闹的晚会 |
| 24 | 在无充分准备的情况下小组发言 |

### 评分标准

每个情境评定两次：

**恐惧程度**：0=无，1=轻度，2=中度，3=重度

**回避程度**：0=从不，1=偶尔（<33%），2=经常（33-67%），3=总是（>67%）

**总分** = 恐惧总分（0-72）+ 回避总分（0-72）（0-144分）

| 总分 | 社交焦虑程度 |
|-----|-----------|
| 0-54分 | 无/轻微 |
| 55-64分 | 轻度 |
| 65-79分 | 中度 |
| 80-94分 | 中重度 |
| ≥95分 | 重度 |

## 二、进食态度量表（EAT-26）

> **来源**：Garner DM等（1982年）编制；中文版已发表，α = 0.87-0.90。

### 基本信息
| 适用对象 | 成人（≥16岁）|
|---------|-----|
| 题目数量 | 26题（3个分量表）|
| 答题形式 | 六级评分 |
| 用途 | 进食障碍倾向筛查 |

### 量表结构（3个分量表）
| 分量表 | 题号 | 内容 |
|-------|-----|-----|
| 节食 | 1-13 | 对食物的过度关注和节食行为 |
| 暴食与食物预占 | 14-19 | 对食物的强迫性思维和暴食 |
| 口腔控制 | 20-26 | 他人对进食行为的控制 |

### 代表性题目（各维度示例）

**节食（第1-13题示例）**
- 我非常害怕超重
- 我不吃我认为会使我发胖的食物
- 我在公共场所饮食时会感到不舒适

**暴食与食物预占（第14-19题示例）**
- 我发现自己脑子里尽是食物
- 我感到一旦开始进食就无法停止

**口腔控制（第20-26题示例）**
- 我努力节食
- 我觉得别人会希望我能多吃一点
- 进食后我有一种呕吐的冲动

### 评分标准
- 六级评分：总是=3，非常频繁=2，频繁=1，有时=0，很少=0，从不=0
- **总分** = 26题之和（0-78分），≥20分为阳性，建议进一步评估

## 三、身体意象量表（BIS，16题）

> **来源**：Hopwood CJ等（2010年）编制；中文版已发表，α = 0.82-0.89。

### 完整题目（16题）

> 以下描述反映您对自身身体的感受程度？

| 题号 | 题目内容 | 计分 |
|-----|---------|-----|
| 1 | 我对自己的外表感到自信 | 正向 |
| 2 | 我非常在意我的外表 | 正向 |
| 3 | 我对自己的体重感到不满意 | 反向 |
| 4 | 我认为我的身材很好 | 正向 |
| 5 | 我对自己的体型感到不舒服 | 反向 |
| 6 | 我通常对自己的外表感到满意 | 正向 |
| 7 | 我对自己的身体感到骄傲 | 正向 |
| 8 | 我希望我的外表不一样 | 反向 |
| 9 | 我对自己的体型感到满意 | 正向 |
| 10 | 我觉得我的体重刚好合适 | 正向 |
| 11 | 我常常感到我的某些身体部位需要改变 | 反向 |
| 12 | 我对自己面部的外观感到满意 | 正向 |
| 13 | 我认为我的外表很吸引人 | 正向 |
| 14 | 我对自己皮肤的外观感到满意 | 正向 |
| 15 | 我觉得我的外貌有很多需要改善的地方 | 反向 |
| 16 | 我对自己的整体外观感到满意 | 正向 |

### 评分标准
- 五级评分：1=强烈不同意，5=强烈同意
- 反向题取反
- **总分** = 16题之和（16-80分），分数越高身体意象越积极
