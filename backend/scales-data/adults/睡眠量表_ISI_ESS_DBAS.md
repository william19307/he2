# 睡眠评估量表系列

---

## 一、失眠严重程度指数量表（ISI）

> **来源**：Morin CM等（1993/2011年）编制；中文版已在多项研究中验证，α = 0.74-0.83。WHO、美国睡眠医学会等广泛推荐使用。
> **状态**：🔶 学术完整

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 成人（≥18岁）|
| 题目数量 | 7题 |
| 答题形式 | 五级评分（0-4分）|
| 评定时限 | 过去两周 |
| 测试时间 | 约3-5分钟 |
| 总分范围 | 0-28分 |

### 完整题目（7题）

| 题号 | 题目内容 | 评分选项 |
|-----|---------|---------|
| 1 | 入睡困难的程度 | 无=0 / 轻度=1 / 中度=2 / 重度=3 / 极重度=4 |
| 2 | 维持睡眠困难（夜间醒来）的程度 | 无=0 / 轻度=1 / 中度=2 / 重度=3 / 极重度=4 |
| 3 | 早醒问题的程度 | 无=0 / 轻度=1 / 中度=2 / 重度=3 / 极重度=4 |
| 4 | 您对目前睡眠模式的满意程度 | 非常满意=0 / 满意=1 / 中等=2 / 不满意=3 / 非常不满意=4 |
| 5 | 您认为您的睡眠问题在多大程度上影响了日常功能（如疲乏、情绪、工作效率）| 无=0 / 轻微=1 / 有些=2 / 很多=3 / 极大影响=4 |
| 6 | 您对睡眠问题有多担心/苦恼 | 完全没有=0 / 轻微=1 / 有些=2 / 很多=3 / 非常=4 |
| 7 | 您认为其他人在多大程度上能注意到您的睡眠问题影响了生活质量 | 完全没有=0 / 轻微=1 / 有些=2 / 很多=3 / 非常=4 |

### 评分标准

| 总分 | 失眠分类 |
|-----|---------|
| **0-7分** | 无临床意义的失眠 |
| **8-14分** | 轻度失眠（亚临床）|
| **15-21分** | 中度临床失眠 |
| **22-28分** | 重度临床失眠 |

## 二、爱普沃斯嗜睡量表（ESS）

> **来源**：Johns MW（1991年）编制；中文修订版（ESSR）信度 α = 0.82，适用于成人及中学生以上群体。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 成人（可用于≥15岁）|
| 题目数量 | 8题 |
| 答题形式 | 四级评分（0-3分）|
| 测试时间 | 约3分钟 |

### 完整题目（8题）

> 在以下情境中，您**打盹或入睡**的可能性有多大？（即使近期没有经历以下情境，请估计可能性。）

| 题号 | 情境 |
|-----|-----|
| 1 | 坐着阅读书刊时 |
| 2 | 看电视时 |
| 3 | 在公共场所坐着不动时（如剧场、会议）|
| 4 | 连续乘坐汽车一小时（无间断）|
| 5 | 条件允许的情况下，下午躺下休息时 |
| 6 | 坐着与人谈话时 |
| 7 | 未饮酒的午餐后安静地坐着 |
| 8 | 遇堵车，在停车的几分钟里 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 从不打瞌睡 | 0分 |
| 轻度可能打瞌睡 | 1分 |
| 中度可能打瞌睡 | 2分 |
| 很可能打瞌睡 | 3分 |

**总分** = 8题之和（0-24分）

| 总分 | 嗜睡程度 |
|-----|---------|
| 0-10分 | 正常 |
| 11-15分 | 过度嗜睡，建议关注 |
| ≥16分 | 严重嗜睡，建议就医 |

## 三、睡眠功能失调性信念量表（DBAS-16）

> **来源**：Morin CM等（2007年）编制；中文版已在中国人群验证，α = 0.80。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 16题 |
| 答题形式 | 十一级评分（0-10分）|
| 用途 | 评估对睡眠的不合理信念和期望 |

### 完整题目（16题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我需要睡满8小时才能在白天感觉神清气爽，表现良好 |
| 2 | 当我有一两晚睡不好的时候，我知道这会影响接下来一周的日程安排 |
| 3 | 当我睡眠不好时，我担心我可能会精神崩溃 |
| 4 | 我觉得我对睡眠没有任何掌控力 |
| 5 | 我担心，如果两三天不睡觉，我可能会患上神经疾病 |
| 6 | 当我感到沮丧或忧虑时，我的睡眠必然会受到影响 |
| 7 | 我相信失眠是由于一种化学失衡造成的 |
| 8 | 我认为当我失眠时我可能会感觉很糟糕，因为我什么事都做不了 |
| 9 | 我永远无法预测我能否好好睡一觉 |
| 10 | 没有服用安眠药我就无法睡个好觉 |
| 11 | 当我晚上睡得不好时，我知道这会影响到我第二天做事 |
| 12 | 为了保持白天的功能并保持健康，我相信一定需要睡一整夜 |
| 13 | 我认为失眠是一种医学疾病 |
| 14 | 当我很难入睡时，就继续躺在床上，努力继续尝试 |
| 15 | 我担心长期缺乏睡眠会对健康产生严重影响 |
| 16 | 我很少能重新入睡，在夜里醒来之后 |

### 评分标准
- 0-10分十一级评分：0=完全不同意，10=完全同意
- **总分** = 16题之和（0-160分），分数越高，功能失调信念越强
- 高分提示对睡眠存在不合理认知，可能是失眠的维持因素

## 四、睡眠前过度觉醒量表（PSAS，简版）

> **来源**：Nicassio PM等（1985年）编制；中文版已验证。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 16题（认知8题+躯体8题）|
| 答题形式 | 五级评分 |
| 用途 | 评估睡前的认知和躯体过度觉醒 |

### 代表性题目
| 维度 | 题目示例 |
|-----|---------|
| 认知觉醒 | 我的大脑停不下来 |
| 认知觉醒 | 我在想明天需要做什么 |
| 认知觉醒 | 我在担心各种各样的问题 |
| 躯体觉醒 | 我感觉到心跳 |
| 躯体觉醒 | 我感觉肌肉紧张 |
| 躯体觉醒 | 我感觉到我的呼吸 |

### 评分标准
- 五级评分：1=完全不符合，5=完全符合
- 认知觉醒分高提示睡前思维活跃；躯体觉醒分高提示睡前生理紧张
