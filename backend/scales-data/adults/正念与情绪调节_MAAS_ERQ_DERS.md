# 正念与情绪调节量表：MAAS、ERQ、DERS-16

---

## 一、正念注意觉知量表（MAAS）

> **来源**：Brown KW & Ryan RM（2003年）编制；周仁来等（2012年）修订中文版，发表于《中国临床心理学》，α = 0.89。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 15题（全部反向计分）|
| 答题形式 | 六级评分 |
| 用途 | 评估日常生活中对当下的注意觉知程度 |

### 完整题目（15题）

> 请根据**过去一个月**内的实际情况，选择最符合您的频率。
> **注意：分数越高代表正念程度越低（需反向计算总分）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 在做某件事时，我可能感受到情绪困扰，但直到一段时间后才意识到这件事 |
| 2 | 我不小心把东西打翻或把东西弄破 |
| 3 | 我发现自己机械地完成任务，并不真正意识到我在做什么 |
| 4 | 我很快速地从一地方移动到另一地方，不太在意一路上发生了什么 |
| 5 | 我忽略身体的不适或紧绷感，因为我正专注于另外一件事情 |
| 6 | 我忘了别人的名字，几乎是刚认识就忘了 |
| 7 | 我好像是在用"自动驾驶"模式运作，并没有意识到自己在做什么 |
| 8 | 我为了完成任务而冲冲忙忙，并不太注意任务本身 |
| 9 | 我太专注于我想达成的目标，以致完全没注意到完成目标的当下 |
| 10 | 我机械地做工作或任务，心思都在别处 |
| 11 | 我发现自己在听别人说话时，同时也在做其他事 |
| 12 | 我开车来到目的地，却不记得行经的路程 |
| 13 | 我发现自己沉浸在过去或未来，而没有注意到现在 |
| 14 | 我发现自己做事情心不在焉 |
| 15 | 我进食时，并没有注意到自己在吃什么 |

### 评分标准

| 选项 | 得分（原始）| 反向后 |
|-----|---------|-------|
| 几乎总是 | 1 | 6 |
| 很频繁 | 2 | 5 |
| 比较频繁 | 3 | 4 |
| 不太频繁 | 4 | 3 |
| 很少 | 5 | 2 |
| 几乎从不 | 6 | 1 |

- 将所有15题原始分取反（6-原始分+1）后求均值
- **正念均分** = 15题反向后均值（1-6分），分数越高正念越强

## 二、情绪调节问卷（ERQ）

> **来源**：Gross JJ & John OP（2003年）编制；中文版（ERQ-CRV）已发表，α = 0.85（认知重评）、0.77（表达抑制）。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 10题（认知重评6题+表达抑制4题）|
| 答题形式 | 七级评分 |

### 完整题目（10题）

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 当我想感受一些积极的情绪（如快乐或高兴）时，我会改变自己思考问题的角度 | 认知重评 |
| 2 | 我不会表露自己的情绪 | 表达抑制 |
| 3 | 当我想少感受一些消极的情绪（如悲伤或愤怒）时，我会改变自己思考问题的角度 | 认知重评 |
| 4 | 当感受到积极情绪时，我会很小心地不让它们表露出来 | 表达抑制 |
| 5 | 在面对压力情境时，我会使自己以一种有助于保持平静的方式来考虑它 | 认知重评 |
| 6 | 我控制自己情绪的方式是不表达它们 | 表达抑制 |
| 7 | 当我想多感受一些积极的情绪时，我会改变自己对情境的考虑方式 | 认知重评 |
| 8 | 我会通过改变对情境的考虑方式来控制自己的情绪 | 认知重评 |
| 9 | 当感受到消极的情绪时，我确保不会表露它们 | 表达抑制 |
| 10 | 当我想少感受一些消极的情绪时，我会改变自己对情境的考虑方式 | 认知重评 |

### 评分标准

七级评分：1=非常不同意，7=非常同意（无反向题）

- **认知重评分** = 第1、3、5、7、8、10题均值（1-7分）
- **表达抑制分** = 第2、4、6、9题均值（1-7分）

## 三、情绪失调量表简版（DERS-16）

> **来源**：Gratz KL & Roemer L（2004年）编制36题版；Bjureberg等（2016年）简化为16题版；中文版已验证，α = 0.87。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 16题（5个维度）|
| 答题形式 | 五级评分 |
| 用途 | 评估情绪调节困难程度 |

### 量表结构（5个维度）
| 维度 | 题号 | 内容 |
|-----|-----|-----|
| 非接纳 | 1、2、3 | 对负性情绪反应的不接纳 |
| 目标 | 4、5、6 | 情绪困扰时难以专注目标 |
| 冲动 | 7、8、9 | 情绪困扰时难以控制行为 |
| 策略 | 10、11、12、13 | 情绪调节策略有限 |
| 清晰度 | 14、15、16 | 情绪意识和清晰度 |

### 完整题目（16题）

> 当您感到不安时，以下情况的发生频率？

| 题号 | 题目内容 |
|-----|---------|
| 1 | 当我感到情绪不好时，我对此感到恼怒 |
| 2 | 当我感到情绪不好时，我对有这样的感受感到羞耻 |
| 3 | 当我感到情绪不好时，我感到难为情 |
| 4 | 当我感到情绪不好时，我很难专注于其他事情 |
| 5 | 当我感到情绪不好时，我很难完成任务 |
| 6 | 当我感到情绪不好时，我很难做任何事情 |
| 7 | 当我感到情绪不好时，我变得失控 |
| 8 | 当我感到情绪不好时，我感到自己可能会失控 |
| 9 | 当我感到情绪不好时，我会做一些事情，后来为此感到遗憾 |
| 10 | 当我感到情绪不好时，我认为我最终会感觉到非常糟糕，很久很久 |
| 11 | 当我感到情绪不好时，我觉得我将来一定要等很长时间才能感觉更好 |
| 12 | 当我感到情绪不好时，我认为这种感觉会持续很长时间 |
| 13 | 当我感到情绪不好时，我认为我在生活中会变成很糟糕的状态 |
| 14 | 当我感到情绪不好时，我不清楚自己感受到了什么 |
| 15 | 当我感到情绪不好时，我难以理解自己的感受 |
| 16 | 当我感到情绪不好时，我感到混乱 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 几乎从不（0-10%）| 1分 |
| 有时（11-35%）| 2分 |
| 大约一半时候（36-65%）| 3分 |
| 大多数时候（66-90%）| 4分 |
| 几乎总是（91-100%）| 5分 |

**总分** = 16题之和（16-80分），分数越高情绪调节困难越多
