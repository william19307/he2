# 物质使用与成瘾量表：AUDIT、CAGE、DAST-10、IAT

## 一、酒精使用障碍识别测验（AUDIT）

> **来源**：世界卫生组织（WHO，1989年）编制；中文版已发表，灵敏度0.96，特异度0.85。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 10题 |
| 评定时限 | 过去一年 |
| 用途 | 筛查危险饮酒、有害饮酒和酒精依赖 |

### 完整题目（10题）

| 题号 | 题目内容 | 选项 |
|-----|---------|-----|
| 1 | 您的饮酒频率如何？ | 从不(0)/每月≤1次(1)/每月2-4次(2)/每周2-3次(3)/每周≥4次(4) |
| 2 | 饮酒时，您通常会喝几杯？（1杯=含10g纯酒精）| 1-2杯(0)/3-4杯(1)/5-6杯(2)/7-9杯(3)/≥10杯(4) |
| 3 | 每次喝6杯以上的情况有多频繁？ | 从不(0)/不到每月(1)/每月(2)/每周(3)/几乎每天(4) |
| 4 | 过去一年中，您是否发现一旦开始喝酒就很难停下来？ | 从不(0)/不到每月(1)/每月(2)/每周(3)/几乎每天(4) |
| 5 | 过去一年中，您是否因为饮酒而不能完成日常工作？ | 从不(0)/不到每月(1)/每月(2)/每周(3)/几乎每天(4) |
| 6 | 过去一年中，大量饮酒后，您是否需要在早上喝酒才能正常生活？ | 从不(0)/不到每月(1)/每月(2)/每周(3)/几乎每天(4) |
| 7 | 过去一年中，您是否在饮酒后感到内疚或后悔？ | 从不(0)/不到每月(1)/每月(2)/每周(3)/几乎每天(4) |
| 8 | 过去一年中，您是否因为饮酒而不能记起前一晚发生的事情？ | 从不(0)/不到每月(1)/每月(2)/每周(3)/几乎每天(4) |
| 9 | 您或他人是否因为您的饮酒而受到伤害？ | 没有(0)/有，但不是过去一年(2)/是，在过去一年(4) |
| 10 | 家人、朋友、医生或其他医疗工作者是否对您饮酒表示担忧，或建议您减少饮酒？ | 没有(0)/有，但不是过去一年(2)/是，在过去一年(4) |

### 评分标准

**总分** = 10题之和（0-40分）

| 总分 | 分类 |
|-----|-----|
| 0-7分 | 低风险 |
| 8-15分 | 危险饮酒 |
| 16-19分 | 有害饮酒 |
| ≥20分 | 可能酒精依赖 |

## 二、CAGE 问卷

> **来源**：Ewing JA（1984年）编制；4题快速筛查，临床广泛使用。

### 完整题目（4题）

| 题号 | 题目（英文首字母缩写）| 题目内容 |
|-----|---------------------|---------|
| 1 | **C**ut down | 您是否曾经感到应该减少饮酒？ |
| 2 | **A**nnoyed | 别人对您的饮酒习惯进行批评是否令您感到恼怒？ |
| 3 | **G**uilty | 您是否曾经对您的饮酒感到内疚？ |
| 4 | **E**ye-opener | 您是否曾经在早晨一起床就需要喝酒来稳定神经或消除宿醉？ |

### 评分标准
- 是=1分，否=0分
- ≥2分：提示可能存在酒精问题，建议进一步评估

## 三、毒品滥用筛查测试（DAST-10）

> **来源**：Skinner HA（1982年）编制，Yudko等（2007年）简化为10题；中文版已发表。

### 完整题目（10题）

> 以下问题涉及**过去12个月内**（非酒精类）毒品、处方药滥用或非法药物的使用。（是=1/否=0）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 您是否曾因非医疗目的使用毒品或使用超量的处方药？ |
| 2 | 您是否曾经滥用过多于一种的药物？ |
| 3 | 您是否在1个月内无法停止使用毒品？ |
| 4 | 您是否在滥用药物后曾经昏迷或过量？ |
| 5 | 您的家人或朋友是否曾经对您的毒品使用感到担忧？ |
| 6 | 您是否曾因为吸毒问题给家人造成麻烦？ |
| 7 | 您是否曾经因吸毒而忽视自己的家人？ |
| 8 | 您是否曾经在违禁情况下（如开车时）吸毒？ |
| 9 | 您是否曾经因吸毒而有过法律方面的问题？ |
| 10 | 您是否曾经在工作时吸过毒？ |

### 评分标准
- 是=1分，否=0分
- **总分** = 10题之和（0-10分）

| 总分 | 分类 |
|-----|-----|
| 0分 | 无问题 |
| 1-2分 | 低度风险 |
| 3-5分 | 中度风险 |
| 6-8分 | 重度风险 |
| 9-10分 | 严重风险 |

## 四、智能手机成瘾量表简版（SAS-SV）

> **来源**：Kwon M等（2013年）编制；中文版已验证，α = 0.88。

### 完整题目（10题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 因为使用智能手机，我错过了计划好的工作 |
| 2 | 使用智能手机使我很难集中精力 |
| 3 | 在使用智能手机时，我感到手腕或颈部疼痛 |
| 4 | 我无法忍受不使用智能手机 |
| 5 | 当我没有智能手机时，我感到不耐烦和烦躁 |
| 6 | 我时刻想着使用智能手机 |
| 7 | 即使影响了我的日常生活，我也不会减少使用智能手机 |
| 8 | 我一直检查手机以确认我没有错过任何通知 |
| 9 | 我使用智能手机的时间比预计的要长 |
| 10 | 家人或朋友告诉我，我使用智能手机的时间太长了 |

### 评分标准
- 六级评分：1=强烈不同意，6=强烈同意
- **总分** = 10题之和（10-60分）
- 阳性阈值：男≥31分，女≥33分（韩国常模；中国常模建议≥33分）
