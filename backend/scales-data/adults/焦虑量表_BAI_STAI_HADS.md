# 成人焦虑评估量表：BAI、STAI、HADS

---

## 一、贝克焦虑量表（BAI）

> **来源**：Beck AT等（1988年）编制；中文版已验证，α = 0.91-0.95。

### 基本信息
| 适用对象 | 成人（≥17岁）|
|---------|-----|
| 题目数量 | 21题 |
| 答题形式 | 四级评分（0-3分）|
| 评定时限 | 过去一周 |

### 完整题目（21题）

> 在过去**一周内**，包括今天，您对以下症状的困扰程度？

| 题号 | 症状描述 |
|-----|---------|
| 1 | 麻木或刺痛感 |
| 2 | 感到发热 |
| 3 | 腿部颤抖 |
| 4 | 无法放松 |
| 5 | 担心发生最坏的事情 |
| 6 | 头晕或感到轻飘飘 |
| 7 | 心跳加速或心悸 |
| 8 | 坐立不安 |
| 9 | 被吓到 |
| 10 | 神经紧张 |
| 11 | 窒息感 |
| 12 | 手发抖 |
| 13 | 颤抖 |
| 14 | 害怕失控 |
| 15 | 呼吸困难 |
| 16 | 害怕快要死去 |
| 17 | 恐慌 |
| 18 | 消化不良或腹部不适 |
| 19 | 昏厥感 |
| 20 | 脸红 |
| 21 | 出汗（不是因为暑热）|

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 无 | 0分 |
| 轻度，无多大烦扰 | 1分 |
| 中度，不舒适但尚能忍受 | 2分 |
| 重度，只能勉强忍受 | 3分 |

| 总分 | 焦虑程度 |
|-----|---------|
| **0-7分** | 无焦虑 |
| **8-15分** | 轻度焦虑 |
| **16-25分** | 中度焦虑 |
| **26-63分** | 重度焦虑 |

## 二、状态-特质焦虑量表（STAI）

> **来源**：Spielberger CD等（1983年）编制；李文利等修订中文版，α = 0.90。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 40题（状态焦虑20题 + 特质焦虑20题）|
| 答题形式 | 四级评分 |

### 第一部分：状态焦虑量表（S-AI，20题）

> 请根据**此刻（现在）**的感受选择。

| 题号 | 题目内容 | 计分 |

| 4 | 我感到压力很大 | 正向 |
| 5 | 我感到轻松 | 反向 |
| 6 | 我感到心烦意乱 | 正向 |
| 7 | 我现在很担心可能会遇到不幸 | 正向 |
| 8 | 我感到休息得很充分 | 反向 |
| 9 | 我感到焦虑 | 正向 |
| 10 | 我感到安慰 | 反向 |
| 11 | 我有自信心 | 反向 |
| 12 | 我觉得神经过敏 | 正向 |
| 13 | 我是紧张不安的 | 正向 |
| 14 | 我感到难以下决心 | 正向 |
| 15 | 我是放松的 | 反向 |
| 16 | 我感到心满意足 | 反向 |
| 17 | 我是忧虑的 | 正向 |
| 18 | 我感到兴奋而慌乱 | 正向 |
| 19 | 我感到快乐 | 反向 |
| 20 | 我感到愉快 | 反向 |

### 第二部分：特质焦虑量表（T-AI，20题）

> 请根据**通常情况下（一般）**的感受选择。

| 题号 | 题目内容 | 计分 |
|-----|---------|-----|
| 21 | 我感到愉快 | 反向 |
| 22 | 我容易疲劳 | 正向 |
| 23 | 我感到想哭 | 正向 |
| 24 | 我希望能像别人一样开心 | 正向 |
| 25 | 我因为想得太多而失眠 | 正向 |
| 26 | 我感到心安 | 反向 |
| 27 | 我保持冷静、从容、沉着 | 反向 |
| 28 | 我感到困难重重、无法克服 | 正向 |
| 29 | 我过于担心不重要的事情 | 正向 |
| 30 | 我是快乐的 | 反向 |
| 31 | 我容易想到烦恼的念头 | 正向 |
| 32 | 我缺乏自信心 | 正向 |
| 33 | 我感到安全 | 反向 |
| 34 | 我容易作出决定 | 反向 |
| 35 | 我感到不合适 | 正向 |
| 36 | 我是满足的 | 反向 |
| 37 | 一些不重要的念头总是缠着我，使我烦恼 | 正向 |
| 38 | 我对失望感到很烦恼，以致不能从脑子里丢开 | 正向 |
| 39 | 我是镇定的 | 反向 |
| 40 | 当我想到我目前的事情和利益时，我就陷入紧张状态 | 正向 |

### 评分标准

**状态焦虑（S-AI）**：
- 正向题：几乎没有=1，有些=2，中等程度=3，非常明显=4
- 反向题：反向计分

**特质焦虑（T-AI）**：
- 正向题：几乎没有=1，有时=2，经常=3，几乎总是=4
- 反向题：反向计分

每部分总分：20-80分，分数越高焦虑越强

## 三、医院焦虑抑郁量表（HADS）

> **来源**：Zigmond AS & Snaith RP（1983年）编制；中文版已发表，α = 0.81-0.85。

### 基本信息
| 适用对象 | 成人（医疗机构患者及普通成人）|
|---------|-----|
| 题目数量 | 14题（焦虑7题+抑郁7题）|
| 答题形式 | 四级评分 |

### 完整题目（14题）

**焦虑分量表（奇数题，共7题）**

| 题号 | 题目内容 | 选项（0-3分）|
|-----|---------|-----------|
| 1 | 我感到紧张或"心悸" | 根本没有/偶尔/很多时候/大多数时候 |
| 3 | 我感到惊恐，好像要发生什么可怕的事 | 完全没有/偶尔/很多时候/非常肯定且非常严重 |
| 5 | 令人烦恼的念头不断在脑海中转 | 只是偶尔/有时/经常但不太烦扰我/大多数时候且非常烦扰我 |
| 7 | 我可以安静地坐着，感觉放松 | 肯定/通常/不经常/根本不 |
| 9 | 我感到像是肚子里有只蝴蝶在飞（内心不安）| 根本没有/偶尔/很多时候/非常频繁 |
| 11 | 我感到坐立不安，就好像我必须不断活动 | 根本没有/不是很多/相当/非常 |
| 13 | 我突然感到恐慌 | 根本没有/不太/相当多/非常频繁 |

**抑郁分量表（偶数题，共7题）**

| 题号 | 题目内容 |
|-----|---------|
| 2 | 我仍然可以欣赏我以前喜欢的事物 |
| 4 | 我能笑，并且看到事物有趣的一面 |
| 6 | 我感到快乐 |
| 8 | 我感到一切都在迟缓下来 |
| 10 | 我对我的外表不再感兴趣了 |
| 12 | 我期待享受事物 |
| 14 | 我能享受好书或好的广播或电视节目 |

### 评分标准

每题0-3分，各分量表总分0-21分：

| 总分 | 焦虑/抑郁程度 |
|-----|------------|
| 0-7分 | 正常 |
| 8-10分 | 轻度 |
| 11-14分 | 中度 |
| 15-21分 | 重度 |
