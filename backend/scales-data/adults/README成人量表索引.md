## 快速评分速查

| 量表 | 阳性/问题阈值 |
|-----|------------|
| ISI（失眠）| ≥8分 轻度；≥15分 中度；≥22分 重度 |
| ESS（嗜睡）| ≥11分 可能嗜睡；≥16分 严重嗜睡 |
| BDI-II | ≥14分 轻度；≥20分 中度；≥29分 重度 |
| CES-D | ≥16分 有抑郁症状 |
| BAI | ≥8分 轻度焦虑；≥26分 重度 |
| HADS | 每分量表≥8分 需关注 |
| GHQ-12 | ≥3分（GHQ计分法）有问题 |
| WHO-5 | <13分 建议PHQ-9进一步筛查 |
| SWLS | ≤15分 不满意；≥26分 满意 |
| PCL-5（PTSD）| ≥33分 可能PTSD |
| OCI-R（强迫）| ≥21分 阳性 |
| AUDIT（酒精）| ≥8分 危险饮酒；≥20分 可能依赖 |
| GDS-15（老年抑郁）| ≥5分 轻度；≥12分 重度 |
| MMSE | <27分 可能认知障碍 |
