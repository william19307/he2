# 成人综合量表：压力、认知、老年心理、身心健康

---

## 一、感知压力量表（PSS-14）

> **来源**：Cohen S等（1983年）编制；中文版已广泛发表，α = 0.78-0.86。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 14题（简版PSS-10已在学生文件中收录）|
| 评定时限 | 过去一个月 |

### 完整题目（14题）

> 在过去**一个月内**，以下情况发生的频率？

| 题号 | 题目内容 | 计分 |
|-----|---------|-----|
| 1 | 因意外发生的事而感到沮丧 | 正向 |
| 2 | 感到无法控制生活中重要的事情 | 正向 |
| 3 | 感到紧张和有压力 | 正向 |
| 4 | 成功地处理了令人讨厌的生活烦恼 | 反向 |
| 5 | 感到自己有效地应对了生活中的重要变化 | 反向 |
| 6 | 对处理个人问题的能力没有信心 | 正向 |
| 7 | 能够按自己的意愿行事 | 反向 |
| 8 | 感到无力控制事情的方式 | 正向 |
| 9 | 因为超出控制的事情而愤怒 | 正向 |
| 10 | 感到困难在增加，以至于无法克服 | 正向 |
| 11 | 为意料之外发生的事情烦恼 | 正向 |
| 12 | 感到能够掌控烦恼事件 | 反向 |
| 13 | 能够管理自己消极的感受 | 反向 |
| 14 | 感到事情正按照自己的意愿发展 | 反向 |

**评分标准**（五级）：0=从不，1=几乎不，2=有时，3=相当多，4=很频繁
- 反向题：4-原分
- **总分** = 14题（含反向处理）之和（0-56分），分数越高压力越大

## 二、老年抑郁量表（GDS-15，简版）

> **来源**：Yesavage JA等（1983年）编制；中文版已验证，α = 0.85。

### 基本信息
| 适用对象 | 老年人（≥60岁）|
|---------|-----|
| 题目数量 | 15题（原版30题的简版）|
| 答题形式 | 是/否 |
| 用途 | 老年人抑郁筛查 |

### 完整题目（15题）

| 题号 | 题目内容 | 阳性答案（计1分）|
|-----|---------|--------------|
| 1 | 总的来说，您对自己的生活感到满意吗？ | 否 |
| 2 | 您是否减少了许多以往的兴趣和爱好？ | 是 |
| 3 | 您是否感到生活空虚？ | 是 |
| 4 | 您是否经常感到厌倦？ | 是 |
| 5 | 您对未来充满希望吗？ | 否 |
| 6 | 您是否有一些无法摆脱的令人烦恼的想法？ | 是 |
| 7 | 您大多数时候精神状态良好吗？ | 否 |
| 8 | 您是否害怕会有坏事降临到您身上？ | 是 |
| 9 | 您大多数时间是否感到幸福？ | 否 |
| 10 | 您是否经常感到无助？ | 是 |
| 11 | 您是否经常感到不安或坐立不安？ | 是 |
| 12 | 您是否宁愿待在家里，而不是去做一些新的事情？ | 是 |
| 13 | 您是否经常担忧将来的事情？ | 是 |
| 14 | 您觉得您的记忆力比大多数人差吗？ | 是 |
| 15 | 您觉得活着是美好的吗？ | 否 |

**总分** = 阳性答案数（0-15分）

| 总分 | 解释 |
|-----|-----|
| 0-4分 | 正常，无抑郁 |
| 5-8分 | 轻度抑郁 |
| 9-11分 | 中度抑郁 |
| 12-15分 | 重度抑郁 |

## 三、认知测试（MMSE，Mini-Mental State Examination）

> **来源**：Folstein MF等（1975年）编制；中文版已发表，广泛用于临床。

### 基本信息
| 适用对象 | 成人（怀疑认知障碍者，≥55岁多用）|
|---------|-----|
| 评估形式 | 面谈式认知测试 |
| 题目数量 | 30道小题 |
| 测试时间 | 约10分钟 |
| 用途 | 痴呆和认知障碍筛查 |

### 评估内容（5个领域，共30分）

| 领域 | 满分 | 内容 |
|-----|-----|-----|
| 定向力 | 10分 | 时间定向（5题）+ 地点定向（5题）|
| 即时记忆 | 3分 | 复述3个词语 |
| 注意力和计算 | 5分 | 从100起每次减7（连续5次）|
| 延迟回忆 | 3分 | 回忆前述3个词语 |
| 语言能力 | 9分 | 命名、复述、阅读、书写、描绘 |

**代表性评估题目**：
- 今天是哪一年？哪个月？几号？星期几？什么季节？（各1分）
- 这里是什么省？什么市？什么区/县？什么地方？这是第几层楼？（各1分）
- 我说3个词，请您重复一遍并记住：苹果、手表、国旗（各1分）
- 请从100减去7，再减去7，共减5次（每次正确得1分）

### 评分标准

| 总分 | 文化背景参考 |
|-----|-----------|
| **≥27分** | 正常 |
| **20-26分** | 轻度认知障碍 |
| **10-19分** | 中度认知障碍 |
| **<10分** | 重度认知障碍 |

> 注：低教育程度者标准应相应调低（小学：≥17分；文盲：≥14分）

## 四、蒙特利尔认知评估量表（MoCA）

> **来源**：Nasreddine ZS等（2005年）编制；中文版已广泛使用。

### 基本信息
| 适用对象 | 成人（≥55岁，或有认知障碍风险者）|
|---------|-----|
| 评估形式 | 面谈式认知测试 |
| 总分 | 30分 |
| 测试时间 | 约10分钟 |

### 评估领域

| 领域 | 分值 |
|-----|-----|
| 视觉空间/执行功能（连线、立方体、画钟）| 5分 |
| 命名（3幅动物图）| 3分 |
| 记忆（学习5个词）| — |
| 注意力（数字广度、连续减法、字母敲击）| 6分 |
| 语言（重复句子、词语流利性）| 3分 |
| 抽象能力（相似性）| 2分 |
| 延迟回忆 | 5分 |
| 定向力 | 6分 |

**评分标准**：
- **≥26分**：正常
- **<26分**：提示轻度认知障碍（教育年限≤12年可加1分）
