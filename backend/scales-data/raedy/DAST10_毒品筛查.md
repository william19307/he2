## 三、毒品滥用筛查测试（DAST-10）

> **来源**：Skinner HA（1982年）编制，Yudko等（2007年）简化为10题；中文版已发表。

### 完整题目（10题）

> 以下问题涉及**过去12个月内**（非酒精类）毒品、处方药滥用或非法药物的使用。（是=1/否=0）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 您是否曾因非医疗目的使用毒品或使用超量的处方药？ |
| 2 | 您是否曾经滥用过多于一种的药物？ |
| 3 | 您是否在1个月内无法停止使用毒品？ |
| 4 | 您是否在滥用药物后曾经昏迷或过量？ |
| 5 | 您的家人或朋友是否曾经对您的毒品使用感到担忧？ |
| 6 | 您是否曾因为吸毒问题给家人造成麻烦？ |
| 7 | 您是否曾经因吸毒而忽视自己的家人？ |
| 8 | 您是否曾经在违禁情况下（如开车时）吸毒？ |
| 9 | 您是否曾经因吸毒而有过法律方面的问题？ |
| 10 | 您是否曾经在工作时吸过毒？ |

### 评分标准
- 是=1分，否=0分
- **总分** = 10题之和（0-10分）

| 总分 | 分类 |
|-----|-----|
| 0分 | 无问题 |
| 1-2分 | 低度风险 |
| 3-5分 | 中度风险 |
| 6-8分 | 重度风险 |
| 9-10分 | 严重风险 |
