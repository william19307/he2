# CDI 儿童抑郁量表

> **来源**：Kovacs M（1992年）编制；中文版已在国内多项研究中验证。
> **题目来源**：来自公开学术平台（medsci.cn、wjx.cn），与发表版本高度一致。

## 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 7-17岁儿童及青少年 |
| 题目数量 | 27题 |
| 答题形式 | 三选一（每题3个陈述，选最符合的，0/1/2分）|
| 测试时间 | 约15分钟 |
| 用途 | 评估儿童抑郁症状严重程度 |

---

## 评分标准

- **A选项** = 0分（无/最轻）
- **B选项** = 1分（中间）
- **C选项** = 2分（最重）

**总分** = 27题之和（0-54分）

| 总分 | 年龄段 | 说明 |
|-----|-------|-----|
| ≥ 13分 | 6-8岁 | 提示存在抑郁症状 |
| ≥ 17分 | 9-12岁 | 提示存在抑郁症状 |
| ≥ 19分 | 12岁以上 | 提示可能存在抑郁 |
| ≥ 20分 | 通用 | 建议进一步临床评估 |
