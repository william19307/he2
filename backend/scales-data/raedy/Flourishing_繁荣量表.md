## 四、繁荣量表（Flourishing Scale）

> **来源**：Diener E等（2010年）编制；中文版已发表，α = 0.87。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 8题 |
| 答题形式 | 七级评分 |
| 用途 | 评估心理幸福感的多个核心维度 |

### 完整题目（8题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我过着有目标和意义的生活 |
| 2 | 我的社会关系是有支持性的和有意义的 |
| 3 | 我参与并对社区的幸福感感兴趣 |
| 4 | 我是有能力和高效率的，完成对我很重要的任务 |
| 5 | 我是一个好人，过着好的生活 |
| 6 | 我对自己的未来充满乐观 |
| 7 | 人们尊重我 |
| 8 | 我一直在参与并对社区感兴趣 |

### 评分标准

七级评分：1=强烈不同意，7=强烈同意

**总分** = 8题之和（8-56分），分数越高，整体幸福感越强
