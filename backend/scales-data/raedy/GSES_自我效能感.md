## 一、一般自我效能感量表（GSES）

> **来源**：Schwarzer R & Jerusalem M（1993年）编制；王才康等（2001年）修订中文版，发表于《应用心理学》。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 中学生至成人（≥13岁）|
| 题目数量 | 10题（单维度）|
| 答题形式 | 四级评分 |
| 测试时间 | 约5分钟 |
| 信度 | Cronbach's α = 0.87；重测信度 = 0.83 |

### 完整题目（10题）

> 请根据您的实际情况选择最符合的选项。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 如果我尽力去做的话，我总是能够解决问题的 |
| 2 | 即使别人反对我，我仍有办法取得我所要的 |
| 3 | 对我来说，坚持理想和达成目标是轻而易举的 |
| 4 | 我自信能有效地应付任何突如其来的事情 |
| 5 | 以我的才智，我定能应付意料之外的情况 |
| 6 | 如果我付出必要的努力，我一定能解决大多数的难题 |
| 7 | 我能冷静地面对困难，因为我信赖自己处理问题的能力 |
| 8 | 面对一个难题时，我通常能找到几个解决方法 |
| 9 | 有麻烦的时候，我通常能想到一些应付的方法 |
| 10 | 无论什么事在我身上发生，我都能够应付自如 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 完全不正确 | 1分 |
| 有点正确 | 2分 |
| 多数正确 | 3分 |
| 完全正确 | 4分 |

- **总分** = 10题之和（10-40分）

| 总分 | 自我效能水平 |
|-----|-----------|
| 10-20分 | 低水平 |
| 21-30分 | 中等水平 |
| 31-40分 | 高水平 |
