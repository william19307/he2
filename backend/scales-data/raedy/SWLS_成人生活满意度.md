## 三、生活满意度量表（SWLS）

> **来源**：Diener E等（1985年）编制；中文版已广泛验证，公共领域可自由使用，α = 0.78-0.87。

### 基本信息
| 适用对象 | 成人（也适用于中学生以上）|
|---------|-----|
| 题目数量 | 5题 |
| 答题形式 | 七级评分 |

### 完整题目（5题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我生活中的大多数方面接近我的理想 |
| 2 | 我的生活条件很好 |
| 3 | 我对自己的生活感到满意 |
| 4 | 迄今为止我在生活中得到了想得到的重要东西 |
| 5 | 如果我能回头重走人生之路，我几乎不想改变任何东西 |

### 评分标准

七级评分：1=完全不同意，7=完全同意

**总分** = 5题之和（5-35分）

| 总分 | 生活满意度 |
|-----|---------|
| 31-35分 | 非常满意 |
| 26-30分 | 满意 |
| 21-25分 | 较为满意 |
| 20分 | 中性（既不满意也不不满意）|
| 15-19分 | 有些不满意 |
| 10-14分 | 不满意 |
| 5-9分 | 非常不满意 |
