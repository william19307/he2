# PHQ-9 抑郁症筛查量表 & GAD-7 焦虑症筛查量表

## PHQ-9 抑郁症筛查量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 患者健康问卷抑郁量表（Patient Health Questionnaire-9） |
| 缩写 | PHQ-9 |
| 编制者 | Kroenke K, Spitzer RL, Williams JB（2001年） |
| 适用对象 | 初中生至成人（≥12岁，部分研究建议≥13岁） |
| 题目数量 | 9题 |
| 答题形式 | 四级评分 |
| 测试时间 | 约5分钟 |
| 用途 | 抑郁症筛查和严重程度评估 |

### 完整题目（9题）

> 在过去的**两周**里，您有多少时间受到以下任何问题困扰？

| 题号 | 题目内容 |
|-----|---------|
| 1 | 做事时提不起劲或没有兴趣 |
| 2 | 感到心情低落、沮丧或绝望 |
| 3 | 入睡困难、睡不安稳或睡眠过多 |
| 4 | 感觉疲倦或没有活力 |
| 5 | 食欲不振或吃太多 |
| 6 | 觉得自己很糟糕——或者觉得自己很失败，或者让自己或家人失望 |
| 7 | 对事物专注有困难，例如阅读报纸或看电视时不能集中注意力 |
| 8 | 动作或说话速度缓慢到别人已经觉察？或正好相反——烦躁或坐立不安、动来动去的情况比平时更多 |
| 9 | 有不如死掉或用某种方式伤害自己的念头 |

### 评分标准

#### 各题评分方式
| 选项 | 频率 | 得分 |
|-----|-----|-----|
| 完全不会 | 0天 | 0分 |
| 几天 | 1-6天 | 1分 |
| 超过一周 | 7-11天 | 2分 |
| 几乎每天 | 12-14天 | 3分 |

#### 总分判断（满分27分）
| 总分范围 | 抑郁程度 | 建议处理 |
|---------|---------|---------|
| **0-4分** | 无抑郁 | 常规随访 |
| **5-9分** | 轻微抑郁 | 持续观察，下次就诊时复查 |
| **10-14分** | 中度抑郁 | 制定治疗计划，考虑咨询、随访和/或药物治疗 |
| **15-19分** | 中重度抑郁 | 积极药物治疗和/或心理治疗 |
| **20-27分** | 重度抑郁 | 立即启动药物治疗，若缺乏反应则转介 |

> **特别注意**：第9题（自杀/自伤想法）若得分≥1，无论总分如何，均需立即进行安全评估。

## GAD-7 广泛性焦虑障碍量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 广泛性焦虑障碍量表（Generalized Anxiety Disorder Scale-7） |
| 缩写 | GAD-7 |
| 编制者 | Spitzer RL, Kroenke K等（2006年） |
| 适用对象 | 初中生至成人（≥12岁） |
| 题目数量 | 7题 |
| 答题形式 | 四级评分 |
| 测试时间 | 约3-5分钟 |
| 用途 | 广泛性焦虑障碍筛查和严重程度评估 |

### 完整题目（7题）

> 在过去的**两周**里，您有多少时间受到以下任何问题困扰？

| 题号 | 题目内容 |
|-----|---------|
| 1 | 感觉紧张、焦虑或急切 |
| 2 | 不能停止或控制担忧 |
| 3 | 对各种各样的事情担忧过多 |
| 4 | 很难放松下来 |
| 5 | 由于不安而无法静坐 |
| 6 | 变得容易烦恼或急躁 |
| 7 | 感到似乎将有可怕的事情发生而害怕 |

### 评分标准

#### 各题评分方式（同PHQ-9）
| 选项 | 频率 | 得分 |
|-----|-----|-----|
| 完全不会 | 0天 | 0分 |
| 几天 | 1-6天 | 1分 |
| 超过一周 | 7-11天 | 2分 |
| 几乎每天 | 12-14天 | 3分 |

#### 总分判断（满分21分）
| 总分范围 | 焦虑程度 | 建议处理 |
|---------|---------|---------|
| **0-4分** | 无焦虑 | 常规随访 |
| **5-9分** | 轻微焦虑 | 持续观察 |
| **10-14分** | 中度焦虑 | 考虑咨询和随访 |
| **15-21分** | 重度焦虑 | 立即积极治疗 |

---

## 联合使用说明

PHQ-9 和 GAD-7 常配对使用，同时评估抑郁和焦虑：

- **第10题（PHQ-9的功能损害题）**：如果您在上述问题中勾选了任何问题，这些问题对您来说在工作、处理家务或与他人相处方面造成了多大困难？
  - 完全没有困难（0分）
  - 有些困难（1分）
  - 很多困难（2分）
  - 极度困难（3分）

---

## 适用对象说明

> **重要提示**：研究显示，PHQ-9 和 GAD-7 在中国儿童青少年中的筛查效能有限，对12岁以下儿童的适用性存疑。建议：
> - **小学生（6-12岁）**：优先使用 SCARED（焦虑）和 CDI（抑郁）
> - **初中生及高中生（12-18岁）**：可使用 PHQ-9/GAD-7，但需注意文化适用性
