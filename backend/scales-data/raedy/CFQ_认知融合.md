## 一、认知融合问卷（CFQ）

> **来源**：Gillanders DT等（2014年）编制；中文版已验证，α = 0.94。

### 基本信息
| 适用对象 | 成人（≥18岁）|
|---------|-----|
| 题目数量 | 7题 |
| 答题形式 | 七级评分 |
| 用途 | 评估对想法的过度认同和融合程度（ACT核心概念）|

### 完整题目（7题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我的想法给我的生活造成了很多困扰 |
| 2 | 我与我的想法纠缠在一起 |
| 3 | 在面对困难的想法和感受时，我很难做到我最在乎的事 |
| 4 | 我很难与令人不安的想法和感受保持距离 |
| 5 | 我与我的想法融为一体 |
| 6 | 我花了很多精力与我的想法和感受斗争 |
| 7 | 与我的想法斗争阻止了我活在当下 |

### 评分标准
- 七级评分：1=从不，7=总是
- **总分** = 7题之和（7-49分），分数越高认知融合越严重
