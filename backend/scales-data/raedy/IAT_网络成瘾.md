## 一、Young 网络成瘾量表（IAT）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 网络成瘾测验（Internet Addiction Test） |
| 缩写 | IAT |
| 编制者 | Kimberly Young（1998年） |
| 中文版 | 国内多家机构完成翻译和适用性验证 |
| 适用对象 | 小学高年级至高中（10岁以上，有上网行为者） |
| 题目数量 | 20题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约5分钟 |
| 用途 | 筛查网络成瘾倾向及严重程度 |

### 完整题目（20题）

> 请根据您最近的实际情况，选择最符合的选项。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 您发现您上网的时间比预期的要长吗？ |
| 2 | 您会忽视家务而把更多的时间花在网上吗？ |
| 3 | 您宁愿在网上寻刺激，而不愿意与亲密伴侣亲热吗？ |
| 4 | 您会在网上与新朋友建立关系吗？ |
| 5 | 他人是否抱怨您花了太多时间上网？ |
| 6 | 您的学业成绩或学校作业是否因为您在网上花了太多时间而受到影响？ |
| 7 | 您会在还有其他事情要做之前先查看电子邮件吗？ |
| 8 | 您的学习效果是否会因上网而受到影响？ |
| 9 | 当有人问您在网上做什么时，您是否会变得戒备或行事秘密？ |
| 10 | 您是否会用想想上网来阻止日常生活中令人烦恼的想法？ |
| 11 | 您是否发现自己期待再次上网？ |
| 12 | 您是否害怕生活中没有网络会变得枯燥、空虚和无趣？ |
| 13 | 当有人在您上网时打扰您，您是否会大发雷霆、咆哮或生气？ |
| 14 | 您是否会因为熬夜上网而出现睡眠不足的问题？ |
| 15 | 当您不上网时，您是否会过度关注网上的事情或幻想上网？ |
| 16 | 当您上网时，您是否会说"再等几分钟"？ |
| 17 | 您是否曾经尝试过减少上网时间但失败了？ |
| 18 | 您是否会试图隐瞒您在网上花了多少时间？ |
| 19 | 您是否宁愿花更多时间上网，而不愿意与他人外出？ |
| 20 | 当您不上网时，您是否感到沮丧、情绪低落或神经过敏，而当您重新上网后，这种感觉消失？ |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 从不 | 0分（或1分） |
| 很少 | 1分（或2分） |
| 有时 | 2分（或3分） |
| 经常 | 3分（或4分） |
| 总是 | 4分（或5分） |

> 注：不同版本计分略有差异，以下使用1-5分版本（满分100分）

| 总分范围 | 网络使用状态 |
|---------|------------|
| **20-39分** | 正常用户，无需担心 |
| **40-59分** | 轻度成瘾倾向，需注意 |
| **60-79分** | 中度成瘾，建议寻求帮助 |
| **80-100分** | 重度成瘾，需立即干预 |
