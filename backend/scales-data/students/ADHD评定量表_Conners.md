# ADHD 注意缺陷多动障碍评定量表系列（Conners）

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Conners 儿童行为量表系列 |
| 编制者 | C. Keith Conners（1969年首版，多次修订） |
| 中文版 | 中国标准化版本已完成 |
| 适用对象 | 3-17岁儿童 |
| 版本 | 父母症状问卷（PSQ）/ 教师评定量表（TRS）/ 简明量表（ASQ） |
| 测试时间 | 10-20分钟 |
| 用途 | 筛查儿童注意力缺陷、多动、冲动等ADHD相关行为 |

---

## 一、Conners 父母症状问卷（PSQ，48题版）

### 量表结构（6个因子）

| 因子 | 主要内容 |
|-----|---------|
| 品行问题 | 不服从、攻击、说谎 |
| 学习问题 | 注意力不集中，学习困难 |
| 心身障碍 | 头痛、胃痛等躯体化症状 |
| 冲动多动 | 过度活动、坐立不安 |
| 焦虑 | 紧张、害怕、担忧 |
| 多动指数 | 综合多动行为指标（10题） |

### 代表性题目（父母版）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 乱动手脚或扭动身体坐不住 |
| 2 | 对大人粗鲁无礼 |
| 3 | 做作业时注意力不集中 |
| 4 | 在同学或玩耍时容易挑起争端 |
| 5 | 兴奋过度，冲动，不能自制 |
| 6 | 吸手指或咬指甲 |
| 7 | 容易哭或常哭泣 |
| 8 | 行为鲁莽，脾气暴躁，不可预测 |
| 9 | 情绪变化快，喜怒无常 |
| 10 | 脾气坏，行为不端 |

## 评分标准

### 各题评分（四级评分）

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 无 | 没有此问题 | 0分 |
| 只一点 | 轻微 | 1分 |
| 比较多/相当多 | 中度 | 2分 |
| 很多 | 明显/严重 | 3分 |

### 简明量表（ASQ）判断标准

| 总分 | 判断 |
|-----|-----|
| < 15分 | 无明显多动倾向 |
| ≥ 15分 | 有多动症可能，分数越高可能性越大 |

### 教师评定量表（TRS）划界分

- 多动指数 ≥ 1.5分（各题平均分）为异常
- 也可按年龄、性别查T分（T分 ≥ 65为有临床意义）

## 附：Vanderbilt ADHD 诊断量表（VADRS）简介

| 属性 | 内容 |
|------|------|
| 编制者 | 美国范德堡大学 |
| 适用 | 6-12岁，父母版和教师版 |
| 题目数 | 55题（父母版）/ 43题（教师版） |
| 特点 | 同时评估ADHD亚型和共病（对立违抗、焦虑、抑郁、学习问题） |
| 评分 | 0-3四级评分，参照DSM-IV标准判断 |

---

## 注意事项

1. Conners量表是**筛查工具**，ADHD诊断需符合DSM-5诊断标准
2. 建议同时收集**父母版和教师版**，多情境信息更可靠
3. 需排除其他可解释症状的因素（焦虑、抑郁、学习障碍等）
4. 确诊需专业儿童精神科医生进行临床评估
