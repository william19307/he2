# 积极心理学量表 II：生命意义、核心自我评价、创伤后成长

---

## 一、生命意义感量表（MLQ）

> **来源**：Steger MF等（2006年）编制；王鑫强等修订中文版，在中学生群体验证，发表于国内心理学期刊。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生至成人（≥12岁）|
| 题目数量 | 10题（拥有意义5题 + 寻求意义5题）|
| 答题形式 | 七级评分 |
| 信度 | 总量表 α = 0.71-0.91 |

### 完整题目（10题）

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 我明白自己生活的意义 | 拥有意义 |
| 2 | 我正在寻觅我人生的一个目的或使命 | 寻求意义 |
| 3 | 我的生活没有明确的目的 | 拥有意义（**反向**）|
| 4 | 我正在寻找自己生活的意义 | 寻求意义 |
| 5 | 我的生活有一个清晰的方向 | 拥有意义 |
| 6 | 我正在寻觅让我感觉自己生活饶有意义的东西 | 寻求意义 |
| 7 | 我知道什么东西能使自己的生活有意义 | 拥有意义 |
| 8 | 我总在尝试寻找自己生活的目的 | 寻求意义 |
| 9 | 我已经发现一个让自己满意的生活目的 | 拥有意义 |
| 10 | 我一直在寻找某样能使我的生活感觉起来是重要的东西 | 寻求意义 |

### 评分标准

七级评分：1=完全不符合，7=完全符合

- **拥有意义分（MLQ-P）** = 第1、3反向、5、7、9题之和（5-35分）
- **寻求意义分（MLQ-S）** = 第2、4、6、8、10题之和（5-35分）
- 分数越高，该方面意义感越强

## 二、核心自我评价量表（CSES）

> **来源**：Judge TA等（2003年）编制；中文版已发表于多个心理学期刊（α = 0.70-0.78）。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生至成人（≥12岁）|
| 题目数量 | 12题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约5分钟 |
| 涵盖维度 | 自尊、一般自我效能、控制点、情绪稳定性 |

### 完整题目（12题）

| 题号 | 题目内容 | 计分 |
|-----|---------|-----|
| 1 | 我坚信自己能够取得应有的成功 | 正向 |
| 2 | 有时候我会感到沮丧 | **反向** |
| 3 | 只要我努力尝试，我总能取得成功 | 正向 |
| 4 | 有时候，当我失败了，我会觉得自己很没用 | **反向** |
| 5 | 我总能很出色地完成任务 | 正向 |
| 6 | 有时候，我感觉自己很难掌控自己的工作 | **反向** |
| 7 | 总体而言，我对自己很满意 | 正向 |
| 8 | 我对自己的能力总是充满怀疑 | **反向** |
| 9 | 我自己能够决定我的生活 | 正向 |
| 10 | 事业上的成功不由我控制 | **反向** |
| 11 | 我能够很好地处理好自己绝大部分的问题 | 正向 |
| 12 | 有些时候，我会觉得事情变得没有希望 | **反向** |

### 评分标准

| 选项 | 正向题 | 反向题 |
|-----|-------|-------|
| 非常不同意 | 1分 | 5分 |
| 不同意 | 2分 | 4分 |
| 不确定 | 3分 | 3分 |
| 同意 | 4分 | 2分 |
| 非常同意 | 5分 | 1分 |

- **总分** = 12题得分之和（12-60分），分数越高核心自我评价越积极

## 三、创伤后成长量表（C-PTGI）

> **来源**：Tedeschi RG & Calhoun LG（1996年）编制；汪际等修订中文版（C-PTGI），发表于《护士进修杂志》（2011年）。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生至成人（经历过压力性事件者）|
| 题目数量 | 20题 |
| 答题形式 | 六级评分（0-5分）|
| 总分范围 | 0-100分 |
| 信度 | 总量表 α = 0.90 |

### 量表结构（5个因子）
| 因子 | 对应题号 | 常模均分 |
|-----|---------|---------|
| 人生感悟 | 2、3、7、11、13、17、19 | 20.06±5.67 |
| 个人力量 | 4、10、12、16 | 9.58±3.37 |
| 新的可能性 | 1、5、14、15、18 | 9.86±4.55 |
| 与他人关系 | 6、8、9、20 | 9.20±3.50 |
| 自我转变 | — | 9.44±4.15 |

### 完整题目（20题）

> 请根据这次经历对您带来的改变，选择最符合的程度。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我培养/发展了新的兴趣 |
| 2 | 我改变了生命中重要事物的先后顺序 |
| 3 | 我对自己的生命价值有了更多的欣赏/认识 |
| 4 | 我有更多依靠自己的感觉 |
| 5 | 我对精神事物有了更好的理解 |
| 6 | 我明白当我遇到困难时可以依靠他人 |
| 7 | 我确立了新的生命之路 |
| 8 | 我有与他人更亲近的感觉 |
| 9 | 我更愿意表达我的情感 |
| 10 | 我知道我能更好地处理困难 |
| 11 | 我能以我的生命做更好的事情 |
| 12 | 我更能接受任何事情的最后结果 |
| 13 | 我能更好地珍惜每一天 |
| 14 | 这次事件给我带来了新的机会 |
| 15 | 我对他人有了更多的同情 |
| 16 | 我花更多精力在人际关系上 |
| 17 | 对需要改变的事物，我更倾向于去改变它 |
| 18 | 我发现自己比想象中的更强 |
| 19 | 我对"人世间如此美好"的体会更深了 |
| 20 | 我更接受自己需要他人 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 完全没有 | 0分 |
| 极少 | 1分 |
| 有一些 | 2分 |
| 中等程度 | 3分 |
| 相当多 | 4分 |
| 非常多 | 5分 |

- **总分** = 20题之和（0-100分），分数越高创伤后成长越明显
