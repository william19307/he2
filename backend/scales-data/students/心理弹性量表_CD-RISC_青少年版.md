# 心理弹性（韧性）量表系列

---

## 一、Connor-Davidson 心理弹性量表（CD-RISC）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Connor-Davidson Resilience Scale |
| 缩写 | CD-RISC |
| 编制者 | Connor KM & Davidson JRT（2003年） |
| 中文版 | 于肖楠、张建新（2007年）修订 |
| 适用对象 | 初中生至成人（12岁以上） |
| 题目数量 | 25题 |
| 答题形式 | 五级评分（0-4分） |
| 测试时间 | 约10分钟 |
| 用途 | 评估个体面对逆境和压力的心理复原能力 |

### 完整题目（25题）

> 请根据**过去一个月**内您的感受，选择最符合的描述。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我能适应变化 |
| 2 | 我与朋友或家人有亲密的关系 |
| 3 | 有时上天或神明会帮助我 |
| 4 | 我能处理任何发生的事情 |
| 5 | 以前成功的经历帮助我面对新挑战 |
| 6 | 当事情看起来没有希望时，我不会轻易放弃 |
| 7 | 我能处理痛苦的感觉 |
| 8 | 有时命运或上天对我是有利的 |
| 9 | 我能从失败中爬起来 |
| 10 | 我认为自己是个坚强的人 |
| 11 | 我能做出不受欢迎或困难的决定 |
| 12 | 我能处理突如其来的状况 |
| 13 | 我能在压力下保持清醒 |
| 14 | 我做事有自信 |
| 15 | 我喜欢接受挑战 |
| 16 | 我在完成目标方面能付出足够努力 |
| 17 | 我对自己的成败坚持不懈 |
| 18 | 我能从批评中走出来 |
| 19 | 我能容忍不舒适的感受 |
| 20 | 我面对困难时会采取行动 |
| 21 | 我为自己的成就感到自豪 |
| 22 | 我感到自己能掌控生活中的事情 |
| 23 | 我能独立解决问题 |
| 24 | 我努力实现我的目标 |
| 25 | 我能从挫折中坚持下来 |

### 评分标准

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 0 | 完全不是这样 | 0分 |
| 1 | 很少是这样 | 1分 |
| 2 | 有时是这样 | 2分 |
| 3 | 经常是这样 | 3分 |
| 4 | 几乎总是这样 | 4分 |

**总分** = 25题得分之和（满分100分）

| 总分范围 | 心理弹性水平 |
|---------|-----------|
| 0-50分 | 低心理弹性，面对逆境能力较弱 |
| 51-70分 | 中等心理弹性 |
| 71-100分 | 高心理弹性，较强的复原力 |

**信度**：内部一致性 α = 0.89；重测信度 = 0.87

## 二、青少年心理韧性量表（RSCA）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 青少年心理韧性量表（Resilience Scale for Chinese Adolescents） |
| 缩写 | RSCA |
| 编制者 | 胡月琴、甘怡群（2008年，中国本土） |
| 适用对象 | 中学生（12-18岁） |
| 题目数量 | 27题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估中国青少年心理韧性 |

### 量表结构（5个维度）

| 维度 | 归属因子 | 题目数 | 主要内容 |
|-----|---------|-------|---------|
| 目标专注 | 个人力 | 6题 | 对目标持续的专注力 |
| 情绪控制 | 个人力 | 6题 | 情绪调节和管理能力 |
| 积极认知 | 个人力 | 5题 | 对困境的积极解释 |
| 家庭支持 | 支持力 | 5题 | 感知到的家庭支持 |
| 人际协助 | 支持力 | 5题 | 感知到的同伴支持 |

### 评分标准

- 五级评分：1=完全不符合，2=基本不符合，3=不确定，4=基本符合，5=完全符合
- **个人力** = 目标专注 + 情绪控制 + 积极认知 之和
- **支持力** = 家庭支持 + 人际协助 之和
- 总分越高，心理韧性越强
