# 自尊量表系列

---

## 一、Rosenberg 自尊量表（SES）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 罗森伯格自尊量表（Self-Esteem Scale） |
| 缩写 | SES |
| 编制者 | Morris Rosenberg（1965年） |
| 中文版 | 季益富、于欣等修订 |
| 适用对象 | 小学高年级至成人（10岁以上） |
| 题目数量 | 10题 |
| 答题形式 | 四级评分 |
| 测试时间 | 约5分钟 |
| 用途 | 评估整体自尊水平 |

### 完整题目（10题）

> 请根据您的实际情况选择最符合的答案。

| 题号 | 题目内容 | 计分方向 |
|-----|---------|---------|
| 1 | 我觉得我是一个有价值的人，至少与其他人不相上下 | 正向 |
| 2 | 我觉得我有很多优点 | 正向 |
| 3 | 总体来说，我倾向于认为自己是失败者 | **反向** |
| 4 | 我有能力和别人一样做好大多数事情 | 正向 |
| 5 | 我觉得我没有什么值得自豪的地方 | **反向** |
| 6 | 我对自己有良好的态度 | 正向 |
| 7 | 总体上我对自己满意 | 正向 |
| 8 | 我希望我能有更多的自尊 | **反向** |
| 9 | 我有时觉得自己无用 | **反向** |
| 10 | 我有时认为自己一点都不好 | **反向** |

### 评分标准

**各题评分**：
- 正向题：非常符合=4分，符合=3分，不符合=2分，非常不符合=1分
- 反向题（第3、5、8、9、10题）：非常符合=1分，符合=2分，不符合=3分，非常不符合=4分

**总分范围**：10-40分，分数越高自尊水平越高

| 总分范围 | 自尊水平 |
|---------|---------|
| 10-19分 | 低自尊 |
| 20-29分 | 中等自尊 |
| 30-40分 | 高自尊 |

## 二、儿童自尊量表（CSES）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 儿童自尊量表（Children's Self-Esteem Scale） |
| 缩写 | CSES |
| 编制者 | 魏运华（1997年，中国本土编制） |
| 适用对象 | 小学三年级至初中（8-15岁） |
| 题目数量 | 26题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估儿童多维度自尊 |

### 量表结构（6个因子）

| 因子 | 内容 | 题目数 |
|-----|-----|-------|
| 体育运动能力 | 在体育活动中的能力感知 | 4题 |
| 成就感 | 学业成就的自我评价 | 5题 |
| 外表 | 对自身外貌的满意度 | 4题 |
| 能力 | 综合能力的自我评估 | 5题 |
| 公德与助人 | 道德行为和助人意愿 | 4题 |
| 纪律 | 遵守规则的自我评价 | 4题 |

### 评分标准

- 五级评分：1=非常不符合，2=基本不符合，3=不肯定，4=基本符合，5=非常符合
- 各因子得分 = 该因子所有题目均分
- 总量表 Cronbach's α = 0.68；各维度系数 0.62-0.77

| 均分范围 | 自尊水平 |
|---------|---------|
| 1.0-2.5 | 较低 |
| 2.6-3.5 | 中等 |
| 3.6-5.0 | 较高 |
