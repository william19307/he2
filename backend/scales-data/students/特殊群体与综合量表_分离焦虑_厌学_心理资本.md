# 特殊群体与综合量表：分离焦虑、厌学、青少年心理资本

---

## 一、儿童分离焦虑量表（SAAS-R）

> **来源**：综合国内研究编制，基于DSM-5分离焦虑障碍的核心症状维度。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 小学至初中（6-14岁）|
| 题目数量 | 12题 |
| 答题形式 | 三级评分 |

### 完整题目（12题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 当我离开家或离开父母时，我感到非常担心 |
| 2 | 我担心父母会发生意外或不测 |
| 3 | 我拒绝单独睡觉，需要家人陪伴 |
| 4 | 我害怕一个人待在家里 |
| 5 | 当我要与父母分开时，我会感到非常难过 |
| 6 | 我担心迷路或失散，与家人失去联系 |
| 7 | 上学前，我会感到非常不安或不舒服 |
| 8 | 我做过关于家人发生意外的噩梦 |
| 9 | 当要离开家时，我会出现头痛、肚子痛等不舒服 |
| 10 | 我跟随家人走，因为害怕独处 |
| 11 | 我无法独自参加朋友的活动 |
| 12 | 与家人分开时，我难以集中注意力学习 |

### 评分标准
- 三级评分：0=从不，1=有时，2=经常
- **总分** = 12题之和（0-24分）

| 总分 | 分离焦虑程度 |
|-----|-----------|
| 0-6分 | 轻微或无 |
| 7-12分 | 中度 |
| 13-24分 | 重度，建议评估 |

## 二、中学生厌学量表

> **来源**：综合国内多位学者编制的学业厌倦量表（苏旭昇等），发表于心理学类期刊。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 题目数量 | 20题（4个维度各5题）|
| 答题形式 | 五级评分 |

### 量表结构（4个维度）
| 维度 | 内容 |
|-----|-----|
| 情绪倦怠 | 对学习感到疲惫和厌烦 |
| 认知功能下降 | 学习时注意力和理解力下降 |
| 行为退缩 | 减少参与学习活动 |
| 对学校的消极态度 | 对学校整体的消极看法 |

### 完整题目（20题）

**情绪倦怠（第1-5题）**
| 题号 | 题目内容 |
|-----|---------|
| 1 | 一想到上学，我就感到烦闷 |
| 2 | 我对上课感到厌倦 |
| 3 | 学习让我感到疲惫不堪 |
| 4 | 我觉得学习毫无意思 |
| 5 | 我不喜欢上学 |

**认知下降（第6-10题）**
| 题号 | 题目内容 |
|-----|---------|
| 6 | 上课时我很难集中注意力 |
| 7 | 学习的内容我很难理解或记住 |
| 8 | 做作业时我总是走神 |
| 9 | 我觉得学习越来越难 |
| 10 | 我对学习内容越来越提不起兴趣 |

**行为退缩（第11-15题）**
| 题号 | 题目内容 |
|-----|---------|
| 11 | 我经常不想完成作业 |
| 12 | 我在课堂上很少主动参与 |
| 13 | 我不想去图书馆或花时间复习 |
| 14 | 我常常找理由不参加学校活动 |
| 15 | 我在课堂上经常开小差或睡觉 |

**消极态度（第16-20题）**
| 题号 | 题目内容 |
|-----|---------|
| 16 | 我认为上学是在浪费时间 |
| 17 | 我觉得学校里没有什么值得我努力的 |
| 18 | 我不认为学习对我的未来有太大帮助 |
| 19 | 我对老师的教学方式感到不满 |
| 20 | 我觉得在学校里得不到应有的尊重和关注 |

### 评分标准
- 五级评分：1=完全不符合，5=完全符合
- **总分** = 20题之和（20-100分），分数越高厌学程度越严重

| 总分 | 厌学程度 |
|-----|---------|
| 20-40分 | 轻微 |
| 41-60分 | 中等，需关注 |
| 61-80分 | 较高，建议干预 |
| 81-100分 | 严重，需专业帮助 |

## 三、青少年心理资本量表

> **来源**：基于Luthans等（2007年）心理资本理论（HERO模型），适用于青少年的中国本土化版本。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 题目数量 | 24题（HERO模型，各6题）|
| 答题形式 | 六级评分 |

### 完整题目（24题）

**希望（第1-6题）**
| 题号 | 题目内容 |
|-----|---------|
| 1 | 在学习方面，我能想出很多方法来达到我的目标 |
| 2 | 我积极追求学习目标，从不放弃 |
| 3 | 当学习遇到困难时，我总能找到解决的方法 |
| 4 | 对于我的学习，我总是充满信心 |
| 5 | 我认为自己能够取得很好的学业成绩 |
| 6 | 我能想出很多实现学业目标的途径 |

**效能感（第7-12题）**
| 题号 | 题目内容 |
|-----|---------|
| 7 | 面对一项困难的学习任务，我相信自己能完成 |
| 8 | 在学习中，我总是相信自己能做到 |
| 9 | 在考试前，我对自己的能力充满信心 |
| 10 | 当别人认为某门功课很难时，我相信自己能学好 |
| 11 | 我相信自己有能力独立完成作业 |
| 12 | 我相信只要努力，就能提高学习成绩 |

**韧性（第13-18题）**
| 题号 | 题目内容 |
|-----|---------|
| 13 | 遇到学习挫折，我能很快调整心态 |
| 14 | 当我考试不理想时，我能从中恢复并继续努力 |
| 15 | 过去的失败使我变得更加坚强 |
| 16 | 无论遇到多难的学习任务，我都不会轻易放弃 |
| 17 | 在遭遇挫折后，我能迅速振作起来 |
| 18 | 在学习上，我经得起各种考验 |

**乐观（第19-24题）**
| 题号 | 题目内容 |
|-----|---------|
| 19 | 在学习方面，事情总是往好的方向发展 |
| 20 | 我对自己的学习前途充满期待 |
| 21 | 关于我的学习，我总是往好的方面去想 |
| 22 | 在学习上，我总是积极乐观 |
| 23 | 我相信我的学习成绩会越来越好 |
| 24 | 我对自己的学业充满希望 |

### 评分标准
- 六级评分：1=强烈不同意，6=强烈同意
- **各维度均分** = 6题得分之和 ÷ 6（1-6分）
- **心理资本总分** = 4个维度均分之均值（1-6分）

## 四、学生感知校园气氛量表（SPAS）

> **来源**：综合国内多项研究及Hoy等（2002年）框架编制。

### 完整题目（12题）

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 在这所学校，学生们互相尊重 | 同伴关系 |
| 2 | 大多数学生都愿意帮助有困难的同学 | 同伴关系 |
| 3 | 在这所学校，学习风气良好 | 学习氛围 |
| 4 | 同学们以学习为荣 | 学习氛围 |
| 5 | 老师真正关心学生的进步 | 教师支持 |
| 6 | 大多数老师会耐心解答学生的问题 | 教师支持 |
| 7 | 学校规则是公平的 | 制度公平 |
| 8 | 每个学生在学校里都受到公平对待 | 制度公平 |
| 9 | 这所学校是安全的 | 安全感 |
| 10 | 在这里，我不用担心被欺凌或嘲笑 | 安全感 |
| 11 | 学校活动丰富多彩，我喜欢参加 | 学校活动 |
| 12 | 我为我的学校感到自豪 | 整体满意 |

### 评分标准
- 五级评分：1=完全不同意，5=完全同意
- **总分** = 12题之和（12-60分），分数越高感知校园气氛越积极
