# 匹兹堡睡眠质量指数（PSQI）

> **来源**：Buysse DJ等（1989年）编制；刘贤臣等（1996年）修订中文版，发表于《中国精神科杂志》。广泛应用于中学生及以上人群的睡眠质量评估。

## 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生至成人（≥12岁）|
| 题目数量 | 19题（前18题用于计分）+ 5题同伴评定 |
| 评定时限 | 最近一个月 |
| 测试时间 | 约5-10分钟 |
| 总分范围 | 0-21分 |

---

## 完整题目（前19题，自评部分）

> 以下问题都是关于您**过去一个月**的睡眠情况。

| 题号 | 题目内容 | 记录/选择 |
|-----|---------|---------|
| 1 | 过去一个月，您晚上通常**上床睡觉**的时间是？ | 填写时间（如22:30）|
| 2 | 过去一个月，您每晚通常需要多长时间**才能入睡**（从躺下到睡着）？ | 填写分钟数 |
| 3 | 过去一个月，您每天早上通常**起床**的时间是？ | 填写时间 |
| 4 | 过去一个月，您每晚**实际睡眠时间**是多少小时（不是卧床时间）？ | 填写小时数 |
| **5** | **过去一个月，因以下情况影响睡眠的频率（共10个小项）：** | **详见子项** |
| 5a | 不能在30分钟内入睡 | 无/＜1次/每周1-2次/每周≥3次 |
| 5b | 在夜间或凌晨醒来（或过早醒来）| 同上 |
| 5c | 夜间起床上厕所 | 同上 |
| 5d | 呼吸不舒服 | 同上 |
| 5e | 大声咳嗽或打鼾 | 同上 |
| 5f | 感到寒冷 | 同上 |
| 5g | 感到太热 | 同上 |
| 5h | 做噩梦 | 同上 |
| 5i | 疼痛不适 | 同上 |
| 5j | 其他影响睡眠的事情（请描述）| 同上 |
| 6 | 过去一个月，您**自我评价**总体睡眠质量如何？ | 很好/较好/较差/很差 |
| 7 | 过去一个月，您使用**催眠药物**（处方药或非处方药）的频率是？ | 无/＜1次/每周1-2次/每周≥3次 |
| 8 | 过去一个月，您在**开车、吃饭或参加社会活动**等时难以保持清醒的情况是？ | 无/＜1次/每周1-2次/每周≥3次 |
| 9 | 过去一个月，**做事情时精力不足**的情况是？ | 无/有时/经常/非常明显 |

---

## 评分方法（7个成分，各0-3分）

### 成分A：主观睡眠质量
根据第6题直接计分：
- 很好 = 0分；较好 = 1分；较差 = 2分；很差 = 3分

### 成分B：入睡时间
| 第2题（入睡时长）| 第5a题 | 成分B分 |
|-------------|-------|--------|
| ≤15分钟 + 无 | → 两者得分之和0 | 0分 |
| 16-30分钟 + ＜1次/周 | → 两者之和1-2 | 1分 |
| 31-60分钟 + 每周1-2次 | → 两者之和3-4 | 2分 |
| ＞60分钟 + ≥3次/周 | → 两者之和5-6 | 3分 |

（具体：第2题≤15=0、16-30=1、31-60=2、＞60=3；5a无=0、＜1次=1、1-2次=2、≥3次=3；两者之和0=0分，1-2=1分，3-4=2分，5-6=3分）

### 成分C：睡眠时长（第4题）
| 实际睡眠时间 | 成分C分 |
|-----------|--------|
| ≥7小时 | 0分 |
| 6-7小时（不含7）| 1分 |
| 5-6小时（不含6）| 2分 |
| ＜5小时 | 3分 |

### 成分D：睡眠效率
睡眠效率 = 第4题（实际睡眠时间）÷（第3题起床时间 - 第1题上床时间）× 100%

| 睡眠效率 | 成分D分 |
|--------|--------|
| ＞85% | 0分 |
| 75-84% | 1分 |
| 65-74% | 2分 |
| ＜65% | 3分 |

### 成分E：睡眠障碍（第5b-5j题）
将9个小项得分相加（每项0-3分，共0-27分）：
- 总和0 = 0分；1-9 = 1分；10-18 = 2分；19-27 = 3分

### 成分F：催眠药物（第7题）
- 无 = 0分；＜1次/周 = 1分；1-2次/周 = 2分；≥3次/周 = 3分

### 成分G：日间功能障碍（第8、9题）
将第8、9题得分相加（各0-3分，共0-6分）：
- 0 = 0分；1-2 = 1分；3-4 = 2分；5-6 = 3分

---

## 总分计算与解释

**PSQI总分** = 成分A + B + C + D + E + F + G（0-21分）

| 总分 | 睡眠质量 |
|-----|---------|
| **0-5分** | 睡眠质量很好 |
| **6-10分** | 睡眠质量较好 |
| **11-15分** | 睡眠质量较差 |
| **16-21分** | 睡眠质量很差 |

**筛查阈值**：总分 **＞7分** 提示存在睡眠障碍（临床常用）

---

## 注意事项

1. 适用于12岁以上学生（更小年龄使用CSHQ）
2. 评定最近一个月整体睡眠情况
3. 量表中的催眠药物相关题目（F成分）对于不使用药物的学生记0分
4. 睡眠效率计算需要第1、3、4题数据，注意跨夜计算
