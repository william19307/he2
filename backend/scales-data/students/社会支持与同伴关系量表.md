# 社会支持与同伴关系量表系列

---

## 一、青少年社会支持量表（SSRS-A）

### 基本信息

| 属性 | 内容 |
|------|------|
| 参考编制 | 叶悦妹、戴晓阳（国内修订版） |
| 适用对象 | 中小学生（10-18岁） |
| 题目数量 | 17题 |
| 答题形式 | 四级评分 |
| 测试时间 | 约5-8分钟 |
| 用途 | 评估青少年感知到的社会支持水平 |

### 量表结构（3个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 家庭支持 | 6题 | 来自父母和家人的支持 |
| 朋友支持 | 5题 | 来自同伴和朋友的支持 |
| 其他支持 | 6题 | 来自教师和其他成人的支持 |

### 代表性题目

| 维度 | 题目示例 |
|-----|---------|
| 家庭支持 | 当我遇到困难时，家人会帮助我 |
| 家庭支持 | 我的父母关心我的学习和生活 |
| 朋友支持 | 我有可以倾诉的好朋友 |
| 朋友支持 | 当我伤心时，朋友会安慰我 |
| 其他支持 | 老师关心我的学习情况 |
| 其他支持 | 如果有需要，老师会帮助我 |

### 评分标准

- 四级评分：1=很少，4=经常
- 总分越高，感知到的社会支持越丰富

## 二、同伴关系量表（班级社会测量法）

### 基本信息

| 适用对象 | 班级学生（可用于多个年级） |
|---------|------------------------|
| 方法 | 提名法 + 评分法 |
| 评估内容 | 同伴接纳度、受欢迎程度、友谊网络 |

### 提名法操作

**正向提名**：写出3名最喜欢一起玩/学习的同学
**负向提名**：写出3名最不想一起玩/学习的同学（需谨慎使用）

**分类标准（基于提名分数）**：
| 社会地位类型 | 特征 |
|------------|-----|
| 受欢迎型 | 正向提名多，负向提名少 |
| 被拒绝型 | 负向提名多，正向提名少 |
| 被忽视型 | 正负提名均少（存在感低） |
| 争议型 | 正负提名均多 |
| 一般型 | 均处于中间水平 |

### 评分法

请对班上每位同学按5分制评分（1=非常不喜欢和他/她玩，5=非常喜欢和他/她玩）

## 三、友谊质量量表（FQQ）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 友谊质量问卷（Friendship Quality Questionnaire） |
| 编制者 | Parker JG & Asher SR（1993年） |
| 适用对象 | 小学至初中（8-14岁） |
| 题目数量 | 40题 |
| 答题形式 | 五级评分 |
| 用途 | 评估与最好朋友之间的友谊质量 |

### 量表结构（6个维度）

| 维度 | 内容 |
|-----|-----|
| 陪伴与娱乐 | 共同活动的频率和乐趣 |
| 帮助与指导 | 提供工具性帮助 |
| 亲密袒露 | 分享秘密和感受 |
| 肯定与关心 | 情感支持和肯定 |
| 矛盾与背叛 | 冲突和背叛行为（负向） |
| 冲突解决 | 如何有效处理分歧 |

### 评分标准

- 五级评分：1=完全不是这样，5=完全是这样
- 负向维度（矛盾与背叛）得分越高说明友谊质量越差
- 其他维度得分越高说明友谊质量越好

## 四、青少年人际关系量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生 |
| 题目数量 | 28题 |
| 评估内容 | 与父母、同学、教师的关系质量 |

### 量表结构（3个分量表）

| 分量表 | 题目数 | 主要内容 |
|-------|-------|---------|
| 亲子关系 | 10题 | 与父母的沟通和情感关系 |
| 同学关系 | 10题 | 与同班同学的友好程度 |
| 师生关系 | 8题 | 与教师的关系质量 |

### 代表性题目

| 分量表 | 题目示例 |
|-------|---------|
| 亲子关系 | 我与父母能够畅所欲言 |
| 亲子关系 | 父母理解和尊重我的想法 |
| 同学关系 | 我在班上有知心好友 |
| 同学关系 | 我能融入班级集体活动 |
| 师生关系 | 老师对我的态度是友好的 |
| 师生关系 | 我愿意向老师寻求帮助 |

### 评分标准

- 五级评分：1=完全不符合，5=完全符合
- 各分量表均分越高，关系质量越好
