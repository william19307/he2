# 情绪认知量表：反刍思维（RRS）& 青少年主观幸福感

---

## 一、反刍思维量表（RRS）

> **来源**：Nolen-Hoeksema S & Morrow J（1991年）编制；韩秀等修订中文版（RRS-CV），发表于国内心理学期刊，总量表 α = 0.90，重测信度 = 0.82。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生至成人（≥12岁）|
| 题目数量 | 22题 |
| 答题形式 | 四级评分 |
| 信度 | α = 0.90；重测信度 = 0.82 |

### 量表结构（3个维度）
| 维度 | 对应题号 | 内容 |
|-----|---------|-----|
| 症状反刍 | 1、2、3、4、6、8、9、14、17、18、19、22 | 对情绪状态的反复关注 |
| 强迫思考/忧思 | 5、10、13、15、16 | 对事情原因的消极沉思 |
| 反省深思 | 7、11、12、20、21 | 较理性的自我分析 |

### 完整题目（22题）

> 以下描述了人们在感到悲伤或情绪不好时可能有的一些想法和行为，请选择符合您实际情况的频率。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我常常想我是多么孤独 |
| 2 | 我常常想："如果我不能停止想这些，那我就不能总是继续做手头的事" |
| 3 | 我常常想我疲劳、痛苦的感觉 |
| 4 | 我常常想："集中注意力是多么困难" |
| 5 | 我常常想我究竟做了什么会导致这样 |
| 6 | 我常常想自己是多么消极被动、毫无动力 |
| 7 | 我常常分析最近发生的事以便理解为什么感到郁闷 |
| 8 | 我常常想我对其他事情感到麻木 |
| 9 | 我常常想我为什么如此不顺心 |
| 10 | 我常常想我为什么总是这样 |
| 11 | 我常常独自思考为什么会这样 |
| 12 | 我常常写下自己正在想的事情并加以分析 |
| 13 | 我常常思考现状，希望它有所好转 |
| 14 | 我常常想如果这种感觉持续的话，就无法集中注意 |
| 15 | 我常常想为什么我有这些问题，而别人却没有 |
| 16 | 我常常想我为什么不能把事情处理得更好 |
| 17 | 我常常想我为什么感到如此伤心 |
| 18 | 我常常想我的缺点、失败、错误和过失 |
| 19 | 我常常想："我对做任何事都提不起劲来" |
| 20 | 我常常分析自己的性格以便理解为何感到压抑 |
| 21 | 我会常常单独到某个地方去想我的感受 |
| 22 | 我常常想我是多么地生自己的气 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 从不 | 1分 |
| 有时 | 2分 |
| 经常 | 3分 |
| 总是 | 4分 |

- **总分** = 22题之和（22-88分），分数越高反刍思维越严重
- **各维度分** = 该维度题目之和
- 症状反刍和强迫思考维度高分与抑郁关系更密切
- 反省深思高分较为中性，属理性思考倾向

## 二、青少年主观幸福感量表（邢占军版）

> **来源**：邢占军（2002年）编制；发表于《应用心理学》，含中学生和大学生常模。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 题目数量 | 36题 |
| 答题形式 | 七级评分 |
| 维度 | 7个生活领域满意度 |

### 量表结构（7个维度）
| 维度 | 题目数 | 评估领域 |
|-----|-------|--------|
| 友谊满意度 | 约5题 | 与朋友关系的满意程度 |
| 家庭满意度 | 约5题 | 家庭生活的满意程度 |
| 学校满意度 | 约5题 | 对学校环境的满意程度 |
| 学业满意度 | 约5题 | 对学习情况的满意程度 |
| 自由满意度 | 约5题 | 对个人自主权的满意程度 |
| 环境满意度 | 约5题 | 对生活环境的满意程度 |
| 自我满意度 | 约6题 | 对自身的评价与满意度 |

### 代表性题目（各维度示例）

**友谊满意度**
- 我的朋友都很尊重我
- 我有很多朋友

**家庭满意度**
- 我喜欢和我的父母在一起
- 我的家庭是一个幸福的家庭

**学校满意度**
- 在学校里感到有所帮助（正向）
- 我在学校里感到不舒服（**反向**）

**学业满意度**
- 我在学业上取得了理想的成就
- 我在学校的成绩令自己满意

**自由满意度**
- 基本上没有人强迫我做自己不喜欢做的事
- 我能按自己的想法生活

**环境满意度**
- 我居住的地方适合我（正向）
- 我生活的环境周围有许多不如意的事情（**反向**）

### 评分标准

七级评分：1=完全不符合，7=完全符合（反向题取反）

- **各维度均分** = 该维度得分之和 ÷ 题目数

**中学生常模参考**（各维度均分）

| 均分 | 解释 |
|-----|-----|
| ≥ 6.0分 | 非常满意 |
| 5.0-5.9分 | 满意 |
| 4.0-4.9分 | 基本满意 |
| 3.0-3.9分 | 不确定 |
| ＜3.0分 | 不满意 |

**总体满意度阈值（初中生）**：总均分 ≥ 5.08 为"相当满意"
