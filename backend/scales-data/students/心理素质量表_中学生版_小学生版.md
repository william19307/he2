# 心理素质量表：中学生版 & 小学生版

---

## 一、中学生心理素质问卷（简化版，24题）

> **来源**：张大均等（西南大学）编制，发表于《西南大学学报》；简化版由张大均、王鑫强等修订，发表于《西南大学学报》2017年第2期。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 题目数量 | 24题（三个维度各8题）|
| 答题形式 | 五级评分 |
| 模型拟合 | CFI=0.948，RMSEA=0.036 |

### 量表结构（3个维度）
| 维度 | 题数 | 内涵 |
|-----|-----|-----|
| 认知特性 | 8题 | 意识性、监控性、好奇性、目的性、应变性 |
| 个性品质 | 8题 | 坚韧性、自制性、情绪调控、独立性 |
| 适应能力 | 8题 | 人际适应、社会适应、应激情境适应 |

### 代表性题目（各维度示例）

**认知特性（8题代表题目）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 做事前，我常会有清晰的思路和步骤 |
| 2 | 我常督促自己，以便如期完成该做的事 |
| 3 | 我对新知识很感兴趣 |
| 4 | 我能根据实际情况调整自己的学习方法 |
| 5 | 遇到问题时，我能想出多种解决办法 |
| 6 | 我能准确理解老师讲课的重点 |
| 7 | 我善于把新知识和已有知识联系起来 |
| 8 | 在做事情时，我常能抓住事情的关键 |

**个性品质（8题代表题目）**

| 题号 | 题目内容 |
|-----|---------|
| 9 | 即使遇到困难，我也能坚持完成任务 |
| 10 | 在困难面前，我不容易退缩 |
| 11 | 我能控制自己的行为，不受诱惑影响 |
| 12 | 我能合理安排自己的娱乐活动，控制娱乐时间 |
| 13 | 我能控制自己的情绪，不会随意发脾气 |
| 14 | 我对自己的行为有清醒的认识 |
| 15 | 我能独立解决生活中遇到的问题 |
| 16 | 在没有人监督的情况下，我也能完成任务 |

**适应能力（8题代表题目）**

| 题号 | 题目内容 |
|-----|---------|
| 17 | 我是一个比较受欢迎的人 |
| 18 | 我能主动与他人交流，建立良好关系 |
| 19 | 我能适应周围环境的变化 |
| 20 | 在新的环境中，我能较快适应 |
| 21 | 遇到突发事件，我能保持冷静 |
| 22 | 我能在压力下保持较好的状态 |
| 23 | 我能很好地融入班级集体 |
| 24 | 我了解社会的基本规则并自觉遵守 |

### 评分标准

- 五级评分：1=非常不符合，5=非常符合
- **各维度均分** = 该维度8题得分之和 ÷ 8（1-5分）
- **总分** = 三个维度均分之和，分数越高心理素质越好

## 二、小学生心理健康评定量表（MHRSP，教师用）

> **来源**：由北京师范大学等机构开发，周步成等改编，适用于教师评定小学生心理健康状况。广泛用于学校团体筛查。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 小学生（6-12岁）|
| 填写者 | 班主任或任课教师 |
| 题目数量 | 80题（8个分量表，各10题）|
| 答题形式 | 三级评分 |
| 测试时间 | 约20-30分钟 |

### 量表结构（8个分量表）
| 代码 | 分量表名称 | 主要评估内容 |
|-----|---------|-----------|
| A | 学习障碍 | 注意力、读写算、理解记忆 |
| B | 情绪障碍 | 焦虑、恐惧、抑郁 |
| C | 性格缺陷 | 自信心、自控力、责任感 |
| D | 社会适应障碍 | 人际关系、合作能力 |
| E | 品德行为障碍 | 诚实、遵规守纪 |
| F | 不良习惯 | 行为习惯、卫生习惯 |
| G | 特种行为障碍 | 强迫、冲动、多动 |
| H | 躯体化障碍 | 身心反应、躯体化症状 |

### 代表性题目（各分量表示例）

**A：学习障碍**
- 上课注意力容易分散，难以集中听讲
- 对学习漠不关心，毫无兴趣
- 读写算有明显困难，跟不上同学

**B：情绪障碍**
- 性情胆怯，常常流露出不安和焦虑
- 对很多事情感到恐惧和紧张
- 表现忧郁、沮丧，情绪低落

**C：性格缺陷**
- 缺乏自信，做事畏首畏尾
- 做事马虎，缺乏责任感

**D：社会适应障碍**
- 与班级同学相处困难，有孤立感
- 不愿参与集体活动

**G：特种行为障碍**
- 有强迫性重复动作或思想
- 情绪冲动，自控能力差
- 上课坐立不安，多动

### 评分标准

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 经常 | 几乎每天有 | 2分 |
| 偶尔 | 有时出现 | 1分 |
| 没有 | 从未有过 | 0分 |

- **各分量表分** = 10题得分之和（0-20分）
- **筛查阈值**：任一分量表分 **≥10分** 提示该方面存在明显问题

| 分量表分 | 说明 |
|--------|-----|
| 0-3分 | 正常范围 |
| 4-9分 | 轻度问题，需关注 |
| ≥10分 | 较明显问题，建议干预 |
