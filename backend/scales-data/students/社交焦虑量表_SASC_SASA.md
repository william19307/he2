# 社交焦虑量表：儿童版（SASC）& 青少年版（SAS-A）

---

## 一、儿童社交焦虑量表（SASC）

> **来源**：La Greca AM & Stone WL（1993年）编制；中文版经多项研究验证，信度系数 α = 0.76。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 小学至初中（6-13岁）|
| 题目数量 | 10题 |
| 答题形式 | 三级评分 |
| 测试时间 | 约5分钟 |
| 两周重测信度 | 0.67 |

### 量表结构（2个因子）
| 因子 | 对应题号 | 内容 |
|-----|---------|-----|
| 害怕否定评价 | 1、2、5、6、8、10 | 担心他人的负面评价 |
| 社交回避及苦恼 | 3、4、7、9 | 回避社交场合的行为和苦恼 |

### 完整题目（10题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我害怕在别的孩子面前做没做过的事 |
| 2 | 我担心被人取笑 |
| 3 | 我周围都是我不认识的小朋友时，我觉得害羞 |
| 4 | 我和小伙伴一起时很少说话 |
| 5 | 我担心其他孩子会怎么样看待我 |
| 6 | 我觉得小朋友们取笑我 |
| 7 | 我和陌生的小朋友说话时感到紧张 |
| 8 | 我担心其他孩子会怎样说我 |
| 9 | 我只同我熟悉的小朋友说话 |
| 10 | 我担心别的小朋友会不喜欢我 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 从无 | 0分 |
| 有时有 | 1分 |
| 一直有 | 2分 |

- **总分** = 10题之和（0-20分），分数越高社交焦虑越明显
- **害怕否定评价分** = 第1、2、5、6、8、10题之和（0-12分）
- **社交回避及苦恼分** = 第3、4、7、9题之和（0-8分）

## 二、青少年社交焦虑量表（SAS-A）

> **来源**：La Greca AM & Lopez N（1998年）编制；朱海东（2004年）修订中文版，发表于相关期刊。总量表 α = 0.816。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（11-17岁）|
| 题目数量 | 18题 |
| 答题形式 | 五级评分 |
| 信度 | α = 0.816；重测信度 = 0.86 |

### 量表结构（3个分量表）
| 分量表 | 对应题号 | 内容 |
|-------|---------|-----|
| 害怕否定评价（FNE）| 2、5、6、7、9、11、13、14 | 担心他人负面评价的苦恼 |
| 陌生情境下社交回避（SAD-New）| 1、3、4、8、10、16 | 陌生情境中的焦虑和回避 |
| 一般情境下社交回避（SAD-General）| 12、15、17、18 | 日常社交环境中的焦虑 |

### 完整题目（18题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我在别人面前做自己不熟悉的事情时会感到焦虑 |
| 2 | 我害怕被别人耻笑打趣 |
| 3 | 当我处于陌生人中间时，我会很害羞 |
| 4 | 我只和我很熟悉的人交谈 |
| 5 | 我感觉同龄人在背后谈论我 |
| 6 | 我担心别人会怎么看我 |
| 7 | 我害怕别人会不喜欢我 |
| 8 | 当和不熟悉的同龄人交谈时我会很紧张 |
| 9 | 我总担心别人会怎么评论我 |
| 10 | 当我认识新朋友时总会很紧张 |
| 11 | 我担心别人不喜欢我 |
| 12 | 与一群人在一起时我总是保持沉默 |
| 13 | 我总觉得别人在取笑我 |
| 14 | 我认为如果我和别人争执的话，别人会不喜欢我 |
| 15 | 我会因为害怕被拒绝而不愿意邀请别人同我一起做事 |
| 16 | 同某些人在一起时，我会感到紧张 |
| 17 | 即使与很熟悉的人在一起，我也会感到害羞 |
| 18 | 对我来说，邀请别人同我一起做事太难了 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 完全不符合 | 1分 |
| 不太符合 | 2分 |
| 不确定 | 3分 |
| 比较符合 | 4分 |
| 完全符合 | 5分 |

- **总分** = 18题之和（18-90分），分数越高社交焦虑越明显
- **各分量表分** = 该分量表题目之和
