	⁃	# 多元补充量表：攻击性、道德推脱、学业拖延、身体意象

## 一、青少年攻击性量表（Buss-Perry，AQ-CN）

> **来源**：Buss AH & Perry M（1992年）编制（29题）；朱海东（2004年）修订中文版（简版22题）。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生至成人（≥12岁）|
| 题目数量 | 22题（中文简版）|
| 答题形式 | 五级评分 |
| 测试时间 | 约8分钟 |

### 量表结构（4个维度）
| 维度 | 题数 | 内容 |
|-----|-----|-----|
| 身体攻击 | 6题 | 用身体伤害他人的行为倾向 |
| 言语攻击 | 5题 | 用言语伤害他人的倾向 |
| 愤怒 | 6题 | 情绪激动、愤怒失控 |
| 敌意 | 5题 | 对他人的不信任和恶意解读 |

### 代表性题目
| 维度 | 题目示例 |
|-----|---------|
| 身体攻击 | 如果有人打了我，我会还击 |
| 身体攻击 | 我曾经跟人打架 |
| 言语攻击 | 我不能控制情绪，无法不与人争吵 |
| 愤怒 | 我能控制我的脾气（**反向**）|
| 愤怒 | 我有时感到快要爆炸了 |
| 敌意 | 我开始怀疑别人是否真的对我友善 |
| 敌意 | 我的朋友说我有点偏执 |

### 评分标准
- 五级评分：1=完全不像我，5=完全像我
- 反向题取反后计分
- **总分** = 22题之和（22-110分），分数越高攻击性越强

## 二、道德推脱量表（24题版）

> **来源**：Moore C等（2012年）编制；王兴超、杨继平等修订中文版，已在初中生和高中生群体验证，各维度 α ≥ 0.80。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生至成人（≥12岁）|
| 题目数量 | 24题（8个机制各3题）|
| 答题形式 | 七级评分（也有五级版本）|

### 量表结构（8个推脱机制，各3题）

| 机制 | 简称 | 内容 |
|-----|-----|-----|
| 道德辩护 | MJ | 认为不道德行为是正当的 |
| 委婉标签 | EL | 用温和词语掩盖不道德行为 |
| 有利比较 | AC | 与他人更坏的行为相比，自己的不道德行为显得微不足道 |
| 责任转移 | DR | 将责任归咎于权威或他人 |
| 责任分散 | DF | 集体行为中个人无需负责 |
| 扭曲结果 | DC | 否认或淡化不道德行为的危害 |
| 非人化 | DH | 否认受害者的人性 |
| 责备归因 | AV | 认为受害者咎由自取 |

### 代表性题目（各机制示例）

| 机制 | 题目示例 |
|-----|---------|
| 道德辩护 | 为了保护自己的群体，打架是可以接受的 |
| 道德辩护 | 在某些情况下，欺骗别人是必要的 |
| 委婉标签 | 把伤害别人的行为称为"开玩笑"是可以理解的 |
| 有利比较 | 跟社会上那些真正的坏事相比，我做的事没什么大不了的 |
| 责任转移 | 只要是上级命令的，执行者就不需要为造成的伤害负责 |
| 责任分散 | 如果大家都这么做，一个人就不需要为此负责 |
| 扭曲结果 | 有些伤害别人的行为实际上没有那么严重 |
| 非人化 | 有些人活该被粗暴对待，因为他们行为恶劣 |
| 责备归因 | 那些受伤害的人往往是因为他们自己不好 |

### 评分标准
- 七级评分：1=完全不同意，7=完全同意
- **总分** = 24题均分（1-7分），或各维度均分
- 分数越高，道德推脱倾向越强

## 三、学业拖延量表（学生版，PASS改编）

> **来源**：基于Solomon LJ & Rothblum ED（1984年）PASS量表及国内适用性改编，多个版本在中学生中使用。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 题目数量 | 16题（中学生常用版本）|
| 答题形式 | 四级评分（A/B/C/D）或五级 |

### 代表性题目（16题版本示例）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我在做作业时，经常先去玩，最后才赶着做 |
| 2 | 我常常推迟完成老师布置的任务 |
| 3 | 即使知道时间不多了，我还是很难开始做作业 |
| 4 | 我能按时完成学习任务（**反向**）|
| 5 | 做作业时，我容易被其他事情吸引而分心 |
| 6 | 我常常在截止日期到来前一晚才开始准备 |
| 7 | 当有难题时，我更容易拖延去做 |
| 8 | 我为自己的拖延感到内疚 |
| 9 | 我善于合理安排完成任务的时间（**反向**）|
| 10 | 我会因拖延而影响学习成绩 |
| 11 | 即使有充分时间，我也会拖到最后才做 |
| 12 | 我很难为长期的作业制定计划 |
| 13 | 做作业前我常常找各种借口推迟 |
| 14 | 我的拖延让老师或父母感到担忧 |
| 15 | 我总是在最后关头才开始备考 |
| 16 | 我希望自己能更好地克服拖延 |

### 评分标准
- A=4分，B=3分，C=2分，D=1分
- 反向题（第4、9题）：A=1分，B=2分，C=3分，D=4分
- **总分** = 16题之和（16-64分）

| 总分 | 拖延程度 |
|-----|---------|
| 16-32分 | 低度拖延 |
| 33-48分 | 中度拖延，需注意 |
| 49-64分 | 高度拖延，建议干预 |

## 四、青少年身体欣赏量表（BAS-2改编版）

> **来源**：Tylka TL & Wood-Barcalow NL（2015年）编制原版BAS-2（10题）；中文版已完成心理测量验证（单因子结构，内部一致性良好）。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生至成人（≥12岁）|
| 题目数量 | 10题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约3分钟 |
| 结构 | 单因子（身体欣赏）|

### 完整题目（10题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我尊重我的身体 |
| 2 | 我对自己的身体有积极的感受 |
| 3 | 我关注我的身体 |
| 4 | 我感觉自己爱自己的身体 |
| 5 | 我欣赏身体不同寻常的特征 |
| 6 | 我对自己的身体感到满意 |
| 7 | 我对我的身体有好感 |
| 8 | 我是接受自己身体的 |
| 9 | 我感觉自己爱我的身体（即便与社会标准有所不同）|
| 10 | 我以欣赏的态度看待我的身体 |

### 评分标准
- 五级评分：1=从不，5=总是
- **总分** = 10题之和（10-50分），分数越高身体欣赏程度越高
- **正相关**：自尊、生活满意度、积极情绪
- **负相关**：进食障碍症状、身体监视、消极情绪
