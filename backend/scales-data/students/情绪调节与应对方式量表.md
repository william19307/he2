# 情绪调节与应对方式量表系列

---

## 一、青少年情绪调节量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 嵇家俊、卢家楣（中国本土编制） |
| 适用对象 | 初中生、高中生（12-18岁） |
| 题目数量 | 25题 |
| 答题形式 | 六级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估青少年情绪调节能力的多个维度 |

### 量表结构（6个维度）

| 维度 | 内容 | 题目数 |
|-----|-----|-------|
| 情绪知觉能力 | 识别和理解自身及他人情绪 | 4题 |
| 情绪评价能力 | 对情绪的准确评价 | 4题 |
| 情绪控制能力 | 管理和调控情绪的能力 | 5题 |
| 调节策略应用 | 有效运用情绪调节方法 | 5题 |
| 情绪调节效能感 | 对自身情绪调节能力的信心 | 4题 |
| 情绪调节反思能力 | 对情绪调节过程的反思 | 3题 |

### 代表性题目

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 我能准确感知自己正处于什么情绪状态 | 情绪知觉 |
| 2 | 我能理解为什么自己会有这种情绪 | 情绪知觉 |
| 3 | 我能感受到他人的情绪变化 | 情绪知觉 |
| 4 | 我能准确判断情绪的强度 | 情绪评价 |
| 5 | 我能分辨相似的情绪（如焦虑和恐惧） | 情绪评价 |
| 6 | 我能调控自己的情绪不让它影响他人 | 情绪控制 |
| 7 | 当情绪激动时，我能让自己平静下来 | 情绪控制 |
| 8 | 我能在需要时适当表达自己的情绪 | 情绪控制 |
| 9 | 我会用多种方法调节自己的情绪 | 策略应用 |
| 10 | 我能找到适合自己的情绪调节方法 | 策略应用 |
| ... | ... | ... |

### 评分标准

- 六级评分：1=完全不符合，6=完全符合
- 各维度均分越高，该方面情绪调节能力越强

## 二、简易应对方式问卷（SCSQ）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 简易应对方式问卷（Simplified Coping Style Questionnaire） |
| 缩写 | SCSQ |
| 编制者 | 解亚宁（1998年） |
| 适用对象 | 初中生至成人（12岁以上） |
| 题目数量 | 20题 |
| 答题形式 | 四级评分 |
| 测试时间 | 约5-8分钟 |
| 用途 | 评估面对压力时的应对倾向 |

### 完整题目（20题）

> 当您遇到困难或压力时，您通常怎么做？

**积极应对（第1-12题）**

| 题号 | 题目内容 |
|-----|---------|
| 1 | 通过工作学习或其他活动解脱 |
| 2 | 与人交谈，倾诉内心烦恼 |
| 3 | 尽量往好处想，看到事物积极的一面 |
| 4 | 改变自己的想法，重新发现生活中什么重要 |
| 5 | 不把问题看得太严重 |
| 6 | 坚持自己的立场，为自己想要的斗争 |
| 7 | 找出几种不同解决问题的方法 |
| 8 | 向亲戚、朋友或同学寻求建议 |
| 9 | 改变原来的一些想法和做法 |
| 10 | 借鉴他人处理类似困难情景的办法 |
| 11 | 寻求业余爱好，积极参加文体活动 |
| 12 | 尽量克制自己的失望、悔恨、悲伤和愤怒 |

**消极应对（第13-20题）**

| 题号 | 题目内容 |
|-----|---------|
| 13 | 通过抽烟、喝酒、服药和吃东西来解除烦恼 |
| 14 | 幻想可能会发生某种奇迹改变现状 |
| 15 | 自暴自弃，认为自己不行了 |
| 16 | 避开困难，幻想它不存在 |
| 17 | 认为时间会改变现状，唯一要做的是等待 |
| 18 | 试图休息或休假，暂时把困难和问题抛在一边 |
| 19 | 通过运动锻炼、冷静思考去减少紧张 |
| 20 | 寻求专业人员（心理咨询/医生）的帮助 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 不采取 | 0分 |
| 偶尔采取 | 1分 |
| 有时采取 | 2分 |
| 经常采取 | 3分 |

- **积极应对分** = 第1-12题得分之和（0-36分）
- **消极应对分** = 第13-20题得分之和（0-24分）
- 积极应对分高、消极应对分低为健康应对方式

## 三、中学生压力知觉量表（PSS-10）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Perceived Stress Scale-10 |
| 缩写 | PSS-10 |
| 编制者 | Cohen S（1983年），国内已完成修订 |
| 适用对象 | 初中生至成人（12岁以上） |
| 题目数量 | 10题 |
| 答题形式 | 五级评分 |
| 用途 | 评估近一个月内感知到的压力水平 |

### 完整题目（10题）

> 请回答**过去一个月内**下列情况发生的频率。

| 题号 | 题目内容 | 计分方向 |
|-----|---------|---------|
| 1 | 因为发生意外的事情而感到心烦意乱 | 正向 |
| 2 | 感到无法控制生活中重要的事情 | 正向 |
| 3 | 感到紧张和压力 | 正向 |
| **4** | **成功地处理了生活中令人烦恼的问题** | **反向** |
| **5** | **感到能有效地应付生活中的重要变化** | **反向** |
| 6 | 对自己处理个人问题的能力缺乏信心 | 正向 |
| 7 | 能够按自己的方式处理事情 | 反向 |
| 8 | 感到有能力应对困难 | 反向 |
| 9 | 因为发生在控制范围之外的事情而感到愤怒 | 正向 |
| 10 | 感到困难积累了太多无法克服 | 正向 |

### 评分标准

- 正向题：从不=0，偶尔=1，有时=2，经常=3，总是=4
- 反向题（第4、5、7、8题）：从不=4，偶尔=3，有时=2，经常=1，总是=0
- **总分** = 10题得分之和（0-40分）

| 总分范围 | 压力水平 |
|---------|---------|
| 0-13分 | 低压力 |
| 14-26分 | 中等压力 |
| 27-40分 | 高压力 |
