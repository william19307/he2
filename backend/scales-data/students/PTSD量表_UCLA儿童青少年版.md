# UCLA 儿童/青少年 PTSD 反应指数量表

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | UCLA创伤后应激障碍反应指数（UCLA Child/Adolescent PTSD Reaction Index） |
| 缩写 | UCLA PTSD-RI |
| 编制者 | 美国加州大学洛杉矶分校（Robert Pynoos等） |
| 中文版 | 台湾、香港及大陆已完成中文版验证 |
| 适用对象 | 6-18岁儿童及青少年（儿童版6-12岁，青少年版13-18岁） |
| 题目数量 | 22题（含4题前提问题） |
| 答题形式 | 五级评分（0-4分） |
| 测试时间 | 约20-30分钟 |
| 用途 | 评估经历创伤事件后的PTSD症状 |

---

## 量表结构

### 第一部分：创伤经历核查（前提问题，4题）

询问是否经历以下类型的创伤事件：
- 自然灾害（洪水、地震、火灾）
- 交通事故
- 人际暴力（被打、性暴力）
- 目睹他人受伤或死亡
- 其他创伤性事件

### 第二部分：PTSD症状评估（22题，按DSM-5分类）

| 症状群 | 维度 | 题目数 |
|-------|-----|-------|
| B：再体验症状 | 闯入性回忆、噩梦、闪回 | 5题 |
| C：回避症状 | 回避创伤相关想法和场所 | 2题 |
| D：认知与情绪负性改变 | 负性信念、情绪麻木、自责 | 7题 |
| E：警觉性增高 | 过度警觉、惊吓反应、睡眠障碍 | 6题 |
| 功能损害 | 影响学习、社交、家庭 | 2题 |

---

## 完整题目（22题）

> 请根据**过去一个月内**的感受，选择最符合的选项。

| 题号 | 题目内容 | 症状群 |
|-----|---------|-------|
| 1 | 当有事物提醒我那件事时，我感到非常不安 | 再体验 |
| 2 | 我脑中会突然出现关于那件事的画面或想法，就像又在经历一样 | 再体验 |
| 3 | 我做关于那件事的噩梦或其他可怕的梦 | 再体验 |
| 4 | 我感觉好像那件事又要再发生了 | 再体验 |
| 5 | 当某些事物让我想起那件事时，我身体会有不舒适的感觉（心跳加快、出汗等） | 再体验 |
| 6 | 我努力不去想关于那件事的事情 | 回避 |
| 7 | 我不去提醒自己那件事，或避免能让我想起的地方和人 | 回避 |
| 8 | 我对一些曾经重要的活动不再感兴趣 | 认知/情绪 |
| 9 | 我感到与其他人疏远或隔离 | 认知/情绪 |
| 10 | 我感到对未来没有期待 | 认知/情绪 |
| 11 | 我感到自己有罪或后悔 | 认知/情绪 |
| 12 | 我对自己或周围的世界有很负面的想法 | 认知/情绪 |
| 13 | 我感到情绪麻木，无法感受正面情绪 | 认知/情绪 |
| 14 | 我无法记住那件事的某些重要部分 | 认知/情绪 |
| 15 | 我会突然大发脾气或爆发愤怒 | 警觉/反应性 |
| 16 | 我很难集中注意力 | 警觉/反应性 |
| 17 | 我总是提高警觉或注意周围的危险 | 警觉/反应性 |
| 18 | 一些东西或突然的响声让我很容易受惊 | 警觉/反应性 |
| 19 | 我很难入睡或保持睡眠 | 警觉/反应性 |
| 20 | 我有冒险或自我伤害的行为 | 警觉/反应性 |
| 21 | 这些问题影响了我在学校的表现 | 功能损害 |

## 评分标准

### 各题评分（五级评分，0-4分）

| 选项 | 含义 | 得分 |
|-----|-----|-----|
| 从不/没有 | 完全没有这种感觉 | 0分 |
| 很少 | 有时候，每月1次左右 | 1分 |
| 有时 | 每周1-2次 | 2分 |
| 经常 | 几乎每天，每周4-5次 | 3分 |
| 几乎总是 | 每天都有 | 4分 |

### 计分方法

**PTSD总分** = 22个症状题得分之和（0-88分）

**各症状群分**：
- 再体验分 = 题1-5之和（0-20分）
- 回避分 = 题6-7之和（0-8分）
- 认知/情绪负性改变分 = 题8-14之和（0-28分）
- 警觉/反应性分 = 题15-20之和（0-24分）

### 结果解释

| PTSD总分 | 严重程度 |
|---------|---------|
| **< 20分** | 轻微或无症状 |
| **20-29分** | 轻度PTSD症状 |
| **30-44分** | 中度PTSD症状，建议临床评估 |
| **≥ 45分** | 严重PTSD症状，需立即专业干预 |

---

## 注意事项

1. 施测前须确认儿童经历了特定创伤事件
2. 结果仅反映症状严重程度，**不等于PTSD诊断**，需专业评估
3. 对于有严重创伤症状的学生，评估过程中需采取安全保护措施
4. 量表可用于追踪干预效果
