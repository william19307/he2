# 学业压力与情绪量表：考试焦虑、学业情绪、时间管理

---

## 一、考试焦虑测验（TAI）

> **来源**：Spielberger CD（1980年）编制；中文版由多位研究者修订，适用于中学生的33题版本广泛流传。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 题目数量 | 33题 |
| 答题形式 | 四级评分（A/B/C/D）|
| 测试时间 | 约10分钟 |

### 量表结构（2个分量表）
| 分量表 | 内容 |
|-------|-----|
| 忧虑性（Worry）| 关于考试失败后果的担忧想法 |
| 情绪性（Emotionality）| 考试前后的紧张、生理反应 |

### 完整题目（33题）

> 请根据符合您情况的程度选择：A=很符合，B=比较符合，C=不太符合，D=很不符合

| 题号 | 题目内容 |
|-----|---------|

| 5 | 在考试前，我感到烦躁，脾气变坏 |
| 6 | 在紧张的复习期间，常会想到："这次考试要是得到个坏分数怎么办？" |

| 12 | 当听到开始考试的铃声响了，我的心马上紧张地急跳起来 |
| 13 | 遇到重要的考试，我的脑子就变得比平时迟钝 |
| 14 | 看到考试题目越多、越难，我就越感到不安 |

| 25 | 在进行重要的考试时，我的头就会痛起来 |
| 26 | 发现剩下的时间来不及做完全部考题，我就急得手足无措、浑身大汗 |
| 27 | 如果我考了个坏分数，家长或老师会严厉地指责我 |
| 28 | 在考试后，发现自己会做的题目没有答对时，就十分生自己的气 |

| 32 | 考试不应当像现在这样的紧张状态下进行 |
| 33 | 不考试，我能学到更多的知识 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| A（很符合）| 3分 |
| B（比较符合）| 2分 |
| C（不太符合）| 1分 |
| D（很不符合）| 0分 |

**总分** = 3×选A次数 + 2×选B次数 + 1×选C次数

| 总分 | 焦虑程度 |
|-----|---------|
| 0-24分 | 镇定自若 |
| 25-49分 | 轻度焦虑 |
| 50-74分 | 中度焦虑 |
| 75-99分 | 重度焦虑 |

## 二、青少年学业情绪问卷（AEQ-A，简版）

> **来源**：董妍、俞国良（2007年）编制，发表于《心理学报》。原版72题，以下为各维度代表性题目说明。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 完整版题目数量 | 72题（13种学业情绪）|
| 答题形式 | 五级评分 |
| 信度 | 各分问卷 α = 0.79-0.92 |

### 四大学业情绪类别

| 类别 | 具体情绪 | 题数（各）|
|-----|---------|---------|
| 积极高唤醒 | 自豪、高兴、希望 | 各6题 |
| 积极低唤醒 | 满足、平静、放松 | 各6题 |
| 消极高唤醒 | 焦虑、羞愧、生气 | 各6题 |
| 消极低唤醒 | 厌倦、无助、沮丧、疲乏 | 各6题 |

### 各情绪代表性题目

**自豪**：学习成绩好时，我会非常高兴；当我在测验中表现出色时，我很为自己自豪

**高兴**：学习时，我感到快乐；做功课的时候，我心情很好

**焦虑**：考试前，我会非常担心；我害怕考试时答不上来

**厌倦**：学习时，我感到很无聊；上课时，我觉得时间过得很慢

**无助**：我感到无论怎样努力，成绩也无法提高；无论我怎么做，都无法把学业搞好

### 评分标准
- 五级评分：1=没有，2=很少很轻，3=中等，4=较多偏重，5=很多严重
- 各情绪维度均分 = 该情绪6题之和 ÷ 6

## 三、青少年时间管理倾向量表（ATMD）

> **来源**：黄希庭、张志杰（2001年）编制，发表于《心理学报》第33卷第4期。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 题目数量 | 44题（三个维度）|
| 答题形式 | 五级评分 |
| 信度 | 各维度 α = 0.80-0.88 |

### 量表结构（3个维度）
| 维度 | 题数 | 主要内容 |
|-----|-----|---------|
| 时间价值感 | 10题 | 对时间价值的认识（社会取向+个人取向）|
| 时间监控观 | 24题 | 目标设定、计划制定、优先顺序、时间分配与反馈 |
| 时间效能感 | 10题 | 对时间管理能力的信念 |

### 代表性题目（各维度示例）

**时间价值感**
| 题目示例 |
|---------|
| 我认为时间是最宝贵的资源 |
| 我认为有效地利用时间是非常重要的 |
| 当我浪费时间时，我会感到内疚 |

**时间监控观**
| 题目示例 |
|---------|
| 我制定日程表、周计划或月计划 |
| 我能按照制定的计划做事 |
| 我根据事情的轻重缓急来安排任务 |
| 我把重要的工作放在效率最高的时候完成 |
| 我及时总结自己利用时间的经验教训 |

**时间效能感**
| 题目示例 |
|---------|
| 我相信自己能够很好地利用时间 |
| 我能胜任时间管理工作 |
| 对于能如期完成计划，我充满信心 |

### 评分标准
- 五级评分：1=完全不符合，5=完全符合
- 各维度均分 = 该维度题目之和 ÷ 题目数
- 总分 = 三个维度均分之和，分数越高时间管理倾向越好
