# 孤独感量表（CLS）

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 儿童孤独量表（Children's Loneliness Scale） |
| 缩写 | CLS |
| 编制者 | Asher SR & Wheeler VA（1985年） |
| 中文版 | 李晓巍等修订（2008年） |
| 适用对象 | 小学至初中（6-15岁） |
| 题目数量 | 24题（16题计分 + 8题辅助） |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估儿童在学校中的孤独感和社交满意度 |

---

## 量表结构

| 类型 | 题目数 | 内容 |
|-----|-------|-----|
| 孤独感主题（计分） | 16题 | 孤独、被拒绝、不受欢迎等 |
| 辅助题（填充，不计分） | 8题 | 关于偏好活动的中性问题 |

### 四个维度

| 维度 | 内容 |
|-----|-----|
| 纯孤独感 | 内心孤独、难过的体验 |
| 社交能力知觉 | 对自身社交能力的评价 |
| 同伴地位评价 | 在同伴中被接受程度 |
| 社交需求未满足 | 对友谊和社交的渴望 |

## 评分标准

### 各题评分（五级评分）

| 选项 | 得分（正向题） | 得分（反向题） |
|-----|-------------|-------------|
| 不是这样 | 1分 | 5分 |
| 很少这样 | 2分 | 4分 |
| 有时这样 | 3分 | 3分 |
| 经常这样 | 4分 | 2分 |
| 总是这样 | 5分 | 1分 |

### 计分说明

- 只计16个主题题目的得分，8道辅助题不计入
- **总分** = 16道计分题得分之和（16-80分）
- 辅助题目不参与评分，仅为使问卷不显得突兀

### 结果解释

| 总分范围 | 孤独感水平 |
|---------|---------|
| 16-32分 | 孤独感低，社交关系良好 |
| 33-48分 | 中等孤独感，有一定社交困难 |
| 49-64分 | 孤独感较高，需要关注 |
| 65-80分 | 孤独感严重，建议干预 |

**信效度**：Cronbach's α = 0.76-0.85；分半信度 0.71-0.85；重测信度 0.71-0.88

---

## 注意事项

1. 量表中插入辅助题的目的是避免只关注孤独话题导致的应答偏差
2. 孤独感高的学生往往同时存在社交技能不足，需要综合评估
3. 结合教师观察和同伴提名社会测量法效果更佳
