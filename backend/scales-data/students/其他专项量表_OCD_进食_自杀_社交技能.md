# 其他专项量表：OCD、进食障碍、自杀风险、社交技能

---

## 一、儿童耶鲁-布朗强迫症量表（CY-BOCS）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 儿童耶鲁-布朗强迫症量表（Children's Yale-Brown Obsessive Compulsive Scale） |
| 缩写 | CY-BOCS |
| 编制者 | Goodman WK等（1989年原版）/ Scahill L等改编儿童版 |
| 适用对象 | 6-17岁儿童及青少年 |
| 题目数量 | 10个核心评估项 + 症状核查清单 |
| 评估方式 | 半结构化访谈（临床医生评定） |
| 测试时间 | 约30-45分钟 |
| 用途 | 评估强迫症（OCD）症状的严重程度 |

### 量表结构

| 量表 | 评估内容 | 题数 |
|-----|---------|-----|
| 强迫思维量表（5题） | 时间占用、干扰程度、痛苦程度、抵抗、控制力 | 5题 |
| 强迫行为量表（5题） | 时间占用、干扰程度、痛苦程度、抵抗、控制力 | 5题 |

### 评分标准

每个项目按0-4分评分：

| 得分 | 强迫思维时间占用 | 强迫行为时间占用 |
|-----|--------------|--------------|
| 0 | 无 | 无 |
| 1 | 轻度（<1小时/天） | 轻度（<1小时/天） |
| 2 | 中度（1-3小时/天） | 中度（1-3小时/天） |
| 3 | 重度（3-8小时/天） | 重度（3-8小时/天） |
| 4 | 极重（>8小时/天） | 极重（>8小时/天） |

**CY-BOCS总分** = 强迫思维5项 + 强迫行为5项（0-40分）

| 总分范围 | 严重程度 |
|---------|---------|
| 0-7分 | 轻微 |
| 8-15分 | 轻度 |
| 16-23分 | 中度 |
| 24-31分 | 重度 |
| 32-40分 | 极重 |

## 二、儿童进食态度量表（ChEAT）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 儿童进食态度量表（Children's Eating Attitudes Test） |
| 缩写 | ChEAT |
| 编制者 | Maloney MJ等（1988年），改编自成人版EAT-26 |
| 适用对象 | 8-13岁儿童 |
| 题目数量 | 26题 |
| 答题形式 | 六级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 筛查儿童进食态度问题和厌食倾向 |

### 量表结构（3个分量表）

| 分量表 | 内容 | 题目数 |
|-------|-----|-------|
| 节食 | 对食物的回避态度和节食行为 | 13题 |
| 贪食与食物关注 | 对食物的过度关注和暴食 | 6题 |
| 口腔控制 | 他人对自己进食行为的控制 | 7题 |

### 代表性题目

| 题目内容 |
|---------|
| 我对自己的进食很担心 |
| 我害怕体重超重 |
| 我会减少自己的食物摄入量 |
| 我喜欢在饭后继续吃东西 |
| 我知道卡路里并且计算我所吃的食物 |
| 如果我多吃一点，我会感到很难受 |

### 评分标准

- 六级评分：从不=0，很少=0，有时=0，经常=1，通常=2，总是=3
- 总分 ≥ 20分提示有进食态度问题

## 三、Beck 自杀意念量表（BSSI）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Beck自杀意念量表（Beck Scale for Suicide Ideation） |
| 缩写 | BSSI |
| 编制者 | Beck AT等（1979年） |
| 中文版 | 已有中文修订版 |
| 适用对象 | 初中生至成人（12岁以上）（需有专业人员在场） |
| 题目数量 | 19题（初筛5题，完整19题） |
| 评估方式 | 结构化访谈或自评 |
| 用途 | 评估自杀意念的程度和特点 |

### 量表结构

前5题为筛查题：
- 若第4、5题均得0分，则不继续评估其余题目

### 代表性题目与评分

**第1题：希望活下去的欲望**
- 0 = 中等到强烈
- 1 = 弱
- 2 = 没有

**第4题：主动自杀欲望**
- 0 = 没有
- 1 = 弱
- 2 = 中等到强烈

**第5题：被动自杀欲望**
- 0 = 希望采取预防措施来保住生命
- 1 = 对生死漠然
- 2 = 希望死掉，不想采取预防措施

**总分解释**：
- 总分范围 0-38分
- > 0分：需引起重视
- > 6分：严重自杀风险，需立即干预

**⚠️ 注意**：此量表需在专业人员监督下使用，不得单独作为学生自评工具分发。

## 四、社会技能评定量表（SSRS）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 社会技能评定系统（Social Skills Rating System） |
| 缩写 | SSRS |
| 编制者 | Gresham FM & Elliott SN（1990年） |
| 适用对象 | 3-18岁，有父母版、教师版、学生版 |
| 题目数量 | 约52题（父母/教师版）/ 34题（学生版） |
| 答题形式 | 三级频率评分 + 三级重要性评分 |
| 用途 | 评估儿童社会技能和行为问题 |

### 量表结构

**社会技能维度**：
| 维度 | 内容 |
|-----|-----|
| 合作 | 帮助他人、遵守规则 |
| 自我维护 | 适当表达和维护自己 |
| 责任感 | 负责任地处理自己的事务 |
| 共情 | 理解他人感受 |
| 自我控制 | 控制情绪和冲动 |

**行为问题维度**（父母/教师版）：
- 外化行为问题
- 内化行为问题
- 多动问题（教师版）

### 评分标准

**频率评分**：
- 0 = 从不
- 1 = 有时
- 2 = 经常

**重要性评分**（评估者判断该技能重要性）：
- 0 = 不重要
- 1 = 重要
- 2 = 非常重要

## 五、儿童创伤问卷（CTQ-SF）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 儿童期创伤问卷简版（Childhood Trauma Questionnaire-Short Form） |
| 缩写 | CTQ-SF |
| 编制者 | Bernstein DP等（2003年简版） |
| 中文版 | 赵幸福等修订 |
| 适用对象 | 中学生至成人（12岁以上，回忆童年经历） |
| 题目数量 | 28题 |
| 答题形式 | 五级评分 |
| 用途 | 评估童年期不良经历的暴露情况 |

### 量表结构（5个维度 + 3题效度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 情感虐待 | 5题 | 被辱骂、威胁、贬低 |
| 躯体虐待 | 5题 | 被体罚、打伤 |
| 性虐待 | 5题 | 遭受性侵 |
| 情感忽视 | 5题 | 情感需求被忽视 |
| 躯体忽视 | 5题 | 基本生活需求被忽视 |
| 效度量表 | 3题 | 检验填写可信度 |

### 评分标准

- 五级评分：1=从不，5=总是
- 各维度总分 = 5题之和（5-25分）

| 维度 | 无到最小 | 轻到中度 | 中到重度 | 重度到极重 |
|-----|---------|---------|---------|---------|
| 情感虐待 | 5-8 | 9-12 | 13-15 | 16-25 |
| 躯体虐待 | 5-7 | 8-9 | 10-12 | 13-25 |
| 性虐待 | 5 | 6-7 | 8-12 | 13-25 |
| 情感忽视 | 5-9 | 10-14 | 15-17 | 18-25 |
| 躯体忽视 | 5-7 | 8-9 | 10-12 | 13-25 |
