# 网络成瘾与手机依赖量表系列

---

## 一、Young 网络成瘾量表（IAT）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 网络成瘾测验（Internet Addiction Test） |
| 缩写 | IAT |
| 编制者 | Kimberly Young（1998年） |
| 中文版 | 国内多家机构完成翻译和适用性验证 |
| 适用对象 | 小学高年级至高中（10岁以上，有上网行为者） |
| 题目数量 | 20题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约5分钟 |
| 用途 | 筛查网络成瘾倾向及严重程度 |

### 完整题目（20题）

> 请根据您最近的实际情况，选择最符合的选项。

| 题号 | 题目内容 |
|-----|---------|
| 1 | 您发现您上网的时间比预期的要长吗？ |
| 2 | 您会忽视家务而把更多的时间花在网上吗？ |
| 3 | 您宁愿在网上寻刺激，而不愿意与亲密伴侣亲热吗？ |
| 4 | 您会在网上与新朋友建立关系吗？ |
| 5 | 他人是否抱怨您花了太多时间上网？ |
| 6 | 您的学业成绩或学校作业是否因为您在网上花了太多时间而受到影响？ |
| 7 | 您会在还有其他事情要做之前先查看电子邮件吗？ |
| 8 | 您的学习效果是否会因上网而受到影响？ |
| 9 | 当有人问您在网上做什么时，您是否会变得戒备或行事秘密？ |
| 10 | 您是否会用想想上网来阻止日常生活中令人烦恼的想法？ |
| 11 | 您是否发现自己期待再次上网？ |
| 12 | 您是否害怕生活中没有网络会变得枯燥、空虚和无趣？ |
| 13 | 当有人在您上网时打扰您，您是否会大发雷霆、咆哮或生气？ |
| 14 | 您是否会因为熬夜上网而出现睡眠不足的问题？ |
| 15 | 当您不上网时，您是否会过度关注网上的事情或幻想上网？ |
| 16 | 当您上网时，您是否会说"再等几分钟"？ |
| 17 | 您是否曾经尝试过减少上网时间但失败了？ |
| 18 | 您是否会试图隐瞒您在网上花了多少时间？ |
| 19 | 您是否宁愿花更多时间上网，而不愿意与他人外出？ |
| 20 | 当您不上网时，您是否感到沮丧、情绪低落或神经过敏，而当您重新上网后，这种感觉消失？ |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 从不 | 0分（或1分） |
| 很少 | 1分（或2分） |
| 有时 | 2分（或3分） |
| 经常 | 3分（或4分） |
| 总是 | 4分（或5分） |

> 注：不同版本计分略有差异，以下使用1-5分版本（满分100分）

| 总分范围 | 网络使用状态 |
|---------|------------|
| **20-39分** | 正常用户，无需担心 |
| **40-59分** | 轻度成瘾倾向，需注意 |
| **60-79分** | 中度成瘾，建议寻求帮助 |
| **80-100分** | 重度成瘾，需立即干预 |

## 二、中学生手机依赖量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 王小辉等（国内本土编制） |
| 适用对象 | 初中生、高中生（12-18岁） |
| 题目数量 | 16题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约5分钟 |
| 用途 | 评估中学生手机依赖程度 |

### 量表结构（3个维度）

| 维度 | 内容 |
|-----|-----|
| 戒断性 | 离开手机时出现的焦虑、烦躁感 |
| 突显性 | 手机使用占据优先位置 |
| 强迫性 | 无法控制手机使用行为 |

### 代表性题目

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 手机不见时我觉得与外界失去联系 | 戒断性 |
| 2 | 如果忘记带手机，我会感到焦躁不安 | 戒断性 |
| 3 | 要我关掉手机是一件很痛苦的事情 | 戒断性 |
| 4 | 我花在手机上的时间比我预想的多 | 强迫性 |
| 5 | 我曾经多次想减少使用手机但失败了 | 强迫性 |
| 6 | 即使手机没有提示，我也会主动去查看 | 强迫性 |
| 7 | 我认为手机是我生活中最重要的事物之一 | 突显性 |
| 8 | 没有手机，我觉得无法度过一天 | 突显性 |
| 9 | 我宁愿用手机而不是面对面交流 | 突显性 |
| ... | ... | ... |

### 评分标准

- 五级评分：1=极不符合，5=非常符合
- 总分 = 16题得分之和（16-80分）
- 总分越高表明手机依赖程度越深
- Cronbach's α = 0.889；分半信度 = 0.882

| 总分范围 | 依赖程度 |
|---------|---------|
| 16-32分 | 无依赖 |
| 33-48分 | 轻度依赖 |
| 49-64分 | 中度依赖 |
| 65-80分 | 重度依赖 |

## 三、中学生网络成瘾量表（IADDS）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 网络成瘾诊断量表（Internet Addiction Diagnostic Scale） |
| 缩写 | IADDS |
| 适用对象 | 初中生、高中生 |
| 题目数量 | 28题 |
| 答题形式 | 是/否二分制 |
| 用途 | 诊断性筛查网络成瘾 |

### 主要评估内容

- 耐受性症状（需要越来越多时间才能满足）
- 戒断症状（减少使用后出现的负性情绪）
- 强迫性使用（无法控制上网行为）
- 社交隔离（网络代替现实社交）
- 功能损害（影响学业、家庭、人际）

### 评分标准

- 二分计分：是=1分，否=0分
- 总分范围 0-28分

| 总分 | 诊断 |
|-----|-----|
| ≤ 10分 | 正常使用 |
| 11-15分 | 轻度成瘾 |
| 16-20分 | 中度成瘾 |
| ≥ 21分 | 重度成瘾 |
