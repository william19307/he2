# 学业相关心理量表系列

## 一、中学生学习动机量表（MSMT）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 中学生学习动机量表（Middle School Students' Motivation Test） |
| 缩写 | MSMT |
| 编制者 | 中国心理学家（基于国内中学生群体本土编制） |
| 适用对象 | 初中生、高中生（12-18岁） |
| 题目数量 | 20题 |
| 答题形式 | 是/否二元选择 |
| 测试时间 | 约5-8分钟 |
| 用途 | 评估中学生学习动机的状态和方向 |

### 量表结构（4个分量表）

| 分量表 | 题目数 | 主要评估内容 |
|-------|-------|------------|
| 动机过弱 | 5题 | 学习动力不足、缺乏主动性 |
| 动机过强 | 5题 | 学习焦虑过高、目标过于紧迫 |
| 学习兴趣 | 5题 | 对学习内容的内在兴趣 |
| 学习目标 | 5题 | 目标设定与执行能力 |

### 代表性题目

| 题号 | 题目内容 | 分量表 |
|-----|---------|-------|
| 1 | 如果别人不督促你，你极少主动地学习 | 动机过弱 |
| 2 | 当你读书时，需要很长时间才能提起精神来 | 动机过弱 |
| 3 | 你一读书就觉得疲劳与厌倦，直想睡觉 | 动机过弱 |
| 4 | 你迫切希望在短时间内大幅度提高学习成绩 | 动机过强 |
| 5 | 为完成作业，你宁愿废寝忘食、通宵达旦 | 动机过强 |
| 6 | 你感兴趣的科目你总是愿意多花时间钻研 | 学习兴趣 |
| 7 | 你觉得学习某些科目真的很有意思 | 学习兴趣 |
| 8 | 你给自己定的学习目标，多数都能完成 | 学习目标 |
| 9 | 你能为学习目标坚持下去，不轻易放弃 | 学习目标 |

### 评分标准

- 是=1分，否=0分（部分题目可能反向计分）
- 各分量表得分 = 该分量表各题之和

| 分量表 | 偏高说明 | 偏低说明 |
|-------|---------|---------|
| 动机过弱 | 学习懈怠，需激励 | 学习意愿充足 |
| 动机过强 | 焦虑过高，需减压 | 紧迫感适度 |
| 学习兴趣 | 兴趣浓厚 | 学习兴趣不足 |
| 学习目标 | 目标清晰坚定 | 目标感缺乏 |

## 二、学业自我效能感量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 梁宇颂、周宗奎（2003年，国内修订） |
| 适用对象 | 初中生、高中生 |
| 题目数量 | 22题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约8分钟 |
| 用途 | 评估学生对自身学习能力和学习行为的信念 |

### 量表结构（2个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 学习能力自我效能感 | 11题 | 相信自己能掌握知识和技能 |
| 学习行为自我效能感 | 11题 | 相信自己能坚持有效的学习行为 |

### 代表性题目

| 类别 | 题目示例 |
|-----|---------|
| 学习能力效能感 | 我相信自己有能力在学习上取得好成绩 |
| 学习能力效能感 | 遇到难题时，我相信自己能通过努力解决 |
| 学习行为效能感 | 我总是在书本或笔记本上划出重点以帮助学习 |
| 学习行为效能感 | 我能坚持做课前预习和课后复习 |

### 评分标准

- 五级评分：1=完全不符合，5=完全符合
- 各维度均分越高，该方面自我效能感越强

## 三、学生学习倦怠量表（MBI-SS）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 学生倦怠量表（Maslach Burnout Inventory-Student Survey） |
| 缩写 | MBI-SS |
| 编制者 | Schaufeli WB等（2002年，适应教育情境） |
| 中文版 | 国内已有中文修订版 |
| 适用对象 | 初中生至大学生（12岁以上） |
| 题目数量 | 15题 |
| 答题形式 | 七级评分 |
| 测试时间 | 约8分钟 |
| 用途 | 评估学生对学习的倦怠程度 |

### 量表结构（3个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 情绪耗竭 | 5题 | 学习引起的情绪疲竭感 |
| 去人格化（冷嘲） | 4题 | 对学校/学习的冷漠、消极态度 |
| 低成就感 | 6题 | 对学业成就的自我评价 |

### 代表性题目

| 题目示例 | 维度 |
|---------|-----|
| 我因为学习而感到情绪耗竭 | 情绪耗竭 |
| 我感到精疲力竭，不想再去上学 | 情绪耗竭 |
| 我怀疑学习的意义 | 去人格化 |
| 我越来越不关心是否完成学业 | 去人格化 |
| 我在上学过程中不断地获得成就感 | 低成就感（反向） |
| 我觉得自己是个优秀的学生 | 低成就感（反向） |

### 评分标准

- 七级评分：0=从来没有，6=每天都有
- 情绪耗竭、去人格化：分数越高倦怠越强
- 低成就感题目反向计分：原始分高=低成就感低

## 四、青少年生活事件量表（ASLEC）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 青少年自评生活事件量表（Adolescent Self-rating Life Events Checklist） |
| 缩写 | ASLEC |
| 编制者 | 刘贤臣等（1987年，国内本土编制） |
| 适用对象 | 小学高年级至高中（11-25岁） |
| 题目数量 | 27题 |
| 答题形式 | 是否发生 + 影响程度六级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估青少年过去一年内的负性生活事件及影响程度 |

### 量表结构（6个因子）

| 因子 | 内容 |
|-----|-----|
| 人际关系 | 与同学、教师、父母关系困难 |
| 学习压力 | 考试失败、学业负担 |
| 受惩罚 | 被批评、处分 |
| 丧失 | 亲人死亡、分离 |
| 健康与适应 | 患病、住院 |
| 其他 | 未分类的压力事件 |

### 代表性题目

| 题号 | 题目内容 | 因子 |
|-----|---------|-----|
| 1 | 与同学严重冲突 | 人际关系 |
| 2 | 被同学嘲笑、欺负 | 人际关系 |
| 3 | 与老师严重冲突或被老师误解 | 人际关系 |
| 4 | 与父母严重冲突 | 人际关系 |
| 5 | 考试失败或成绩不理想 | 学习压力 |
| 6 | 学习中感到无助和困难 | 学习压力 |
| 7 | 被老师或父母惩罚 | 受惩罚 |
| 8 | 受到严厉处分或批评 | 受惩罚 |
| 9 | 亲密朋友的离去（转学、搬家） | 丧失 |
| 10 | 亲人死亡或重病 | 丧失 |

### 评分标准

**是否发生**：未发生选A（不计入）；发生则选B-F（计入）

**影响程度评分**（仅对"已发生"事件评分）：

| 选项 | 得分 |
|-----|-----|
| 未发生 | 不计分 |
| 发生但无影响 | 1分 |
| 轻度影响 | 2分 |
| 中度影响 | 3分 |
| 重度影响 | 4分 |
| 极重度影响 | 5分 |

**总分** = 所有已发生事件的影响程度之和

- 信度：Cronbach's α = 0.92，重测信度 = 0.73
