# EPQ 艾森克人格问卷（88题版）

> **来源**：H.J. Eysenck & S.B.G. Eysenck 原编，龚耀先教授修订中文版（湖南医科大学，1983年）。以下88道完整题目来自公开学术网站（心理学网 xinlixue.cn）。

## 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 中学生至成人（建议15岁以上） |
| 题目数量 | 88题 |
| 答题形式 | 是 / 否 |
| 测试时间 | 约15-20分钟 |

---

## 量表结构（4个量表）

| 量表 | 代码 | 主要测量内容 |
|-----|-----|------------|
| 外向性 | E | 社交性、活力、冒险性 |
| 神经质 | N | 情绪稳定性、焦虑倾向 |
| 精神质 | P | 攻击性、冷漠度、不顺从 |
| 掩饰性 | L | 答题真实性（测谎） |

## 评分标准

### 维度与计分方向

各量表对应题目（根据龚耀先修订版标准答案，答"是"计1分的方向）：

**E（外向性）正向题**（答"是"=E+1）：1、5、10、13、14、17、25、33、37、41、49、53、55、61、65、71、80、84

**E（外向性）反向题**（答"否"=E+1）：21、29、45

**N（神经质）正向题**（答"是"=N+1）：3、7、12、15、19、23、27、31、35、39、43、47、51、57、59、63、67、73、74、77、78、82、86

**P（精神质）正向题**（答"是"=P+1）：13、22、26、30、34、46、50、68、70、75、81、85

**P（精神质）反向题**（答"否"=P+1）：2、6、9、11、18、42、56、58、62、72、87、88

**L（掩饰性）正向题**（答"是"=L+1）：20、36、38、58、62、72、87

**L（掩饰性）反向题**（答"否"=L+1）：4、8、16、24、28、32、40、44、48、52、54、60、64、70、76、79、83

> 注：以上分题规则为龚耀先修订版常用版本，以正式手册为最终依据。

### T分转换与解释

各量表原始分需查对应性别和年龄组的常模表转换为T分：

| T分范围 | 解释 |
|--------|-----|
| **T > 61.5** | 典型高分（外向/情绪不稳定/精神质明显） |
| **56.7-61.5** | 趋向高分 |
| **43.3-56.7** | 中间型 |
| **38.5-43.3** | 趋向低分 |
| **T < 38.5** | 典型低分（内向/情绪稳定） |

**L量表**：T > 60 提示掩饰性强，答卷可靠性下降

## 注意事项

1. EPQ描述人格特征，**无好坏之分**，结果仅供参考
2. L分过高（T > 60）说明结果可能不准确
3. **第69题**（"不如死了为好"）答"是"需特别关注
