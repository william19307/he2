# 专项行为量表：网络欺凌、亲社会行为、学校联结

---

## 一、网络欺凌量表（施害者版 + 受害者版）

> **来源**：综合国内外多项研究，基于Tokunaga（2010年）综述框架编制，多个中文版已在中学生中验证。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁）|
| 题目数量 | 各10题（施害者版 + 受害者版）|
| 答题形式 | 五级评分 |
| 评定时限 | 过去三个月 |

### 网络欺凌类型（5类）
| 类型 | 内容 |
|-----|-----|
| 骚扰 | 反复发送具有攻击性的信息 |
| 人肉搜索 | 散布他人个人信息 |
| 排斥 | 在网络群体中孤立他人 |
| 冒充 | 盗用他人账号发送不当内容 |
| 网络诽谤 | 散布虚假信息或诋毁 |

### 施害者版题目（10题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我曾通过网络向某人发送过侮辱或威胁信息 |
| 2 | 我曾在网上散布过他人的隐私 |
| 3 | 我曾故意在网络群体中排斥某人 |
| 4 | 我曾盗用他人账号发送过不良内容 |
| 5 | 我曾在网上发布过他人的尴尬照片或视频 |
| 6 | 我曾在网上故意说谎或编造谣言来伤害某人 |
| 7 | 我曾反复骚扰过某人 |
| 8 | 我曾通过网络威胁过某人 |
| 9 | 我曾冒充别人在网上发帖 |
| 10 | 我曾在网上公开嘲笑或羞辱某人 |

### 受害者版题目（10题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 有人通过网络向我发送过侮辱或威胁信息 |
| 2 | 有人在网上散布了我的隐私 |
| 3 | 有人故意在网络群体中排斥我 |
| 4 | 有人盗用了我的账号发送过不良内容 |
| 5 | 有人在网上发布了我的尴尬照片或视频 |
| 6 | 有人在网上故意说谎或编造谣言来伤害我 |
| 7 | 有人反复骚扰过我 |
| 8 | 有人通过网络威胁过我 |
| 9 | 有人冒充我在网上发帖 |
| 10 | 有人在网上公开嘲笑或羞辱我 |

### 评分标准
- 五级评分：1=从未，2=偶尔，3=有时，4=经常，5=很频繁
- **总分** = 10题之和（10-50分），分数越高欺凌行为/受害程度越严重

## 二、亲社会行为量表（中小学生版）

> **来源**：综合Caldarella & Merrell（1997年）框架及国内研究编制，邓慧华等（2012年）中国本土修订版。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 小学至初中（6-15岁）|
| 题目数量 | 20题（4个维度各5题）|
| 答题形式 | 五级评分 |

### 量表结构（4个亲社会行为类型）
| 维度 | 内容 |
|-----|-----|
| 分享 | 主动与他人分享物品或资源 |
| 助人 | 主动帮助有需要的人 |
| 安慰 | 关心并安慰他人的情绪 |
| 合作 | 与他人协作完成任务 |

### 完整题目（20题）

**分享（第1-5题）**
| 题号 | 题目内容 |
|-----|---------|
| 1 | 我会与同学分享我的文具和学习用品 |
| 2 | 我愿意把自己喜欢的东西借给别人 |
| 3 | 我会与朋友分享零食 |
| 4 | 当别人没有某些东西时，我会主动分享 |
| 5 | 我把我的书本或玩具分给需要的人 |

**助人（第6-10题）**
| 题号 | 题目内容 |
|-----|---------|
| 6 | 当同学学习遇到困难时，我会帮助他们 |
| 7 | 我主动帮助老人或行动不便的人 |
| 8 | 当有人摔倒时，我会主动上前帮扶 |
| 9 | 我会帮助同学解决问题 |
| 10 | 我主动帮助班级或学校做一些事情 |

**安慰（第11-15题）**
| 题号 | 题目内容 |
|-----|---------|
| 11 | 当同学心情不好时，我会主动安慰他们 |
| 12 | 我会关注朋友是否遇到了麻烦 |
| 13 | 看到别人难过时，我会感到不安并想帮助他们 |
| 14 | 我会询问难过的朋友发生了什么事 |
| 15 | 我会鼓励情绪低落的同学 |

**合作（第16-20题）**
| 题号 | 题目内容 |
|-----|---------|
| 16 | 小组活动时，我积极配合组员共同完成任务 |
| 17 | 我愿意听取他人的意见并做出妥协 |
| 18 | 在合作中，我会考虑到组员的感受 |
| 19 | 我能与不同的人一起工作 |
| 20 | 完成集体任务时，我会尽力贡献自己的力量 |

### 评分标准
- 五级评分：1=从来没有，5=总是如此
- **各维度均分** = 5题均分（1-5分）
- **总分** = 20题均分，分数越高亲社会行为越强

## 三、学校联结量表（SCS-10）

> **来源**：Maddox SJ & Prinz RJ（2003年）综合框架；中文版综合多项研究，包含学生自评10题版本。

### 基本信息
| 属性 | 内容 |
|------|------|
| 适用对象 | 小学至高中（6-18岁）|
| 题目数量 | 10题 |
| 答题形式 | 五级评分 |

### 完整题目（10题）

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 我觉得我在这所学校很重要 | 归属感 |
| 2 | 我感到与这所学校的同学有联结 | 同伴联结 |
| 3 | 当我在学校遇到问题时，能获得帮助 | 教师支持 |
| 4 | 老师关心我 | 教师支持 |
| 5 | 这所学校的生活让我感到满意 | 满意度 |
| 6 | 我为我的学校感到骄傲 | 归属感 |
| 7 | 我在这所学校感到安全 | 安全感 |
| 8 | 我感到这里的人关心我的学习 | 教师支持 |
| 9 | 我的同学会在我需要时帮助我 | 同伴联结 |
| 10 | 我喜欢来上学 | 满意度 |

### 评分标准
- 五级评分：1=完全不同意，5=完全同意
- **总分** = 10题之和（10-50分），分数越高学校联结越强
