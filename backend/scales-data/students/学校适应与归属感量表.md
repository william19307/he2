# 学校适应与归属感量表系列

---

## 一、儿童学校适应量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 基于国内学者编制的本土化版本 |
| 适用对象 | 小学至初中（6-15岁） |
| 题目数量 | 32题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估学生对学校环境的适应程度 |

### 量表结构（4个维度）

| 维度 | 题目数 | 主要内容 |
|-----|-------|---------|
| 学业适应 | 8题 | 对学习任务和课程的适应 |
| 社交适应 | 8题 | 对同伴和集体的融入 |
| 情绪适应 | 8题 | 在学校环境中的情绪状态 |
| 行为适应 | 8题 | 遵守学校规则和常规 |

### 代表性题目

| 维度 | 题目示例 |
|-----|---------|
| 学业适应 | 我能按时完成课后作业 |
| 学业适应 | 我对大多数科目能够理解和跟上 |
| 社交适应 | 我在班里有朋友 |
| 社交适应 | 我能融入班级的集体活动 |
| 情绪适应 | 我在学校里感到开心 |
| 情绪适应 | 我不会因为学校的事情而感到焦虑 |
| 行为适应 | 我能遵守课堂规则 |
| 行为适应 | 我按时到校、不迟到早退 |

### 评分标准

- 五级评分：1=完全不符合，5=完全符合
- 各维度均分越高，该方面适应越好

## 二、学校归属感量表（PSSM）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 学校心理归属感量表（Psychological Sense of School Membership） |
| 缩写 | PSSM |
| 编制者 | Goodenow CS（1993年） |
| 中文版 | 国内已完成中文修订 |
| 适用对象 | 小学高年级至高中（10-18岁） |
| 题目数量 | 18题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约5分钟 |
| 用途 | 评估学生对学校的归属感和被接纳感 |

### 完整题目（18题）

> 以下描述符合您在学校里的感受吗？

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我感觉自己是学校的一份子 |
| 2 | 学校里的人都注意到我 |
| 3 | 我在学校里有时感到孤独 |
| 4 | 我能够很容易地融入学校的各种活动 |
| 5 | 学校里的其他学生喜欢我 |
| 6 | 在学校里，我常常感到格格不入 |
| 7 | 学校里有很多我尊敬的老师 |
| 8 | 我有时候不想上学 |
| 9 | 我在学校感到被重视 |
| 10 | 我有时候感到自己是多余的 |
| 11 | 我感到学校里的老师关心我 |
| 12 | 我并不真正属于学校 |
| 13 | 我不想改变学校 |
| 14 | 我在学校里感到很孤单 |
| 15 | 老师真的理解我 |
| 16 | 我觉得我的班级是我的班级 |
| 17 | 我在学校里感到我很重要 |
| 18 | 我在学校里觉得我很无助 |

### 评分标准

- 正向题（1、2、4、5、7、9、11、13、15、16、17）：1=完全不符合，5=完全符合
- 反向题（3、6、8、10、12、14、18）：反向计分
- 总分越高，学校归属感越强

## 三、小学生心理适应量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 苏志强等（中国本土） |
| 适用对象 | 小学三至六年级（8-12岁） |
| 题目数量 | 47题 |
| 答题形式 | 四级评分 |
| 测试时间 | 约15分钟 |
| 用途 | 评估小学生的心理适应特点和行为倾向 |

### 量表结构

| 维度 | 内容 |
|-----|-----|
| 学习适应 | 学习行为和习惯的适应性 |
| 环境适应 | 对学校物理环境的适应 |
| 情绪适应 | 情绪调控和稳定性 |
| 自我适应 | 自我概念的适应性 |
| 人际适应 | 与同伴和教师关系的适应 |

### 代表性题目

| 题目示例 |
|---------|
| 我能集中注意力听课 |
| 我喜欢我们班的教室 |
| 我在学校里感到很快乐 |
| 我认为自己是个有用的人 |
| 我容易和同学成为好朋友 |
