# 幸福感与生活满意度量表系列

---

## 一、青少年幸福感量表（EPOCH）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 青少年幸福感量表（EPOCH Measure of Adolescent Well-Being） |
| 缩写 | EPOCH |
| 编制者 | Kern ML等（2016年） |
| 中文版 | 已完成中国标准化 |
| 适用对象 | 中小学生（10-18岁） |
| 题目数量 | 20题 |
| 答题形式 | 五级评分 |
| 测试时间 | 约5-8分钟 |
| 用途 | 评估青少年积极心理健康与幸福感 |

### 量表结构（5个维度，各4题）

| 维度 | 英文 | 主要内容 |
|-----|-----|---------|
| 积极情绪 | Positive Emotion | 感受快乐、愉悦的频率 |
| 投入 | Engagement | 对活动的专注投入程度 |
| 关系 | Relationship | 与他人的积极关系质量 |
| 意义 | Meaning | 感受到生活有意义和目的 |
| 成就 | Accomplishment | 对目标达成的感知和自豪感 |

### 完整题目（20题）

> 请根据**自己平时的感受**，选择最符合的选项。

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 我感到快乐 | 积极情绪 |
| 2 | 我感到喜悦 | 积极情绪 |
| 3 | 我感到积极开朗 | 积极情绪 |
| 4 | 我感到满足 | 积极情绪 |
| 5 | 我对我做的事情感兴趣 | 投入 |
| 6 | 我对我做的事情充满热情 | 投入 |
| 7 | 我对我做的事全力以赴 | 投入 |
| 8 | 我喜欢学习新事物 | 投入 |
| 9 | 我和他人关系很好 | 关系 |
| 10 | 我感到被他人接受和喜欢 | 关系 |
| 11 | 人们会帮助和支持我 | 关系 |
| 12 | 我感到我是集体的一份子 | 关系 |
| 13 | 我的生活是有意义的 | 意义 |
| 14 | 我所做的事情是有价值的 | 意义 |
| 15 | 我的生活有目标和方向 | 意义 |
| 16 | 我认为生活是有意义的 | 意义 |
| 17 | 我努力实现我的目标 | 成就 |
| 18 | 我设法完成我开始做的事 | 成就 |
| 19 | 我擅长处理问题 | 成就 |
| 20 | 我是成功的 | 成就 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 几乎从未 | 1分 |
| 有时 | 2分 |
| 有些时候 | 3分 |
| 经常 | 4分 |
| 几乎总是 | 5分 |

- 各维度得分 = 该维度4题之和（4-20分）
- 总分 = 20题之和（20-100分），分数越高幸福感越强
- 内部一致性 Cronbach's α = 0.85

## 二、青少年主观幸福感量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 编制者 | 邢占军（2002年，中国本土） |
| 适用对象 | 中学生（12-18岁） |
| 题目数量 | 36题 |
| 答题形式 | 七级评分 |
| 测试时间 | 约15分钟 |
| 用途 | 评估学生在多个生活领域的满意度 |

### 量表结构（8个领域）

| 领域 | 主要评估内容 |
|-----|------------|
| 友谊满意度 | 与朋友关系的满意程度 |
| 家庭满意度 | 家庭生活的满意程度 |
| 学校满意度 | 对学校环境的满意程度 |
| 学业满意度 | 对自身学习情况的满意程度 |
| 自由满意度 | 对个人自主权和自由的满意程度 |
| 环境满意度 | 对生活环境的满意程度 |
| 自我满意度 | 对自身的评价与满意度 |
| 安全满意度 | 对人身安全的满意程度 |

### 评分标准

- 七级评分：1=完全不符合，7=完全符合
- **各领域均分解释**：
  - ≥6分：非常满意
  - 5-5.99分：满意
  - 4-4.99分：基本满意
  - 3-3.99分：不确定
  - <3分：不满意

## 三、儿童生活满意度量表（SLSS）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Student Life Satisfaction Scale |
| 缩写 | SLSS |
| 编制者 | Huebner ES（1991年） |
| 适用对象 | 小学三年级至高中 |
| 题目数量 | 7题 |
| 答题形式 | 六级评分 |
| 用途 | 测量学生整体生活满意度 |

### 完整题目（7题）

| 题号 | 题目内容 |
|-----|---------|
| 1 | 我的生活过得很美好 |
| 2 | 我的生活正如我所愿 |
| 3 | 我希望我的生活有所不同（反向） |
| 4 | 我有着美好的生活 |
| 5 | 我有着自己想要的东西 |
| 6 | 我的生活比大多数同龄人好 |
| 7 | 我觉得我的生活很棒 |

### 评分标准

- 六级评分：1=非常不同意，6=非常同意
- 反向计分：第3题按6-1反向计分
- 总分范围7-42分，分数越高生活满意度越高
