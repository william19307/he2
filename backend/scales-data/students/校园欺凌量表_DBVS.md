# 校园欺凌量表系列

---

## 一、特拉华校园欺凌受害量表（DBVS-S，学生版）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Delaware Bullying Victimization Scale-Student |
| 缩写 | DBVS-S |
| 适用对象 | 小学高年级至高中（10-18岁） |
| 题目数量 | 12-18题（版本略有差异） |
| 答题形式 | 六级评分 |
| 测试时间 | 约10分钟 |
| 用途 | 评估学生遭受校园欺凌的频率和类型 |

### 量表结构（4个维度）

| 维度 | 典型行为 | 题目数 |
|-----|---------|-------|
| 言语欺凌 | 被嘲笑、骂人、起绰号 | 4题 |
| 身体欺凌 | 被打、推、踢、抢夺 | 4题 |
| 社交/关系欺凌 | 被排斥、散布谣言、孤立 | 4题 |
| 网络欺凌 | 在网上被骚扰、诽谤 | 4-6题 |

### 完整题目（16题）

> 请回答在**过去一个月**内，以下事情发生了多少次？

| 题号 | 题目内容 | 维度 |
|-----|---------|-----|
| 1 | 有同学嘲笑我或叫我不好听的名字 | 言语 |
| 2 | 有同学说我的坏话，让别人讨厌我 | 言语 |
| 3 | 有同学公开羞辱我、让我难堪 | 言语 |
| 4 | 有同学威胁我 | 言语 |
| 5 | 有同学打我或踢我 | 身体 |
| 6 | 有同学推我或拍打我 | 身体 |
| 7 | 有同学抢走或损坏我的东西 | 身体 |
| 8 | 有同学强迫我做我不想做的事 | 身体 |
| 9 | 有同学故意不让我和他们一起玩 | 社交/关系 |
| 10 | 有同学不理我，故意孤立我 | 社交/关系 |
| 11 | 有同学在背后说我的坏话、散布谣言 | 社交/关系 |
| 12 | 有同学做鬼脸或用不友善的眼神看我 | 社交/关系 |
| 13 | 有人通过手机或网络发消息骚扰我 | 网络 |
| 14 | 有人在网上散布关于我的谣言 | 网络 |
| 15 | 有人在网上发布令我难堪的图片或视频 | 网络 |
| 16 | 有人盗用我的账号或在网上冒充我 | 网络 |

### 评分标准

| 选项 | 得分 |
|-----|-----|
| 从来没有 | 1分 |
| 只有1次 | 2分 |
| 2-3次 | 3分 |
| 4-5次 | 4分 |
| 6-9次 | 5分 |
| 每天都有 | 6分 |

- 各维度得分 = 该维度各题均分
- 总分越高表示遭受欺凌越频繁

## 二、青少年欺凌者与被欺凌者量表

### 基本信息

| 属性 | 内容 |
|------|------|
| 适用对象 | 初中生、高中生（12-18岁） |
| 题目数量 | 59题（欺凌者30题 + 被欺凌者29题） |
| 答题形式 | 五级评分 |
| 用途 | 同时评估欺凌施加行为和受欺凌经历 |

### 量表结构（5个维度）

| 维度 | 欺凌者题数 | 被欺凌者题数 |
|-----|----------|-----------|
| 传统身体欺凌 | 7题 | 6题 |
| 传统言语欺凌 | 6题 | 6题 |
| 传统社交欺凌 | 7题 | 6题 |
| 网络图像欺凌 | 5题 | 5题 |
| 网络信息欺凌 | 5题 | 6题 |

### 评分标准

- 五级评分：1=从不这样，5=总是这样
- 分数越高表示欺凌行为频率越高（欺凌者）或遭受欺凌越严重（被欺凌者）

## 三、网络欺凌量表（简版）

### 适用背景

随着社交媒体普及，网络欺凌已成为独立问题，该量表专门评估网络环境中的欺凌行为。

### 主要评估内容

| 欺凌类型 | 举例 |
|---------|-----|
| 骚扰 | 反复发送威胁或侮辱性信息 |
| 人肉搜索 | 未经允许散布个人信息 |
| 排斥 | 在网络群体中故意孤立 |
| 冒充 | 盗用他人账号发布不当内容 |
| 传播谣言 | 网上散布不实信息 |
| 上传羞辱性内容 | 发布他人的尴尬图片/视频 |

---

## 注意事项

1. 量表结果为**筛查依据**，对于报告频繁受害的学生应进一步深度访谈
2. 发现欺凌受害者应提供即时支持，并启动学校干预程序
3. 欺凌行为与心理健康问题密切相关，建议同时评估焦虑、抑郁状况
4. 保护填写者的匿名性，有助于获得真实数据
