# 青少年自伤行为与危机评估量表系列

> ⚠️ **重要提示**：本节量表涉及自伤、自杀等危机行为评估，必须由经过培训的专业人员使用，不得作为普通课堂调查工具分发。

---

## 一、青少年非自杀性自伤行为问卷（NSSI）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 非自杀性自伤行为问卷（Non-Suicidal Self-Injury Questionnaire） |
| 适用对象 | 初中生、高中生（12-18岁，专业人员使用） |
| 题目数量 | 约20题 |
| 用途 | 评估非自杀性自伤行为的频率、类型和功能 |

### 主要评估内容

1. **自伤行为类型**（是否有过以下行为）：
   - 割伤皮肤
   - 烫伤自己
   - 撞击头部或身体
   - 抓伤皮肤
   - 拔头发
   - 打自己

2. **行为频率**：过去12个月内发生次数

3. **行为功能**（自伤的目的）：
   - 减轻情绪痛苦
   - 感受到真实存在
   - 惩罚自己
   - 寻求注意

4. **相关危险因素**：是否有自杀意念

## 二、哥伦比亚自杀严重性评定量表（C-SSRS）

### 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 哥伦比亚自杀严重性评定量表（Columbia Suicide Severity Rating Scale） |
| 缩写 | C-SSRS |
| 编制者 | 哥伦比亚大学（美国FDA认可标准） |
| 中文版 | 已完成官方中文版 |
| 适用对象 | 所有年龄段（含儿童版），必须由专业人员实施 |
| 用途 | 评估自杀意念和自杀行为的严重程度 |

### 量表结构

**自杀意念部分（5级）**：

| 级别 | 描述 |
|-----|-----|
| 1级 | 希望死去（被动自杀意念）：希望死去或希望睡不醒来 |
| 2级 | 非特异性主动自杀意念：想要死亡或结束生命，没有具体想法 |
| 3级 | 有方法的自杀意念：有自杀的想法，且有方法，但没有计划 |
| 4级 | 有意图的自杀意念：有自杀想法，且有意图去实施 |
| 5级 | 有计划的自杀意念：有自杀想法、方法、意图和具体计划 |

**自杀行为部分**：

| 行为类型 | 描述 |
|---------|-----|
| 预备行为 | 采取了会增加自杀机会的步骤 |
| 中断行为 | 他人终止了自杀行为 |
| 停止行为 | 本人终止了自杀行为 |
| 自杀未遂 | 有自杀行为但未死亡 |
| 完成自杀 | 死亡 |

### 使用原则

1. 必须进行**面对面访谈**，不可作为自评问卷使用
2. 任何5级（有计划意念）→ **立即危机干预**
3. 任何3-4级意念 → **当天完成安全评估**
4. 1-2级意念 → **制定安全计划**，密切随访

## 三、危机评估标准化程序（快速版）

### SLAP 风险评估四要素

| 要素 | 英文 | 评估内容 |
|-----|-----|---------|
| 特异性 | Specificity | 自杀计划的具体程度 |
| 致死性 | Lethality | 所选方法的危险程度 |
| 可用性 | Availability | 工具是否容易获得 |
| 预防性 | Preventability | 其他人知道/能阻止的程度 |

### 危机干预优先级

| 风险级别 | 判断标准 | 应对措施 |
|---------|---------|---------|
| **高风险** | 有计划、手段可及、近期有意图 | 立即通知家长、转介精神科 |
| **中等风险** | 有想法、无具体计划 | 当天深度评估、建立安全计划 |
| **低风险** | 被动自杀意念、无计划 | 随访跟进、告知支持资源 |

## 四、青少年心理危机及风险行为筛查量表

### 适用场景

用于学校全体学生初步筛查，发现需要进一步关注的高风险个体。

### 筛查问题示例

| 问题 | 关注意义 |
|-----|---------|
| 最近2周，您有没有感到活着没有意思？ | 被动自杀意念 |
| 最近是否有过伤害自己身体的行为？ | 非自杀性自伤 |
| 您最近是否觉得对任何事情都提不起兴趣？ | 抑郁筛查 |
| 您是否曾经想过不想活了？ | 自杀意念 |

> **任何一题答"是"或"有时"→ 需要个别深入评估，不得忽视。**
