# 儿童自我概念量表（Piers-Harris）

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | Piers-Harris儿童自我概念量表（Piers-Harris Children's Self-Concept Scale） |
| 缩写 | PHCSCS |
| 编制者 | E.V. Piers & D.B. Harris（1964年，1984年修订） |
| 中文版 | 苏林雁等修订（1993年） |
| 适用对象 | 小学三年级至高中（8-16岁） |
| 题目数量 | 80题 |
| 答题形式 | 是/否二元选择 |
| 测试时间 | 约15-20分钟 |
| 用途 | 全面评估儿童自我概念的多个维度 |

---

## 量表结构（6个维度）

| 维度 | 题目数 | 主要评估内容 |
|-----|-------|------------|
| 行为 | 16题 | 对自身行为方式的满意程度 |
| 智力与学业地位 | 17题 | 对学习能力的自我评价 |
| 躯体外貌与特性 | 13题 | 对外貌和身体的满意程度 |
| 焦虑 | 14题 | 焦虑状态和满足感 |
| 受欢迎程度 | 12题 | 在同伴中的受欢迎程度 |
| 愉快与满足 | 10题 | 整体满足感和愉快感 |

---

## 代表性题目

| 维度 | 题目示例 |
|-----|---------|
| 行为 | 我在班里表现好（是=正向） |
| 行为 | 我在家里常闯祸（否=正向） |
| 智力与学业 | 我很聪明（是=正向） |
| 智力与学业 | 我在班上学业成绩比较差（否=正向） |
| 躯体外貌 | 我长得帅气/漂亮（是=正向） |
| 躯体外貌 | 我希望我的外貌好一点（否=正向） |
| 焦虑 | 我常常感到担忧（否=正向） |
| 焦虑 | 我在学校做事很紧张（否=正向） |
| 受欢迎程度 | 我在班里受欢迎（是=正向） |
| 受欢迎程度 | 我有很多朋友（是=正向） |
| 愉快与满足 | 我总体是快乐的（是=正向） |
| 愉快与满足 | 我对自己感到满意（是=正向） |

---

## 评分标准

### 各题计分

- 答"是"并且该题正向计分 = 1分
- 答"否"并且该题反向计分 = 1分
- 其余情况 = 0分

### 总分计算

**总分** = 80题中符合正向答案的题目数（0-80分）

**各维度分** = 该维度符合正向答案的题目数

### 结果解释

| 总分范围 | 自我概念水平 |
|---------|-----------|
| 0-39分 | 低自我概念（<P15） |
| 40-59分 | 中等自我概念 |
| 60-80分 | 高自我概念（>P85） |

---

## 注意事项

- 总分有一定的年龄和性别差异，解释时参考常模
- 各维度分析可帮助识别儿童自我概念的优势和弱点区域
- 量表内设效度指标（极端作答模式可能无效）
