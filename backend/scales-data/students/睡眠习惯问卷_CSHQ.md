# CSHQ 儿童睡眠习惯问卷

## 基本信息

| 属性 | 内容 |
|------|------|
| 全称 | 儿童睡眠习惯问卷（Children's Sleep Habits Questionnaire） |
| 缩写 | CSHQ |
| 编制者 | Judith Owens等（2000年，波士顿儿童医院） |
| 中文版 | 已完成中国标准化 |
| 适用对象 | 学龄儿童（3-12岁，父母填写） |
| 题目数量 | 48题（33题计分）|
| 答题形式 | 三级评分 |
| 测试时间 | 约10-15分钟 |
| 用途 | 筛查儿童睡眠障碍和不良睡眠习惯 |

---

## 量表结构（8个维度）

| 维度 | 题目数（计分）| 主要评估内容 |
|-----|------------|------------|
| 就寝阻力 | 6题 | 上床时的抵抗、挣扎行为 |
| 入睡延迟 | 1题 | 入睡所需时间 |
| 睡眠时长 | 3题 | 睡眠总时间 |
| 睡眠焦虑 | 4题 | 害怕独自入睡 |
| 夜醒 | 3题 | 夜间觉醒次数和持续时间 |
| 异态睡眠 | 7题 | 梦游、说梦话、噩梦等 |
| 睡眠呼吸 | 3题 | 打鼾、呼吸暂停 |
| 白天嗜睡 | 8题 | 白天过度困倦 |

---

## 完整题目（计分部分，33题）

> 请回想孩子**上周情况最典型**的一周，选择最符合实际的选项。

## 评分标准

### 各题评分（三级评分）

| 选项 | 含义（过去一周内） | 得分（问题行为） |
|-----|----------------|--------------|
| 极少 | 每周0-1次 | 1分 |
| 有时 | 每周2-4次 | 2分 |
| 经常 | 每周5-7次 | 3分 |

> 注：部分题目为正向描述（如"孩子容易就寝"），需反向计分（经常=1分，极少=3分）

### CSHQ总分计算

**总分** = 33道计分题得分之和

**筛查阈值**：总分 ≥ **41分** 提示存在临床意义的睡眠问题
- 灵敏度：0.80
- 特异度：0.72

### 各维度判断

各维度分数越高，表示该方面睡眠问题越明显。建议参照正常儿童常模进行比较。

---

## 注意事项

1. 量表由**父母填写**，针对孩子上周最具代表性的一周
2. 阳性结果需进一步评估是否存在器质性因素（如腺样体肥大）
3. 睡眠问题与儿童行为、注意力、情绪问题密切相关，建议联合评估
4. 适用年龄3-12岁，12岁以上使用青少年版睡眠量表
